　　《玄黄不灭》，一重仙元炼玉身，只需消耗大量仙元能量就能练成无上玉身，等同无上金身；二重五行开道宫，需要金木水火土五行之力，炼进五脏，打开五大道宫，成就无上神力。三重四象化四极，需要聚地水火风四象之力，成就双手双脚四肢之威。
<br/>　　四重龙脉融龙骨，需要大量龙脉之气，或者龙族龙血，融进自己脊骨之中，化成无上龙骨。五重星辰入神窍，人体三百六十五主窍对应太古三百六十五祖星辰，需将三百六十五星辰之力炼进人体主窍之中，练成无上神窍之威。
<br/>　　六重阴阳筑神魂，用阴阳之力，练成阴阳之魂，阴阳之魂融入肉身之中，神与身合，不死不灭。七重虚空成虚境，需用虚空之力，炼入身体之中，使身体重组，空间扩大，能装下的能量也就更多，力量也就更大。
<br/>　　八重混沌衍洞天，混沌之力，演化洞天，洞天加持，神力无边。九重玄黄成不灭，玄黄二气，铸就己身玄黄不灭之威，肉身不朽。
<br/>　　因《玄黄不灭》分九重，故而，元始将人体肉身境界也划分九重，分别为：玉身境、道宫境、四极境、龙骨境、星窍境、神魂境、虚空境、洞天境、不灭境。
<br/>　　因元始如今修为已经达到大罗金仙之境，肉身也已然不弱，《玄黄不灭》第一重境只需足够能量就行，所以，元始早已将第一重练至大成，一旦催动此功法，全身玉光大涨，法相高达数万丈，洁白如玉，固若金汤，同时力大无穷。而且，随着实力的上升，这玉身法相也会越来越强。
<br/>　　至于第二重五行开道宫，这需要大量的五行宝物，但，元始并没有五行宝物，五行魔神真身他倒是有一具，但这只能让元始第二重小成，想要大成，还需要顶尖的五行宝物。所以这需要元始出去寻找。
<br/>　　至于后面几重，都需要大量的天才地宝，故而，一时也不是那么好修炼的。
<br/>　　元始所创修炼道法的功法名为《元始无名天经》，有名，万物之母，无名，天地之始，是为道。元始，既表示元始大道，也表示自己的大名。
<br/>　　《元始无名天经》：“大道存天地，天地分玄黄，宇宙有洪荒……形上谓道，形下谓器……裂盈昃日月外，踏列张辰宿上，可为圣道……
<br/>　　夫昭昭生于冥冥，有伦生于形，精神生于道，形本生于精，而万物以形相生……
<br/>　　……非阴非阳，处于天地之间，直为人，将反与宗，尽皆归一……”
<br/>　　天地都有其本源，天地有起始，大道也有其本源，有起始，最初都会起于始，最后都会归于终。
<br/>　　就好像“道生一，一生二，二生三，三生万物……”
<br/>　　如果万物翻转，将尽皆归于初始。
<br/>　　《元始无名天经》，融万道尽归本源，尽归初，尽归元，尽归始。以元始悟万道，以元始修万道，以元始掌万道，以元始合万道，以元始御万道。
<br/>　　《元始无名天经》，并没有具体的层次，有的只是道，参悟的大道越多，修炼速度也就越快，众多大道尽归元始，元始也将衍化众道。
<br/>　　“好！我元始之大道，——元始大道！今日终于迈出最为关键的一步。今后大道坦途，吾又有何惧？哈哈哈！”元始大笑，终于走出了自己的道，是多么令人高兴的一件事。
<br/>　　有多少人还在求道路之中，又有多少人还在走别人的道，还有多少人没有道？有了自己的道，才算真正的道啊！
<br/>　　“元始……大道……什么……道？”这时，一个断断续续的声音在元始耳边响起。
<br/>　　这一声响，对于元始来说，不亚于十级大地震，因为此处山谷之中，有着天道布下的天然三才大阵的保护，并没有任何人能够进来，而此刻，却又一道声音在他耳旁响起，他却毫无所知，你说震不震惊？
<br/>　　不过，元始也是经历过大涛大浪的人了，虽然震惊，但他还是面色平静，没有自乱阵脚，而是开始搜索这道声音的来源。
<br/>　　元始定睛一看，终于发现的声音的来源。原来是三清中的太清老子啊。
<br/>　　“原来是你啊，没想到这么多年过去了，你们也开了智慧了。”元始看了看老子，再看看上清通天，发现他们经过数个元会灵气的滋养，都已经开启了智慧，已经会断断续续的交流了。
<br/>　　要知道，此刻距开天才过了数十个元会，龙族凤族等才刚刚发展，凶兽王朝之劫才刚刚过去，三清就已开启灵智，这已经领先其余大神好多步了。像女娲、冥河、鲲鹏等都还没有开启灵智呢。他们可没有元始在身边，聚集大量的灵气啊。
<br/>　　所以，太清老子与上清通天能够如此快的开启灵智，还得多谢元始。
<br/>　　“既然你们已经开启了智慧，能够简单交流，那么，趁着今日吾高兴，你大哥我就给尔等讲一番大道，能不能就此悟出大道，开启修道之路，就看你们的造化了。”元始对着面前两道元神说道。
<br/>　　要问元始为何要自居大哥之称？那是因为同属三清，元始又在两道元神之前化形，故而，元始也不顾两道元神的意见，自居大哥。要不然，让元始退居老二，那怎么可能？
<br/>　　话毕，不管他俩同意与否，元始直接开讲大道。
<br/>　　“鸿蒙孕道，鸿蒙初判而大道生。有物焉，悟道因。”
<br/>　　“吾所知道，周而不殆，可以为天地母也。”
<br/>　　“道无名，道无不明，曰其大，可为逝，曰其逝，可为远，曰其远，可为大，故曰，天大，地大，道亦大！”
<br/>　　“域中四大，万物居其一焉。灵地法，人法地，地法天，天法道，道法自然，故，自然为道。”
<br/>　　“大道以天地为烘炉兮，万物滋养成以为道。故，道无常相，无所为知道。”
<br/>　　“道者，无名也，天地母也，天地之始也……”
<br/>　　……
<br/>　　元始讲道，讲得是大道本质，而不是讲修炼之法，没有天花乱坠，没有地涌金莲，没有天女散花等现象，有的只是大道之音，平平凡凡，实实在在，返璞归真。
<br/>　　元始也不知他俩能否领悟，只是看到他俩周围吸收灵气的速度更快了，元始也没有多管，继续讲道。这一讲，就讲了上千年。
