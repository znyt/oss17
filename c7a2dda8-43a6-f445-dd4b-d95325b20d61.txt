<br/>　　十几个特种部队队员快速登上“超种马”运输直升机，在直升机起飞的过程中，直升机后舱门收起关闭，在山顶别墅上空调整了方向，朝着联合行动司令部的方向飞去。

<br/>　　“生化武器！？”
<br/>　　詹姆斯两条眉头皱在了一起，李然说得这种情况不是不可能，西伦科特在“暗月”中待过，而“暗月”掌握着这个世界上最致命的融合病毒“灭亡”Perish！
<br/>　　接过李然手中的卡片机，詹姆斯脸色严肃的转身朝着那个被当做审讯室，来审讯拉蒙的仓库走去，不管拉蒙知不知道西伦科特藏在山顶别墅地下室里的东西，詹姆斯都要撬开他的嘴，把所有他知道的跟西伦科特有关的东西从他嘴里撬出来。
<br/>　　罗昊把自己的武器交给了周立，解开战术背心交给了李然，然后朝着云熙走去。
<br/>　　云熙双手背在身后，冷艳的脸上挂着笑容，脚步轻快的朝着罗昊走来。
<br/>　　罗昊走上前张开双手将云熙抱住。
<br/>　　被抱住的云熙过了一会儿，用拳头轻敲着罗昊，羞涩道：“这里这么多人呢，你就不能换个地方？”
<br/>　　罗昊抬起头看到了安装在附近，处于工作状态的监控探头，对着监控探头竖起一个中指，随后带着云熙离开了。
<br/>　　联合行动司令部，Damien中校拍了拍手，对周围人说道：“好了，好戏看不到了，大家抓紧时间工作。我可不想在这个毒虫到处乱飞的地方继续待下去了。”
<br/>　　云熙知道厂区什么地方安装了监控探头，哪里又是监控探头的死角，带着罗昊走到一处没有被监控探头监控的地方，靠在一堆堆积在地上的杂物上，看着鱼肚白的天空。
<br/>　　罗昊轻轻叹了口气，还在为西伦科特从自己眼前逃跑的事情而耿耿于怀。

<br/>　　原本罗昊也只是一句玩笑，可是现在云熙贴在他耳边说得话，倒是真把他内心的火焰给撩拨起来了，罗昊眼神灼热的看着自己身边的女人。

<br/>　　罗昊赶紧追上去，一把拉住她，捏了捏她的鼻子，说道：“怎么还跟孩子一样，动不动就耍小脾气。”
