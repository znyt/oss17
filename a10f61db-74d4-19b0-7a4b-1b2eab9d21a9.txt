　　第201章
<br/>　　大家久久不肯散去，这些汉子死都不怕，却依旧害怕分离。
<br/>　　程大雷连打带骂，才算是将大家赶走，时间紧，实在无时间一一道别。这些人将由黄三元带领着回到山寨，分了山寨中的财宝干粮，从青牛山经黑石城离开。那里如今是戎族的地盘，千余人聚在一起，目标太大，必须散开，之后各自寻找安身之地。
<br/>　　如果行动够快，还是来得及走脱的。
<br/>　　之后，程大雷身边只剩下徐神机、秦蛮、林少羽、刘关张、赵子龙，林少羽、林冲九人。
<br/>　　“大当家，我和你一起走。”徐神机。
<br/>　　“嗯。”程大雷点点头，目视着郭烦人：“道长，今日多谢搭救，你们也快离开吧，和我们在一起，不太安全。”
<br/>　　郭烦人冲程大雷施了一礼，道：“青山不改，绿水长流，老道看得出来，他日帝国，定还能听见程当家的名字。”
<br/>　　“哈哈，我当这是临别祝福了。”程大雷骑在马上，放声大笑。
<br/>　　那小道士看着程大雷，似乎奇怪他在这种情况下还能笑得出来。
<br/>　　程大雷也多看了他两眼，婴儿肥一张脸，头上挽着发髻，虽然脸不知道多久没洗了，但模样看上去很清秀。
<br/>　　“这是小徒红尘了，他日江湖上遇见，还请程当家照料一二。”
<br/>　　郭烦人拱拱手，带着小道士红尘了缓缓离开，那书着『看破人间』四个大字的白幡在空中晃啊晃。
<br/>　　“大当家，咱们往哪里走？”徐神机。
<br/>　　“向西。”程大雷伸手一指。
<br/>　　其他人向北逃，程大雷就要往西走。对杨龙停来说，程大雷自然比其他山贼重要，若和大家一起走，只会把官兵引过去。
<br/>　　咚咚咚！
<br/>　　正在此时，迅疾的马蹄声响起，只见齐德强率领数十骑快马奔来，人人身着铠甲，手持长戈，脸上带着凶悍的杀气。
<br/>　　“程大雷……”
<br/>　　齐德强快马奔来，手紧紧摁在刀柄上，双眼盯着程大雷灼灼如火。
<br/>　　“老齐，怎么了？”程大雷一脸懵懂。
<br/>　　“没，没什么。”齐德强干笑一声，握刀的手放开：“老程，你怎么这么慢，主公派我来接你。对了，怎么只有几位弟兄，其他山寨的兄弟呢？”
<br/>　　“嗨，在山寨打扮呢，都是没见过世面的粗人，把他们激动得，昨天一晚上没睡觉，想着今天要穿什么衣服。我这不是怕杨龙停等得着急么，我们先过来。”
<br/>　　“老程……对主公，最好不要直呼其名。”
<br/>　　“喔喔，怪我，怪我，应该是幽州王，不，是我家主公，以后咱们都是自己人。”
<br/>　　齐德强呵呵笑着，道：“既然咱们碰上了，就和我一起走吧，主公等着你呢。”
<br/>　　“一起走，一起走。”程大雷笑笑，忽然道：“老齐，我今天头一次见主公，准备送上一柄剑，是从戎族那里缴获的，好像还是什么王子用的，我其实也没什么见识，你帮我看看这剑好不好，合不合适送给主公？”
<br/>　　“什么剑啊，若是戎族王子用的，当应该不错。？”
<br/>　　齐德强拍马向前，凑了脑袋过来。
<br/>　　程大雷手摁剑柄，锵地一声，匹夫剑出鞘，干净利落的刺出。
<br/>　　“直来直往剑。”
<br/>　　齐德强脖子上出现一个血洞，随之鲜血喷涌，失神的睁大眼睛，嘴巴下意识张大。
<br/>　　“好，好剑！”
<br/>　　噗通，齐德强跌落马匹。
<br/>　　“杀！”
<br/>　　那里还用程大雷吩咐，在程大雷和齐德强胡说八道的时候，其他人心里都憋着一股火，此时程大雷一动手，九人同时出手。
<br/>　　秦蛮，林冲，赵子龙，林少羽，刘关张等人，又有都不是省油的灯，同时出手，当真如杀鸡宰狗一般，瞬间便将对方骑兵小队攻破。
<br/>　　“走啦，向西。”
<br/>　　程大雷大吼一声，一斧头将攻向徐神机的敌人砸下马匹，快马加鞭，向西奔去。
<br/>　　西面有什么？
<br/>　　有天下第一城的京城，有成片的草原，有荒漠，有山林，也有数不尽的追兵。
<br/>　　程大雷成功把追兵吸引到西面，自己却面临巨大的压力，直到三天后，他们还没有走出幽州界。
<br/>　　和其他人已经失散，现在程大雷身边只有徐神机，秦蛮二人，被官兵围在一座山上。
<br/>　　秦蛮右臂中了一箭，实力大减，程大雷身上也挂了彩，倒是徐神机安然无恙，活蹦乱跳着很有精气神。
<br/>　　“不知道少羽几个怎样了，有没有遇到危险。”秦蛮将草药嚼碎了，糊在伤口上。
<br/>　　“官兵都是冲着我们来的，少羽几个应该都逃脱了吧。”徐神机道。
<br/>　　几人窝在一个潮湿的山洞中，以野果为食。
<br/>　　程大雷笑笑，道：“秦大哥，让我看看你的伤怎样？”
<br/>　　“没什么大碍，我撑得住。”
<br/>　　突然，程大雷一记手刀敲在秦蛮脖子上，秦蛮一怔，两眼一搭晕了过去。
<br/>　　若是从前，程大雷怕是挨着秦蛮都不能，但现在他的实力已是顶级，并不弱于秦蛮。
<br/>　　“大当家，你……”徐神机一惊。
<br/>　　“其实，他们是冲我来的，我走了，你们也就安全了。”程大雷在山洞中立起身。
<br/>　　“大当家，你莫做傻事，咱们一起走，未必冲不出去。”
<br/>　　“等我他们引开后，你们找机会走。”
<br/>　　不等徐神机说什么，程大雷一巴掌将他拍晕了。
<br/>　　手提大斧，腰间匹夫剑，一匹瘦马向山下官兵展开冲锋。
<br/>　　“吃我杀呀！”
<br/>　　“不好太来了。”
<br/>　　“围住他，围住他。”
<br/>　　“放箭，放箭！城主说了，生死不论，死的也行。”
<br/>　　剑与箭在空中相撞，那决绝的剑意寒彻夜幕，兵器碰撞声不绝于耳。
<br/>　　“不好，王将被他斩了！”
<br/>　　“柳将来了，这下好了，柳将一定能杀了他。”
<br/>　　“啊，什么，柳将也被他杀了。”
<br/>　　“快追，快追，他闯出去了。”
<br/>　　如狼似虎的追兵在其身后紧追不舍，程大雷纵马狂奔，闯出重围，一路引着追兵西下，逃到一片山林中。
<br/>　　瘦马忽地一声长嘶，咚地一声倒地，口中吐出白沫。程大雷被摔得跌下来，在地上翻了几下才稳住身子。
<br/>　　一路丝毫不爱惜马力，瘦马能撑到现在，已经是不易。程大雷拍拍它的脑袋：“老伙计，你自己逃命去吧，若我不死，咱们当还能再遇见。”
<br/>　　说罢，程大雷提剑上山。
<br/>　　瘦马昂首，眼神中若有不舍。
