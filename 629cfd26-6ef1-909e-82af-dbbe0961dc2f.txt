        与傅夜沉确定恋爱关系后，这是童筱颖第一次跟他约会。
<br/>
<br/>        虽然，彼此约定的时间还未到，但是童筱颖却提前早到了。
<br/>
<br/>        那天之后，他将她送回了私宅，而他却依旧住在了傅氏集团大厦的总裁办里。
<br/>
<br/>        因为，她决定，要在他二十七岁生日这天，也就是今天，亲自跟他一起去民政局，一起拍结婚证照，一起领结婚证。
<br/>
<br/>        童筱颖站在路边，左顾右盼，嘴角微扬，心里满是欢喜和期待。
<br/>
<br/>        今天，她可是精心打扮了一番，还化了淡妆才出门的，精致的浅粉色轻纱上衣，下配着卡其色A字短裙，粉色蕾丝发带缠着的长发及腰，让她整个人看起来清爽可人。
<br/>
<br/>        “快看！那个男人是不是大明星啊？”
<br/>
<br/>        “好帅！”
<br/>
<br/>        “我觉得他好浪漫！又是跑车又是紫玫瑰！”
<br/>
<br/>        “看起来很有钱诶！”
<br/>
<br/>        “这世上居然真的有又帅又浪漫又有钱的男人！”
<br/>
<br/>        “当他女朋友，一定很幸福！”
<br/>
<br/>        身旁路过的女人们，惊赞声不绝于耳，童筱颖循声望去。
<br/>
<br/>        只见路边不知何时停着一辆超级拉风的红色跑车，跑车的后座上堆满了昂贵的紫色玫瑰花，而驾驶座上的男人，脸上虽然戴着墨镜，却遮不住他帅气冷峻的脸庞。
<br/>
<br/>        他穿着与她的衣服颜色极其相配的浅粉色休闲衬衫，领口处的两粒纽扣未扣，随意又不失优雅，反倒让他多了几分青春的气息。
<br/>
<br/>        童筱颖一眼望去，就觉得这男人很眼熟，但心里又觉得应该不是她要等的那个人。
<br/>
<br/>        虽然，她喜欢浅粉色，但大哥应该不会喜欢浅粉色吧！
<br/>
<br/>        然而，当那位大帅哥从跑车的驾驶座上下来，绕过车头向她走来的时候，她惊愕得下巴都快要掉了。
<br/>
<br/>        真、真、真的是大哥？！
<br/>
<br/>        他这一抹颀长高挺的身影走过来，卡其色休闲裤包裹着宛如模特般的大长腿，更是教人无法将视线从他身上移开。
<br/>
<br/>        “傻丫头，看呆了吗？”傅夜沉摘下脸上的墨镜，抬起手来，在童筱颖的鼻梁上轻轻地刮了刮，“是你老公我！”
<br/>
<br/>        平日里看惯了他西装革履，没想到他现在这样与她穿情侣装的风格，竟燃烧起了她十足的少女心。
<br/>
<br/>        她跟他并未事先约好，没想到今天会这么默契。
<br/>
<br/>        “大哥！”童筱颖惊喜不已。
<br/>
<br/>        傅夜沉发出迷人的轻笑声，一掌探向她脑后，将她拥近，低头快速在她唇上落下一吻：“乖，叫老公。”
<br/>
<br/>        童筱颖身心怔了怔，一只手下意识地轻抚着被他吻过的唇瓣，嘴角却完全不受控制地往上微扬，撒娇道：“才不要！”
<br/>
<br/>        傅夜沉也不跟她计较，其实，更多的是他已经打心底地默认了她对他的这个专属昵称。
<br/>
<br/>        他一手搂向她的后腰，将她带往副驾驶座旁，很绅士地为她拉开车门，朝她比了个邀请的动作。
<br/>
<br/>        童筱颖乖巧地坐进了副驾驶座，目光一直落在傅夜沉的身上，为他心跳加速。
<br/>
<br/>        却不知道，此时此刻的她，俨然成为了路人们眼中令人最羡慕的女人。
<br/>
<br/>        今天，他们可谓是民政局里最拉风的一对领证的夫妻。
<br/>
<br/>        给他们俩办证，还是第一次给他们办证的那个男同志。
<br/>
<br/>        领到证后，童筱颖看着捧在手心里的红本本，心里甜丝丝。
<br/>
<br/>        明明今天是他的生日，但是她却比他还要开心。
<br/>
<br/>        领完证后，童筱颖便让傅夜沉带她回私宅，因为，她准备了生日惊喜给他。
<br/>
<br/>        “这……”当傅夜沉回到私宅后，目瞪口呆地望着餐厅里的烛光、鲜花、蛋糕，还有飘在天花板上的彩色爱心氢气球，以及拴住氢气球的绳子下还吊着“和生日快乐”相结合的中英文，顿时让他感觉自己像是进了幼稚园。
<br/>
<br/>        他倒是忘了，筱颖比他小了整整六岁。
<br/>
<br/>        也就是说，他二十岁的时候，筱颖才十四岁。
<br/>
<br/>        所以，他脑子里装的东西，与她脑子里装的东西，完完全全是两个世界里的东西。
<br/>
<br/>        怪不得筱颖总是说，他跟她之间有两条沟，今天总算是见识了。
<br/>
<br/>        而童筱颖却在满心憧憬地看着傅夜沉，等待着他的下文或者是他应该要对她表示感谢什么的，结果，他只是上上下下地打量着这里的一切，不发表任何意见。
<br/>
<br/>        在用眼神确定出他似乎是不大喜欢这些东西后，童筱颖双手叉腰，按捺不住地先发制人：“大哥，这些是我一个人亲手布置的诶！不管是气球还是气球下的字，还有生日蛋糕、牛排、披萨、水果沙拉、红酒，全都是我自己亲手做的噢！所以，我不许大哥说不喜欢，也不许大哥说这里被我弄得很幼稚，还有——唔？！”
<br/>
<br/>        她警告的话才说到一半，傅夜沉突然弯下腰，两条长臂一把紧紧地拥她入怀，心情略微激动地覆上了她那喋喋不休的小嘴。
