
<br/>　　<br/>　　    第二天早上，贺然醒来的时候还记得昨晚那个梦呢，他也没当回事，只以为是对方给自己的印象太深刻了，就在梦里外又回味了一下，这种白天发生晚上做梦的状况一般人都碰到过，没什么可奇怪的。<br/>　　<br/>　　    这天上班，施工队伍开始准备爆破山体最前面的那一段，整个白天在做着最后的爆破总结，然后就得点火了，所以务必控制好爆炸的威力和爆炸过后碎石的走向，免得崩碎了之后飞的太远在误伤了人，一直到晚上才算安排完，然后又回到了驻地进行总结，准备明天实施爆破山体。<br/>　　<br/>　　    晚间，开了会后，他们三人就开车回住的地方，上了车后贺然发觉了座位上放着的针线盒，就跟刘洋和徐伟说道：“把这事给忘了呢，今天不是说给人送东西的么，忙了一天才有空，我都没想起来”<br/>　　<br/>　　    刘洋无语的说道：“大哥，工程要紧还是送这个要紧啊？等等的吧，明天爆破完后有空再打听下那个村子在哪”<br/>　　<br/>　　    徐伟撇嘴笑道：“就送个东西，急啥，你今天不送还能要你命不成？”<br/>　　<br/>　　    这个小插曲其实三人谁也没当回事，总觉得晚送也不是个事，于是就这么给掀过去了。<br/>　　<br/>　　    隔天，爆破山体施工地，经过一上午的最后测算和统计，觉得万无一失了，于是开始实施爆破，施工的队伍们都躲到了安全地带，负责爆破的工程师开始倒数计时。<br/>　　<br/>　　    刘洋拍了下贺然的肩膀，小声说道：“我他么的，怎么感觉右边眼皮好像跳了好几下呢”<br/>　　<br/>　　    “你别瞎说，右眼跳灾，你说点好话不行么”贺然小声说道。<br/>　　<br/>　　    “三，二，一……准备，起爆！”<br/>　　<br/>　　    “轰！”“轰，轰”几声巨响之后，山体中间都被炸开了，一阵地动山摇后才算平稳下来。<br/>　　<br/>　　    贺然就起身说道：“今个的活完事了，一会刘洋，徐伟你俩留下跟我……”<br/>　　<br/>　　    贺然一句话没说完，从他前方一块拳头大小的石头就急速的飞了过来，几乎是擦着他的耳边砸了过去，紧接着贺然就听到身后传来“噗”的一声闷响，随即一股温热的液体就落到了他的脖子上。<br/>　　<br/>　　    贺然下意识的就用手摸了下脖子，然后低头一看手上沾着的白色的粘稠物，有点腥味，他狐疑茫然的转过脑袋，就看见而他相隔不到一米远的刘洋半边脑袋都给砸瘪了剩下的半边头颅里还嵌着块拳头大小的碎石。<br/>　　<br/>　　    这工夫也就三五秒的时间，随即刘洋的尸体朝后仰头“噗通”一下倒在了地上。<br/>　　<br/>　　    施工队里一片寂静，足足宁静了能有半分钟，惊吓的人声才传了过来，施工领导连忙喊道：“是爆炸的碎石，趴下，趴下啊……别动了”<br/>　　<br/>　　    施工队的人全都趴在了地上，因为这时候谁都怕还会再有落石砸过来，贺然离刘洋的尸体最近，他趴在地上眼睁睁的看着刘洋的脑袋里，脑浆混合着血液缓缓的流淌到了地面上，他剩下的那半边脑袋上，一只眼珠子还在睁着呢。<br/>　　<br/>　　    几分钟后，感觉差不多没事了，地上趴着的人才起来，全都围到了刘洋的尸体旁边，贺然和徐伟四目茫然相对，两人一脸震惊，平日里他们三个几乎天天都在一起，一直朝夕相处，不久前还和他们一起工作的同事就这么死了，他俩谁都难以接受。<br/>　　<br/>　　    施工暂时停了，工地最怕出现事故，特别还是有关人命的事故，就不得不先暂停然后调查原因了。<br/>　　<br/>　　    其实原因很简单，就是爆破山体的时候一块碎石巧合的飞了过来把人给砸死了，这都是明摆着的，以往中铁几个局在施工的时候也不是没有出过状况，几乎每年都有人员伤亡，所以处理起来也不是多么复杂，毕竟这是国字头的单位。<br/>　　<br/>　　    工程当天停了，全部回到指挥部开会，讨论事故问题，而这一整天徐伟和贺然都还处于无法接受的状态中，晚间睡觉的时候他俩都久久难眠，始终无法相信刘洋就这么死了。<br/>　　<br/>　　    这天过后，第二天，由于工程停了人暂时就都闲了下来，贺然就跟徐伟商量道：“老刘走了，这时候咋伤心都于事无补了，他家里还有老婆孩子呢，咱们到时候多扔点钱尽尽心意吧”<br/>　　<br/>　　    徐伟叹了口气，沉闷的抽着烟说道：“这到行，拿多少都没问题，我他么就是挺难接受的，小贺啊你说这人咋这么脆弱呢，他死前还跟咱们说话呢，然后说死就死了……”<br/>　　<br/>　　    “世事难料，可能他命里该着有此一劫吧”贺然掐了烟头，说道：“咱俩下山一趟，去镇子里，我们那都有规矩，人死了之后晚上得烧纸，守灵，老刘的家人也不在，尸体又送到县里的殡仪馆去了，咱们也去不了，就在山上给他烧纸吧？”<br/>　　<br/>　　    “行，走吧！”<br/>　　<br/>　　    贺然和徐伟跟领导说了一声，要去山下的镇子里买点纸钱回来，施工经理也没拦着他们，烧纸这都是正常事，人都死了你不得求个心安么，就让他俩下去了。<br/>　　<br/>　　    皮卡从驻地开出来，半个多小时后，顺着山路就来到了一段悬崖峭壁附近，这就是标准的盘山公路，一侧是山坡，一侧是悬崖。<br/>　　<br/>　　    皮卡的车速不快，开这种路基本就保持在三四十左右，防止发生侧滑什么的。<br/>　　<br/>　　    车子在盘山路上开了没多久，皮卡突然之间就熄火了，这辆车在山地里跑了好几年，车况也不太好，说趴窝就趴窝，让人十分防不胜防。<br/>　　<br/>　　    “小贺，你下去推一把，我挂上档位别一下，咱俩试试看能不能再开起来，不行的话，那就得让队里开车过来给拖回去了”<br/>　　<br/>　　    “嗯，你慢点啊，脚下带着点刹车……”<br/>　　<br/>　　    贺然从皮卡上下来，在后面推着车厢，徐伟一觉踩着离合把档位推到了二挡上，然后一手拧着钥匙门，就冲着外面喊道：“你推吧，速度上来后我就打火了，尽量使劲啊”<br/>　　<br/>　　    “唉，妥，你放心吧！”<br/>　　<br/>　　    贺然回了一句，然后低着脑袋，两手用力的推着车身，渐渐的皮卡就走了起来，徐伟一脚一脚的踩着离合，等车速上来之后，手就拧了下钥匙门。<br/>　　<br/>　　    “咔嚓，咔嚓”钥匙门空响了几下。<br/>　　<br/>　　    徐伟接着喊道：“不行，使劲，再使劲点推”<br/>　　<br/>　　    “啊！”贺然卯足了力气，用力的推着皮卡，同时脚下蹬着地面，速度起来后，随即就听见车子发动机“嗡”的一下就被发动了。<br/>　　<br/>　　    但同时，贺然也猛然感觉自己两手一空，他踉跄着往前冲了几步然后顺势抬头，就惊恐的发现，皮卡车的车头居然朝着悬崖的一侧歪了过去，直接就失控了一头扎下了悬崖。<br/>　　<br/>　　    变故，来的太突然，太措手不及了。<br/>　　<br/>　　    悬崖下方，似乎隐约传来了徐伟绝望的一声惊呼。<br/>　　<br/>　　    几秒钟后，悬崖底下连续“咣当，咣当”传来了几声巨响，最后一点动静都没有了。<br/>　　<br/>　　    贺然当场懵逼了，人呆立着一动也没动。
