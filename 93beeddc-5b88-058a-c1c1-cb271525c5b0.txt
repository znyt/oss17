        而傅夜沉完全不在意苏漫雪的话，他傅夜沉是什么人，岂会轻易被一个女人的三言两语给气到？
<br/>
<br/>        童筱颖从楼上下来，回到厨房里继续帮刘婶整理内务时，刘婶见童筱颖脸都红了，便关心地问道：“大小姐，你的脸这是？”
<br/>
<br/>        刘婶在一旁问话，而童筱颖的脑海里却在想自己和傅昼景的事情。
<br/>
<br/>        刚刚苏漫雪攀附在傅夜沉身上时的那个样子，实在是太撩人了。那前露美胸，下露美腿，姿势妩媚，十足的秀色可餐。
<br/>
<br/>        童筱颖忽然觉得，自己是不是太平淡无趣了，所以昼景才会对她没“兴趣”。
<br/>
<br/>        上次她之所以能成功，完全是因为那瓶香水里有催情的成分所致，而且还惹昼景不开心，结果那事儿只进行到一半。
<br/>
<br/>        那么下次呢？
<br/>
<br/>        下次，她又该如何“服侍”昼景呢？
<br/>
<br/>        “大小姐，你在想什么呢？”刘婶又拉高了分贝，扯着嗓子问道。
<br/>
<br/>        童筱颖这才回过神来，对刘婶赧然一笑：“没、没什么！”
<br/>
<br/>        “哎，你这好端端的，怎么就嫁给了二少爷呢？”刘婶手里拿着抹布，一边抹这灶台，一边感慨道。
<br/>
<br/>        若不是苏漫雪多嘴念叨，她这给大少爷在私宅里当管家的还真不知道，童筱颖现在是傅昼景的合法妻子。
<br/>
<br/>        童筱颖一脸茫然地反问：“我为什么就不能嫁给昼景？”
<br/>
<br/>        “也没为什么，二少爷也挺好的。自幼乖巧懂事，但就是不太讨老太爷的欢喜。”刘婶叹息了一声。
<br/>
<br/>        童筱颖却情不自禁地抿嘴微笑：“讨我欢喜就行了。别人不喜欢，我喜欢。”
<br/>
<br/>        “你这丫头。”刘婶忍俊不禁地应了声，“你们夫妻俩就好好过日子吧！”
<br/>
<br/>        筱颖这丫头，对二少爷真的是爱得纯粹、爱得简单。只是可怜了她家那位大少爷……
<br/>
<br/>        刘婶虽然心知肚明，但是并未多嘴说一些不该说的话给童筱颖听。
<br/>
<br/>        童筱颖给刘婶帮完忙从厨房里出来，迎面遇上一个外边穿着白马褂，里面穿着黑衬衫佩戴着领带，胸口口袋上还挂着单片眼镜的男人。
<br/>
<br/>        这个男人，童筱颖见过。
<br/>
<br/>        上次她昏迷不醒被傅夜沉带到这里时，这个男人当时也在场。
<br/>
<br/>        “哟！我出差办事才多久，咱家小沉沉就找女人回来了！”连华生打趣地说。
<br/>
<br/>        刘婶也跟着从厨房里出来后，看到连华生，欠了欠身，问候道：“连医生回来了”
<br/>
<br/>        “刘婶，赶紧跟我介绍介绍这位小美女吧！”连华生饶有兴趣地打量起童筱颖来。
<br/>
<br/>        刘婶接着介绍道：“这位是童筱颖小姐，傅昼景少爷的新婚妻子，傅家的二少奶奶。大小姐，这位是连华生，连医生，大少爷的私人医生。”
<br/>
<br/>        “连医生，你好。”童筱颖微微颔首问候。
<br/>
<br/>        连华生朝童筱颖凑近了一步，微微俯身，在她脖颈旁嗅了嗅。
<br/>
<br/>        童筱颖不好意思地后退，躲开连华生的靠近。
<br/>
<br/>        连华生会心一笑，口无遮拦地说：“我在你身上闻到了‘处子的芬芳’，这股熟悉的味道……嗯？你是‘夜惑’里的那个小谢？”
<br/>
<br/>        “啊？”童筱颖怔愣地瞪着眼睛看着连华生。
<br/>
<br/>        他是怎么知道的？
<br/>
<br/>        此时，傅夜沉正从楼上下来，无意间听到连华生这么一说，惊怔地停下了脚步。
<br/>
<br/>        “你认错人了……”童筱颖赧然一笑，心里却在纳闷这个男人是怎么回事，为什么会认出她来。
<br/>
<br/>        难道，上次她在这里过夜的时候，他摘掉了她的假面眼镜吗？
<br/>
<br/>        童筱颖心里揣测着，有点儿惴惴不安。
<br/>
<br/>        连华生抬起手来，竖起食指摇了摇：“NO、NO、NO，我的嗅觉非常灵敏。每个人身上，都有一种独特的气味。你身上的气味，我记得。是一种来自少女与药草的香味。如果，我没猜错的话。你家应该是开药店的，又或者，你在这之前，常年与药草打交道。”
<br/>
<br/>        童筱颖听他这么一说，自己下意识地抬起手臂来闻了闻，但闻到的只有衣服上的洗衣液的味道。
<br/>
<br/>        “看样子，我猜得一点都没错喽！”连华生得意地笑了笑，“你真是‘夜惑’里的那个小谢。”
<br/>
<br/>        “嗯。”童筱颖微微点头后下意识地应了声，并一脸崇拜地看着连华生，好奇地问，“那你也教教我，怎么识别人身上的气味。”
<br/>
<br/>        “这个我没法教，这是我与生俱来的能力。嗅觉非常灵敏。”连华生笑道。
<br/>
<br/>        而站在楼梯上的傅夜沉不禁欣慰地嘴角微扬。
<br/>
<br/>        原来……
<br/>
<br/>        他从始至终都未吻错人，虽然他没有连华生那敏感的嗅觉，但至少，他对筱颖的感觉，从始至终都是对的。
<br/>
<br/>        他一心向着她，那种直觉，总让他身心都在向她靠拢。
<br/>
<br/>        童筱颖的一颦一笑，全映在傅夜沉的眼中，甚至深深地陷入了他的心里。
<br/>
<br/>        这世上最痛苦的爱情，或许就是他这种，爱而不得，爱而无法诉说，爱而无可奈何……
<br/>
<br/>        此时，苏漫雪跟在傅夜沉身后见楼下来了客人，立即上前挽住傅夜沉的胳膊，装出一副很亲昵的样子，拉着傅夜沉继续往楼下走。
<br/>
<br/>        “哎呦喂！家里这是来了客人吗？长得可真俊！刘婶，还不给客人泡茶。”苏漫雪就像一个女主人一样吆喝了起来。
<br/>
<br/>        “是。”刘婶一看到苏漫雪就翻着白眼，转身进了厨房。
<br/>
<br/>        连华生循声望去，看清楚苏漫雪的样子后，煞是吃惊，又见她十分亲密地搂着傅夜沉，心里揣摩着这个女人可能就是傅夜沉跟他经常提起的“苏漫雪”。
<br/>
<br/>        只不过，这苏漫雪和童筱颖倒是长得很像双胞胎。
<br/>
<br/>        “想必，这位美女，一定是夜沉经常跟我提起的漫雪小姐了。”比起对童筱颖的自然挑逗，连华生对苏漫雪倒是非常彬彬有礼地问候道。
<br/>
<br/>        苏漫雪顿时感到受宠若惊，欣喜地反问：“是真的吗？夜沉经常跟你提起我呀？”
<br/>
<br/>        “当然，你去韩国旅游的那段日子，夜沉可是饱受相思之苦！还为了你……”连华生说着说着，眼睛又看向了童筱颖，随即欲言而止。
<br/>
<br/>        总不能揭了他兄弟的底吧！
<br/>
<br/>        说“正宫娘娘”不在家的日子里，他傅大少爷在“夜惑”里和歌女小谢暧昧上了！！
<br/>
<br/>        “为了我什么？”苏漫雪好奇地追问道。
<br/>
<br/>        连华生咧嘴一笑，将目光落在了傅夜沉身上。
<br/>
<br/>        傅夜沉一脸淡定，似乎压根就不怕他说漏嘴。
<br/>
<br/>        兄弟嘛！
<br/>
<br/>        没办法，该帮忙遮的，就还是遮一遮，比较好！
<br/>
<br/>        “为了你找我喝闷酒啊！向我吐露对你的思念之情。”连华生接着说道。
<br/>
<br/>        童筱颖听连华生这么一说，心里更加笃定了自己之前的猜想。
<br/>
<br/>        傅夜沉之前之所以对她表现得格外亲热，是因为把她当苏漫雪了。
<br/>
<br/>        时间的先后顺序，童筱颖自然是不知道，但她看到傅夜沉是真的很爱苏漫雪，心里也算是有所欣慰了。
<br/>
<br/>        虽然苏漫雪称不上是个什么好女人，但是她作为她曾经的闺蜜与好姐妹，苏漫雪能幸福，她自然也替她感到开心。
<br/>
<br/>        傅夜沉似乎也不打算跟连华生解释，就任由他误会自己和苏漫雪之间的关系。
<br/>
<br/>        大家越是认为他真正爱的人是苏漫雪，那么对筱颖来说，就越是一件好事。
<br/>
<br/>        至少，他不会给她带来困扰；至少，他还能以“大哥”的身份，站在她身后，护她周全。
<br/>
<br/>        在这里，她苏漫雪明明是当局者，却把傅夜沉的事情看得通透。
<br/>
<br/>        傅夜沉倘若愿意把她当童筱颖的替身，她或许还有翻身的机会，但是，他没有，他只是拿她当幌子，让她一点想要得到他的心的机会都没有。
