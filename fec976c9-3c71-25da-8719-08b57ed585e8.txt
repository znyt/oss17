　　挑了一下眉毛，刘哔也没有开口拒绝，反正他本身就是想出来转转，看看能不能瞎猫碰死耗子，接到什么任务，如果任婷婷想要陪同，也是无所谓的事情。
<br/>　　第一次接到这样的连环任务，刘哔也想要借此机会对任家多一些了解，毕竟在原著中，对于这个家族也没有过多的介绍。
<br/>　　怀着这样的心思，刘哔带着任婷婷慢悠悠的走着，虽说是要买胭脂水粉，但任婷婷却没表现出一点着急的样子来。
<br/>　　“任小姐，近期任家有发生什么奇怪的事情没有？”闲聊了几句之后，刘哔似是无意，开口打探了起来。
<br/>　　自从在咖啡厅接到任务之后，他仔细回想了很久，却仍旧没有发现任家有什么不同的地方，更找不到其他能够连接后续任务的地方。
<br/>　　虽然不能说一无所知，但刘哔手中掌握的资料，也寥寥无几，根本不可能从中找出什么线索。
<br/>　　“奇怪的事情？不知道啊，我刚从省城回来，对家里的事情还不太清楚。”任婷婷侧了侧头，有些疑惑的回答道。
<br/>　　她有些好奇，为什么刘哔突然问起了这种事情。
<br/>　　“哦，没事，就是随口问问，你不是要买胭脂水粉吗？先陪你一起去吧。”微微颔了颔首，不想让任婷婷心中起什么怀疑，刘哔也没有继续在这个话题上继续纠缠下去。
<br/>　　他倒是忘了有这一茬，任婷婷刚刚回来，看来从她这里是得不到什么消息了，不过问题不大，反正七天之后，后续的任务就会自行显示出来。
<br/>　　不解的看了刘哔一眼，任婷婷也没将此事放在心上，听到刘哔肯陪她去买胭脂水粉，自然也兴奋不已。
<br/>　　因为不经常回来，故此她在这里也没有几个朋友，并且在她的眼中，这里的人大部分都观念老旧，难得有人肯陪她去买这些化妆用的东西。
<br/>　　但苦恼的是，因为对这里不太熟悉，她并不知道镇上的胭脂店在哪里，而今天第一次出门的刘哔，对此更是一筹莫展。
<br/>　　无奈之下，任婷婷只能找人询问，在得知了确切位置之后，拉着刘哔兴冲冲的往胭脂店里跑。
<br/>　　路旁不远处的一家胭脂店，刘哔被任婷婷拉着进了门，抬头便看见了秋生，正一脸震惊的看着两人。
<br/>　　“师弟啊，今天怎么有空来我这里了，这位是？”呆愣了几秒，秋生回过神来，主动开口，和刘哔打了一声招呼，但是两颗眼珠子，却止不住向任婷婷的方向瞥。
<br/>　　平常满镇子里乱窜，美女他也算见过不少了，但像任婷婷这么漂亮的，还真的不是很常见。
<br/>　　“哦，她呀，是任老爷的千金，今天我陪师傅出来吃饭，没什么事了，就陪她出来买点东西。”挥了挥手，刘哔随口回应道。
<br/>　　胭脂店里总是有一股若有若无的气味，有些像是劣质香水的味道，让他禁不住皱起了眉头。
<br/>　　在前世的时候，他就对这种味道十分敏感，哪怕只有一点很轻微的余香，都会让他鼻子瘙痒，如同过敏一般。
<br/>　　如今变成了丧尸，这个毛病却也没有改变，反而随着体质的提升，感知也更加敏锐了。
<br/>　　“哦，那随便看，随便看，既然是任老爷的千金，那我给你打一个八折好了。”从柜台里面出来，秋生殷勤的凑在任婷婷身边。
<br/>　　脸上带着笑容，任婷婷拿起货架上的商品，仔细观摩着，花费了好一阵的功夫，挑选出了几件中意的东西。
<br/>　　似乎是想要找刘哔征询什么意见，任婷婷手中拿着几盒胭脂，颇为兴奋的转过头来，却发现刘哔已经离开了店内，瘪了瘪嘴，有些不太开心。
<br/>　　连个招呼都不打，直接走了，这未免太没有风度了！
<br/>　　“小姐，你的眼光可真是好啊，这几件货，都是刚进来的新品，效果都很不错呢。”看到任婷婷有些气恼的眼神，秋生眼珠一转，凑上前来，主动拉起了话题。
<br/>　　听到夸赞，任婷婷顿时眉开眼笑，和秋生两人攀谈了起来，一时间，把刘哔抛在了脑后。
<br/>　　……
<br/>　　从胭脂店里走出来的刘哔，也没有继续在门口等下去的心思，任婷婷刚刚回来，肯定什么事都不知道，在这里和她浪费时间也没什么意义。
<br/>　　但是初来乍到，他和这个城镇的居民也不太熟悉，只能漫无目的的在街上闲逛，一直到太阳落山，都没有什么收获。
<br/>　　看了看逐渐暗下来的天色，刘哔轻叹一声，街上的一些商贩都已经开始收摊了，九叔之前也叮嘱过，让他晚上早些回去，似乎还有什么事情。
<br/>　　夏末的晚上还是有些冷的，尤其是镇子的地势偏低，四面都是一些湖泊河流，更是带来了一股阴冷潮湿的感受。
<br/>　　裹了裹身上的长衫，刘哔已经收回了中午心中的想法，此时他很庆幸，自己穿的不是短袖，否则就算身体被丧尸病毒强化过，也非得冻感冒不可。
<br/>　　走到一半的时候，刘哔却放慢了急匆匆的脚步，街边几个正在收摊的大妈，正在七嘴八舌的讨论着什么。
<br/>　　虽然距离算不得近，但变成丧尸之后，身体强度和感知都要上升了一个档次，故此他也能听得非常清楚。
<br/>　　“哎，昨天任府出事了，你们有谁知道？”
<br/>　　“任府……你说的任老爷家里？”
<br/>　　“嗨，还能有谁，听说昨天任府闹鬼了，死了好几个下人呢，连镇上的保安队长都给惊动了！”
<br/>　　“闹鬼？这个事情你是听谁说的？”
<br/>　　“我小舅子就在里面打下手，昨天和我家老头喝酒的时候说的，他说任府要出大事，说什么都不肯回去了！”
<br/>　　“我草，这么劲爆！那后来呢，警察局是怎么处理的？”
<br/>　　“我也不知道，但是听说和二十年前的一个道士有关，最近任家的人也急急火火的找人给他们老太爷迁坟呢！”
<br/>　　……
<br/>　　站在街边听了一会，直至几位大妈全部收摊离开了，刘哔方才迈开脚步，继续向前走着。
<br/>　　对于这几个中年妇女所说的，他并不敢大意，虽然在原著中，迁坟之前任发和任婷婷都一直活得好好的，但难保不会有什么意外发生。
<br/>　　为了能让任务顺利的进行，刘哔心中有些犹豫，要不要去任府一趟，好好查探一下。
