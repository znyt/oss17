　　第93章
<br/>　　山贼进城打劫，往往是悄无声息，摸黑进庄，能不杀人尽量不杀人。有机会就抓个把人出来，回头砸石头进去，告诉主家什么时间什么地点拿钱赎人。
<br/>　　像程大雷这样大张旗鼓，还没动手，就搞得全城皆知。如此，却是有史以来第一次。
<br/>　　高飞豹恨不得上去摸摸程大雷脑门：“程当家，你没发烧吧？”
<br/>　　“什么话，你才发烧了。”
<br/>　　“那你脑子可要想想清楚，贾家是能动的嘛，人家外号贾半城，半个黑石城都是贾家的，凭你们六个人最后别痴心妄想！”
<br/>　　“行啊，二爷，文化见长，连痴心妄想都会了。”
<br/>　　高飞豹瞪大眼睛，默默摇摇头：“突然不想和你说话了。”
<br/>　　这时候徐神机和刘悲醒来，徐神机道：“大当家，打听清楚了，过两天贾家娶媳妇，秦蛮三个已经提前混进去了。”
<br/>　　“好，过两天咱们去给他贺一下。”说着，程大雷冲小白狼和高飞豹道：“二位，怎样一起去看看贾家的新媳妇儿？”
<br/>　　“哼，我们还没有蠢到和你一块儿去送死。”
<br/>　　如今的黑石城可谓是满城风雨，沸沸扬扬传着一件事。事情自然是程大雷搞出来的，一夜间满城贴的都是程大雷所写的东西，想要不知道这件事还真不容易。
<br/>　　凡写在名单的上的人皆是人人自危，可也有人不当会事。程大雷虽然恶名在外，但名声这种东西自然会随着时间而流逝，旧的新闻自然也会被新的新闻取代。
<br/>　　午后，临街一座茶楼中，坐着四名青壮汉子。一壶沸茶，中间桌子上搁的正是程大雷亲手所书的传单。
<br/>　　为首一人二十出头年纪，生得皮肤黝黑，头上扣着一顶小鹿皮帽子。
<br/>　　“这蛤蟆大王……”他手指轻敲着桌面：“什么来历？”
<br/>　　“打听出来是一伙青牛山的山贼，曾靠着打败黑石城一千精兵而扬名。薛半川派人打过，铩羽而归。”
<br/>　　“黑石城的兵也算得上精兵？”男人付之一笑：“不过我倒觉得这山贼有些意思，有机会可以见一见。”
<br/>　　“小……公子，山贼有什么好见的。我打听到，这蛤蟆大王生性狡猾，又作恶多端，平素里欺男霸女，什么坏事都干！”
<br/>　　一伙人说话的口音十分奇怪，叽里呱啦也听不清他们说什么。可最后一句话却还是听清楚了。
<br/>　　旁边一个着月白长袍，腰上佩剑的人不瞒道：“喂，几个外乡人，你们知道实情嘛，不知道不要瞎说好不好。”
<br/>　　男人的手下当场便要站起来，手摁住腰上的弯刀。男人却是笑了笑，挥手让手下坐下。
<br/>　　“这位公子不知怎么称呼？”
<br/>　　“某姓欧名巴字伐若，你对我直呼其名就可以。”
<br/>　　“在下呼延蛮，见过公子。不知公子为何要替替山贼说话？”
<br/>　　“所谓路不平有人铲，事不平有人管。在下本来不想多口，可实在听不得你们信口污蔑。那蛤蟆大王心怀仁义，又生得英俊潇洒，仪表堂堂。方圆百里谁不知道，有言说，平生不见程大雷，便称英雄也枉然。”
<br/>　　呼延蛮笑笑：“怎么我听说的，和公子听说的不一样。”
<br/>　　“传言么，自然有人听到的是假的，有人听到的是真的。”
<br/>　　“公子说得如此真切，莫非见过那蛤蟆大王？”
<br/>　　“虽未亲见，慕名良久。”
<br/>　　“平生不见程大雷，便称英雄也枉然，好，有机会我倒要亲眼见见他。”
<br/>　　“我想会有机会的。”
<br/>　　一伙人放下铜板后，从茶馆离开。程大雷拥着双手看了一眼呼延蛮的背影，脸上挂出古怪的笑容。
<br/>　　“这呼延蛮什么来历，看上去来头不小呐。”徐神机道。
<br/>　　“什么呼延蛮，他叫呼尔勒，戎族人，估计在戎族，身份不低。”
<br/>　　“咦，大当家怎么知道的？”
<br/>　　“难道我被神仙传艺，能掐会算的事情你没听说么？”程大雷白了他一眼：“还有，在城里叫我公子，欧巴欧公子。”
<br/>　　程大雷三人是天亮进城的，黑石城是座小城，说白了是座破城。城墙豁子到处都是，不走城门也有七八条道进城。
<br/>　　程大雷在城里逛了一圈，摸了摸黑石城的情况。晃悠到晚上，就来到贾府门前。
<br/>　　今天是贾家大喜的儿子，门前张灯结彩，宾客络绎不绝。
<br/>　　“呦呵，贾家这么大口气，敢称府，家里是出过什么大官么？”程大雷。
<br/>　　“不过是有人在城主府当个小官，现在世道乱了，以前若敢挂这两个字，就足够灭满门了。”刘悲道。
<br/>　　这时代等级森严，若是不够格，门前就不能挂『府』这个字，像苏樱家，也不过挂个『苏宅』而已。
<br/>　　程大雷三人混了进去，拣了张八仙桌坐下。事先，秦蛮、林少羽、张肥已经混进贾府，大家互相照过面，却没有招呼。
<br/>　　“老徐，你冷静一下，咱家也不是没吃的，你这样狼吞虎咽的，很丢我脸的。”
<br/>　　“废话，你手上也不慢，吃了多少天冷饭，我就不能吃顿热乎的。”
<br/>　　“哎呀呀，有失体面，有失体面。放下！那羊腿是我的。”
<br/>　　三人吃得狼狈不堪，同桌的客人稍稍和他们拉开距离，不想让大家觉得和他们认识。
<br/>　　“三位，怎么称呼，看着有些脸生啊？”
<br/>　　程大雷正啃着一只鸡腿，一抬头肉末子恨不得喷在对方脸上。
<br/>　　“你管得着么！”
<br/>　　“呃……”
<br/>　　对方自重身份，不想和程大雷这种粗鲁人打交道。
<br/>　　这时候新郎携着新娘从外面进来，过火盆，拜过天地就可以送入洞房。
<br/>　　程大雷探出脑袋往外瞧：“也不知道这新娘子长得俊不俊？”
<br/>　　大喜日子，宾客尽欢，忽然院外一声音响起：“不许成亲！”
<br/>　　咦，抢亲的来了，这是谁呀比我动作都快。
<br/>　　紧跟着一个宽袍大袖的年轻人从外面冲进来，口中大呼小叫。
<br/>　　随之，窃窃私语声便响起来，程大雷侧耳听着，很快就明白是怎么会事。
<br/>　　这冲进来的年轻人名叫罗子开，与今天的新娘定有婚约，和今天的新郎称兄道弟。后来罗家衰落，新娘与他毁婚，他的兄弟撬了他的女人。
<br/>　　“啧，嫌贫爱富，重色轻友，我都忍不住要管管闲事了。”徐神机悠悠叹道。
<br/>　　“啃你的猪蹄子，你又不是居委会大妈。”
<br/>　　
<br/>(本章完)
