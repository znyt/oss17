　　八劫古帝境界的海妖，在整个大罗天星域之中，也只有一方势力可以拥有。
<br/>　　沧海都。
<br/>　　这三名年轻男子，分属于大罗天星域的三方势力之中。
<br/>　　看到这三人现身，楚风眠都是不由的眉头一皱。
<br/>　　如果这三人都是属于五行天的圣子的话，那很有可能是五行天想要来对付楚风眠。
<br/>　　毕竟楚风眠跟五行天积怨已深，血神古帝是楚风眠所杀，还有那日炎公子等人，楚风眠杀了的五行天强者无数。
<br/>　　这些事瞒得过一时，却是瞒不过一世，被五行天察觉倒也正常。
<br/>　　可是这齐林山，沧海都的圣子，也都前来对付楚风眠，那可就不寻常了。
<br/>　　楚风眠可记得，那些齐林山弟子的交谈之中，可是要三方联合，来对付圣龙秘境。
<br/>　　莫非这三方势力，已经动手了？
<br/>　　楚风眠眼神之中露出几分冰冷，如果这三方势力开始动手，那在这赤红大陆之内，所有圣龙秘境弟子的处境，可就危险无比了。
<br/>　　而楚风眠可以说是圣龙秘境之中，最为耀眼的一名新人，楚风眠自然是首当其冲。
<br/>　　“你们为何要围攻我？难道想要不顾规矩？”
<br/>　　楚风眠看着三人，不动声色的说道。
<br/>　　大罗天星域的四大势力之中，明面上有着条约，不允许互相杀戮。
<br/>　　虽然暗地里这样的事情做的不少，但是明面上却是没有敢如此大摇大摆的。
<br/>　　“规矩？”
<br/>　　那三名年轻男子，对视了一眼，眼神之中露出了几分嘲笑之色道。
<br/>　　“楚风眠，你还信这什么规矩？看来你还什么也不知道啊，从今日起，圣龙秘境也将会彻底消失了，再也没有什么规矩了。”
<br/>　　那五行天圣子盯着楚风眠狞笑起来道。
<br/>　　“不过对于你，小天主倒是有着几分兴趣，现在可以给你一个机会，现在跪下，随我回去臣服小天主，可以让你加入到五行天之中，让你为五行天效力，饶你一条命如何？”
<br/>　　“火景，你打算将好处独吞？”
<br/>　　听到这五行天圣子的话，旁边齐林山的圣子，眉头一皱道。
<br/>　　“这小子身上奇遇众多，五行天想要独吞，只怕不好吧。”
<br/>　　沧海都圣子也是沉声说道。
<br/>　　“二位不要焦急，等这小子臣服了小天主，小天主自然有手段让这小子将奇遇交出来，倒时候此人身上的奇遇，我们三方平分就是。”
<br/>　　火景冷笑一声，看向楚风眠说道。
<br/>　　“小子，考虑的这么样了，是生是死？自己做决定。”
<br/>　　“是生是死？当然是我生，你们死！”
<br/>　　突然之间，就在楚风眠的身上传来了一阵惊天的气势，十二万九千六百滴远古战龙精血的力量，在这一刻瞬间爆发而出。
<br/>　　龙威浮现而出，在楚风眠的背后，化为了一道远古战龙的模样，八劫古帝的气息，当即爆发出来。
<br/>　　“古帝？”
<br/>　　“八劫古帝？”
<br/>　　看到楚风眠身上气息境界的一刻，这三名年轻男子的都是大吃一惊。
<br/>　　根据他们的了解，楚风眠只是一名小小的准帝，所以从一开始他们就没有将楚风眠放在眼中过。
<br/>　　可没有想到，短短几日的时间不见，楚风眠的实力已经是正式踏入了帝尊境，更是一举踏入到了八劫古帝的层次。
<br/>　　“踏入了八劫古帝，又能如何？以我们三人出手，九劫古帝都可以抗衡一二，更何况一名刚刚突破的八劫古帝。”
<br/>　　火景先看了楚风眠一眼，随即眼神之中还是露出了几分不屑。
<br/>　　他们可都是踏入了八劫古帝数百年的人物了，底蕴深厚，远不是一名刚刚突破的八劫古帝所可以抗衡的。
<br/>　　更何况他们也都是超级势力之中的圣子，奇遇虽然没有楚风眠那么惊人，可也都不少，修炼的也都是绝世功法，自然不会怕了楚风眠。
<br/>　　“小子，听说你的圣龙秘境万年不遇的天才，今日我就斩了你这天才！”
<br/>　　火景看了楚风眠一眼，刹那间出手了。
<br/>　　那齐林山圣子，沧海都圣子二人，都没有想要出手的打算，能够成为圣子，他们也都是绝世天才。
<br/>　　在他们看来，一个火景，对付楚风眠，便是足以。
<br/>　　“五行凝聚！神芒灭世！”
<br/>　　这火景出手了，五行之力在他的手心之中凝聚交织着，笼罩在他的的背后，身上的气息节节攀升起来。
<br/>　　在这火景的面前，五件灵器飞了出来，这五件灵器，是五行灵器，组成一套，蕴含五行之力，每一件五行灵器，都不亚于寻常的古灵器。
<br/>　　这五件加起来，几乎比起蜕变之前的祖龙至尊剑都毫不逊色。
<br/>　　如果得到机缘，这五件灵器合为一体，也很有可能向着极道仙兵所蜕变，是难得是至宝。
<br/>　　五件灵器一出，火景身上的力量不知道强横了多少倍，猛然手心一动。

<br/>　　这神芒凝聚五行之力，甚至是带有着几分混沌气息，天下五行，皆出自混沌。
<br/>　　火景这一招打出，神芒横扫过去的一刻，皆是化为了一片虚无，湮灭其中。

<br/>　　这一道神芒，破空而来，直接向着楚风眠轰击而去。

<br/>　　看到这一道神芒的到来，楚风眠头都没有抬起，从容不迫，一股灵力凝聚在楚风眠的手心之中。
<br/>　　始祖第一式，当即打出。

<br/>　　龙吟咆哮。

<br/>　　楚风眠目光冰冷无比。
<br/>　　三大势力的圣子，都是花费了无数资源培养而出的，数量绝不会多。
<br/>　　多杀一个，就是多削弱三大势力一点力量。
<br/>　　对待敌人，楚风眠可不会手下留情。
<br/>　　楚风眠身形闪烁，一步踏入到了那火景的面前去，猛然一爪向着火景的头颅直接抓了过去。
