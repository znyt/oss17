　　第631章
<br/>　　最近宋伯康的日子也很忙碌。一面要炮制攻打百里胜的理由，一面暗中调兵遣将，另外还要留出精力对付程大雷，差人将朔方城搬空，留给程大雷一座空城。
<br/>　　虽然忙碌，却也不断有好消息传来。用朔方城交换薛问婷后，宋伯康立刻差人将她送到京州，崔相心情大悦，答应给宋伯康一批粮草支援。
<br/>　　现在百里胜已经是瓮中之鳖，龙字军答应协助，保证了程大雷不会暗中使坏。如此，消灭百里胜已经是十拿九稳的事情。
<br/>　　但这自然无法满足宋伯康的野心，等收拾掉百里胜后，他下一个目标就是对付程大雷，让自己这凉州王变得名副其实。
<br/>　　目前要做的是，是集中精力对付百里胜，暂时安抚好程大雷。朔方城已经如一座深潭，令程大雷陷了进去，如果程大雷没有壮士断腕的勇气，就不可能从这泥潭中脱身。宋伯康要做的是想个法子加剧程大雷消耗，如此回过头来收拾他时也能容易些。
<br/>　　“王上，咱们去哪里找人送给程大雷？”陆洛问。
<br/>　　“嗳，浩甲、武威、金城还有咱们所在的凉州城，多的是无家可归的流浪汉。把他们都送给程大雷，我们也能少个麻烦。”宋伯康道。
<br/>　　帝国多流民，这些人或者是因为战乱缘故家园被破坏而流离失所，或者是因为恶霸地主欺凌，或者是因为天灾人祸颗粒无收，种种原因综合在一起，于是只能跑到城里乞讨为生。
<br/>　　不仅凉州一地，帝国各处都有这种无家可归的流民，不然的话，当初林少羽揭竿而起也不会有那么多人响应。
<br/>　　“王上，可将这些人送给程大雷，他如果从中招兵买马，咱们岂不是变相帮了他。”
<br/>　　“嗳，这就是你多虑了。城内的流民，要么是好勇斗狠之徒，要么是好逸恶劳的懒汉，剩下的就是老弱妇孺，靠这些人能够打仗？呵，笑话。就算程大雷能把他们聚拢起来，可这一张张嘴都是要吃饭的，凭程大雷那点家底能养活他们多久。就算秋天打了粮食……但你觉得，我会容他们活过这个秋天么。”
<br/>　　“王上，高呐。”陆洛情不自禁道。
<br/>　　宋伯康摆摆手，道：“奉承话就不要多说了，这件事抓紧时间去办，一定要做得滴水不漏，莫引起程大雷的怀疑。目前，我们最要紧的还是如何对付百里胜。”
<br/>　　程大雷是未雨绸缪，百里胜才是眼前的首要敌人。关于他，宋伯康已做了许多准备，切断百里胜的粮道，由武威和金城两路出兵，对百里胜的主要兵力采取围而不攻。
<br/>　　真如果打起攻坚战，哪怕最后能拿下百里胜，宋伯康方面也是损兵折将严重。他并不愿付出如此大的代价，围着不打等着百里胜耗尽粮草，然后再一举歼灭敌人。
<br/>　　百里胜坐镇的地方名叫狼山关，听名字便知道是凶险之地，土地贫瘠，关外便是一望无垠的草原，草原上生活着穷凶极恶的戎族。
<br/>　　围城已有些日子，百里胜手中有三万兵力，到现在双方还未见过一仗。可这三万仗嘴却是无形的敌人，一日一日消耗着狼山关的粮草。
<br/>　　大厅内乱糟糟一片，副将裨将牙将……仿佛一个人长了八张嘴巴，恨不得所有嘴巴都闭上，大家都听自己的。
<br/>　　“打，怎么打得了。姓宋的根本不和我们动手，况且宋伯康的兵力远超我们，咱们怎么是人家对手。”
<br/>　　“宋伯康手里可握着朝廷圣旨，说咱们是乱贼。为国为民死了也算不得什么，可咱们兄弟就算死在沙场上，也得让人戳我们脊梁骨。”
<br/>　　乱哄哄一片，终于有人看向百里胜，道：“侯爷，您倒是拿个主意呀。”
<br/>　　百里胜坐在帅案后，手托双腮一直没有开口说话。大厅内的吵闹争论，他并没有参与。他心中自然也不痛快，也想抱怨，起哄，乃至破口大骂。可身为一地之主，他必须担起责任来，现在也没有时间抱怨。
<br/>　　“吵呐，为什么不吵了，看能不能吵出个法子来？”百里胜道。
<br/>　　大厅内渐渐安静下来，诸人低头唉声叹气。百里胜的副将常鸣之道：“侯爷，眼下或许只有一人能够帮助我们？”
<br/>　　“谁？”百里胜道：“你说的是程大雷是么。难道你不知道，宋伯康送了一座城给程大雷，程大雷这等见利忘义的小人，现在他们两个已经狼狈为奸的勾搭在一起了。”
<br/>　　百里胜唏嘘长叹，他和程大雷很早就打过交道，互相照面也并非一次两次。可这程大雷竟然一点交情不念，这次竟与宋伯康联手置自己于死地。百里胜对宋伯康倒是不如何恨，但提起程大雷却是咬牙切齿。
<br/>　　“侯爷，那咱们该如何是好，关内的粮草撑不了几日了。再如此下去，只能伸长脖子等着人家来砍头。”手下又闹哄起来。
<br/>　　“侯爷，属下有一计，或许可以破局，但不知当讲不当讲。”孙鸣之道。
<br/>　　“都到这个时候了，你还卖什么关子，有话直说。”百里胜道。
<br/>　　“那宋伯康这次是铁了心要赶尽杀绝，两军对垒我们未必是对手，更何况宋伯康根本不与我们交锋。如今之计，只有一个法子可以破局。”
<br/>　　“说？”
<br/>　　“向东不行，只能向西走。西面便是茫茫草原，宋伯康说我们勾结戎族，十恶不赦，我们索性就反了帝国，去投靠戎族。”
<br/>　　一席话说完，大厅内鸦雀无声，诸多人一切窃窃私语，神情闪烁。
<br/>　　帝国与戎族乃是生死仇敌，镇守边关本就是防备戎族入侵。大家如果投靠戎族，便成了帝国的千古罪人。可帝国这次所作所为，的确又对不起他们。现如今，他们已被帝国抛弃，成了无人在意的弃子。
<br/>　　最后，大家的目光都落在百里胜身上，等他最后拿主意。
<br/>　　过了许久，百里胜长身而起，拱手冲诸人道：“诸位，我们生是帝国的兵，死是帝国的魂，此战死则死矣，绝不可卖国求生。”
<br/>(本章完)
