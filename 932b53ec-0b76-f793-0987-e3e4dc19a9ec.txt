　　因为自己知道这些人最后达到的成就，在听到这些人年轻时候的过往，不禁有点唏嘘，天赋虽然很重要，但真的不是决定一个人的上限，这几个目前所谓的天才少年之中，还有个别后来走上邪修的道路，或者沉迷名利和酒色，最终把自己给毁了。
<br/>　　“对了，张晓寒呢？”叶少阳突然想起这个人，之前被张道士极力吹捧，说是当今法术界最具天赋、也是实力最强的二代弟子，但是后世却没留下什么资料，至少叶少阳没有听说过，因此更加感到好奇。
<br/>　　毛小方听到张晓寒，愣了一下，说道：“张晓寒啊，他是天选之子，今天也不过二十多岁，法力深不可测，至少已经是地仙牌位，很可能是灵仙，甚至更高。不过……他虽然是九华山俗家弟子，却是万法皆通，最重要的是，他是法术公会钦点的法术界第一弟子，重点培养的。”
<br/>　　“法术公会钦点的？”叶少阳一听就愣住了，“法术界还管这事？”
<br/>　　“是的。上次龙华会，就是法术公会发起的，会上安排有潜力的二代弟子斗法，最后张晓寒脱颖而出，成为第一，被法术公会看中，定位第一弟子，有他们亲自培养了几年，他一手符印术冠绝天下，自然不是佛门法术，但也不完全是道门的，而是法术公会所传授，与人间法术不太一样，但是实力极强，因此，张晓寒的真实实力，比他的牌位境界还要高出许多。就算是在一代弟子中，也算是第一等的了。”
<br/>　　叶少阳沉吟不语，过了半晌，说道：“他为人怎么样？”
<br/>　　“恃才自傲，性格并不讨喜，除了跟他一起被选去接受法术公会栽培的几人之外，没什么人喜欢他们。”
<br/>　　停顿了片刻，毛小方接着说道：“不过法术界不喜欢他，倒不是因为他的性格，而是……张晓寒的崛起，代表的是法术公会的成功，相比之下，好像我们人间法术界真的不如法术公会似的。”
<br/>　　叶少阳点点头，这种感觉，他当然能够理解。
<br/>　　毛小方轻轻叹了口气，说道：“不过啊，人家确实很强，这方面也让人不得不服，所以法术界也接受了他第一弟子的名分，并且张晓寒所到之处，一般门派都会给他方便。我也见过这个人一面，说实话，我很不喜欢这个人，想要带领法术界崛起、延续荣光，我觉得这样一个倨傲的人是不合适的。”
<br/>　　叶少阳笑道：“强者一般都是高傲的。”
<br/>　　毛小方立刻说道：“那不一样。强者是应该高傲，不过对于不如他的人，至少不会嘲讽，也不会蔑视吧？他安那种就是完全看不起人的模样，让这样的人引领法术界，怎么会有前途。”
<br/>　　叶少阳听他这么一形容，对比也一下道风，道风也是高傲的——应该说是孤傲，但是如毛小方所说，道风并不蔑视比他弱的人，只是懒得理而已，其实他是面冷心善，内心有着巨大的担当。至于这个张晓寒……叶少阳听毛小方的形容，忍不住想起了一个人来：
<br/>　　死在自己手中的凌宇轩。光听描述的话，这两人性格还真的挺像的。凌宇轩也是顶着天才的名头成长起来的，二代弟子中的佼佼者，不过对比这个张晓寒的实力，肯定是要差得远了。
<br/>　　毛小方冲着叶少阳笑了笑：“我这么说，你可不要误会我是嫉妒人家的实力，我自己的实力我清楚的，一辈子也赶不上人家，不过我倒是希望将来能出一个天才，来打败他。我很想看看他败在别人手下，会是一副什么样的表情。”
<br/>　　突然想到什么，对叶少阳说道：“对了少阳子，你在你那个时代，也是数一数二的天才，如果你法力复原，你觉得自己跟这个张晓寒想比，能是他的对手吗？”
<br/>　　叶少阳耸了耸肩，“这东西不好比吧。我也不知道他到底有多强，又没打过。”
<br/>　　毛小方哈哈大笑，“其实我倒是希望你能打败他，我们这些人，是没有希望了。可惜你实力没有恢复，而且你一心要回你的时代，不能在这里久留，不然我倒是真的很想见识一下呢。”
<br/>　　叶少阳微微一笑作为回应。争强斗胜的心思，他当然是有的，不过……说起来，他一向不喜欢跟人斗法，而且对跟法师之前的战斗，根本提不起兴趣。
<br/>　　自从突破灵仙境界之后，在人间能跟他一争高低的可能只有道渊真人了，那段时间，叶少阳自己也听飘的，尤其是大帝钦赐牌匾，更是让他有一种把逼装到了极致的感觉。不过也就飘了一阵子，之后去了青冥界，遇到黎山老母那一干强者，才知道自己的实力，在那些阐教金仙和佛门大能面前，不说一文不值，至少算不上绝顶强者。
<br/>　　更不要说李浩然这样的变态。还有尸族的女魃，后卿……这些都不是自己所能对付的。
<br/>　　以前，叶少阳还没有觉得什么，当上灵仙，在人间和鬼域装装逼还是挺好的，但是星宿海一战之后，整个世界都改变了，冷玉暴走成了转世鬼童，被后卿带走，自己无论是想要救出她，还是将来要保护她，都需要拥有更强大的实力。
<br/>　　穿越之后的这段日子，叶少阳经常想起这个问题，更加觉得自己太弱了，以前，他没有追求绝对实力的雄心，现在，他有了，只可惜事与愿违，现在的自己只剩下一两成的实力……
<br/>　　叶少阳感到很无力。
<br/>　　在山上吃完干粮，两人一直聊到天黑。天黑之后，月亮出来，今晚的月色还不错，从这里俯瞰山谷，大致能看清下面的情况。
<br/>　　山谷里那股浓郁的阴气，颜色转深，看上去一片漆黑。
<br/>　　叶少阳抬头望着月亮，猛然想到了一件事：这个世界的月亮，跟自己所在那个时代的月亮，是不是也是同一个呢？
<br/>　　这个问题，自然也是没有答案的。
