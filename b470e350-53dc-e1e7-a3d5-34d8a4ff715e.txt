        落以柔今天羞辱自己的时候，童筱颖一个字都未跟傅夜沉提起，只是十分乖巧地拉起傅夜沉的手，给他的右手手背上那两排已经结痂了的齿印，贴上了创可贴。
<br/>
<br/>        “嗯，大哥，以后，我再也不咬你了。”童筱颖也觉得，一个大男人的手背上多了这么两排牙齿印确实挺伤风雅的，特别容易让人想入非非。
<br/>
<br/>        “我是不是可以理解为，你这算是默认了我以后可以对你为所欲为？”傅夜沉勾唇一笑。
<br/>
<br/>        童筱颖忍不住恼羞地捶了捶傅夜沉的胸膛：“大哥！你得节制点！适可而止，这样才对身体好！”
<br/>
<br/>        “我觉得，其实你也挺享受的。你老实说，我有没有让你觉得很舒服？”傅夜沉双手抱着童筱颖，双臂将她圈在自己的身前，笑不拢嘴地问。
<br/>
<br/>        童筱颖“唰”地一下，双颊红了个通透，立即推开傅夜沉，答非所问：“我要下去工作了！”
<br/>
<br/>        “下班后，在前坪等我，我们一起回家。”傅夜沉接着说道。
<br/>
<br/>        童筱颖一边往外走，一边向傅夜沉俏皮地做了一个“OK”的手势。
<br/>
<br/>        要说对落以柔不上心，作为已经被男人背叛过两次的女人来说，童筱颖心里还是拉起了警戒灯。
<br/>
<br/>        第一次，何明旭和林若琴是什么时候背着她开始的，她并不清楚，事后也不太在意。但是第二次，傅昼景和季思妍的事情，是她恍然间醒悟过来时，才意识到的。
<br/>
<br/>        那个时候，倘若她没记错的话，季思妍是傅昼景身边的秘书，工作上朝夕相处。
<br/>
<br/>        总经理和秘书……
<br/>
<br/>        现在是总裁和助理秘书……
<br/>
<br/>        如此类似的情况。
<br/>
<br/>        她该怎么做？
<br/>
<br/>        睁只眼闭只眼，还是完全放任不管？
<br/>
<br/>        又或者说，她应该放宽心态完全信任大哥能管住他自己。
<br/>
<br/>        有那么一刻，她心里是迷茫的。
<br/>
<br/>        回到设计部的集中办公室里后，有个男同事趁着部长孙莉莉下午不在办公室，便在集中办公室里推销起自己的产品来。
<br/>
<br/>        “微型窃听器，带定位功能的，你们谁要？很适合放在老婆的手提包里，或是老公的公文包里，安装在车内都可以。男女朋友也行。”
<br/>
<br/>        设计部的集中办公室里，带她童筱颖一起也就三十六个人。
<br/>
<br/>        那男同事一个个推销下来，不知道有没有卖几个出去，但是到了她童筱颖这儿，童筱颖毫不犹豫地一口就拒绝了。
<br/>
<br/>        虽然说，她是有点担心大哥也会出轨，但是不至于到了这种非法安装窃听器的地步。
<br/>
<br/>        所以，落以柔的事情被童筱颖给压箱底了，只是心里积压着，就会老惦记着这事，造成她了随后的这几天里有个时候还会疑神疑鬼。
<br/>
<br/>        结果，日有所思夜有所梦，童筱颖这天晚上的一声呓语，将傅夜沉给惊醒，他伸手便摸开了床头旁的小夜灯。
<br/>
<br/>        “大哥，不要出轨啊！”
<br/>
<br/>        童筱颖说这句梦话的时候，吐词十分清楚，而她整个脸色都白了，柳眉紧锁，就连粉唇都被她抿得发白。
<br/>
<br/>        她的双手揪着被单，紧紧地抱在胸前，像是害怕又像是在愤怒，下一秒，眼角还溢出了眼泪。
<br/>
<br/>        一时间，傅夜沉有些束手无策，但还是决定将她从那噩梦中给摇醒。
<br/>
<br/>        “筱颖、筱颖，醒醒，筱颖。”他动作很轻柔，只是在试图将她唤醒。
<br/>
<br/>        童筱颖睁开惺忪的睡眼，看到傅夜沉正一手撑着脑袋侧着身子盯着自己，于是迷迷糊糊地问道：“嗯？大哥？你怎么还没睡？”
<br/>
<br/>        “你做噩梦了。”傅夜沉凑过身去，温柔地亲了亲她的额头，一边替她抹去眼角的泪珠，一边关心地说道。
<br/>
<br/>        童筱颖闭了闭眼睛，再次睁开双眼的时候，目光有些涣散，大脑更是有些转不过来。
<br/>
<br/>        她刚刚做梦了吗？
<br/>
<br/>        做什么梦呢？
<br/>
<br/>        怎么就突然间不记得了。
<br/>
<br/>        “你是不是有心事？”傅夜沉接着问道。
<br/>
<br/>        童筱颖怔愣了一下，猛然想起了什么，忍不住抓着被单，拉过头顶。
<br/>
<br/>        “你是不是担心我会出轨？”傅夜沉笑道，抬起另一只手来，将童筱颖头顶的被单拉了下来。
<br/>
<br/>        “那你会吗？”童筱颖对上傅夜沉目光，一脸认真地问。
<br/>
<br/>        傅夜沉嘴角微扬，不假思索且十分笃定地回答：“不会。”
<br/>
<br/>        夫妻之间，其实不怕苦不怕累，最怕对方背叛自己，对自己不忠。
<br/>
<br/>        对于这种害怕，女人比男人更加敏感。
<br/>
<br/>        童筱颖随即伸出手来，抱住傅夜沉的腰身，往他怀里钻了钻，找了个舒适的位置，安静了下来。
<br/>
<br/>        直到这一刻，傅夜沉才发现，筱颖对感情缺乏安全感。
