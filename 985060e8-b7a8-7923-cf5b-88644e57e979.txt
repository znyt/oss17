　　“您放心力哥，我明白，我全都明白！”
<br/>　　三儿的额头上渗出了一层冷汗，他如何能听不明白，李霄力这是在敲打警告自己，不要忘记自己姓甚名谁。
<br/>　　“记住，这个世界上聪明的人要比笨的人活得更长，但是过分聪明，只会死得更快。”
<br/>　　“是，力哥。我记住了。”
<br/>　　……
<br/>　　“碧海潮生”夜总会。
<br/>　　钱鑫手中拎着那个沉甸甸的淡粉色安全套，坐在床边发呆，先前发生在这个房间里的事情，就像是电影一样在钱鑫眼前回放。
<br/>　　自己是什么样的情况，钱鑫很清楚，要不然他也不会去买什么进口“神油”能让自己表现的更男人一些，钱鑫真的不敢相信，先前就在这张床上把陆岑狠狠地折腾了一个小时，生猛如虎的男人就是自己。
<br/>　　那种感觉真是太舒爽了，钱鑫觉得自己从来就没有像刚才那么男人过。
<br/>　　一想到之前跟陆岑所发生的事情，钱鑫心中就觉得有些兴奋，恨不得把陆岑拖回来再战三百回合，把那些在片子里看到，还没来得及用的花样逐个用在陆岑身上，这么想着，钱鑫又开始骚动起来；可心中骚动，但身体却没什么反应，这让钱鑫觉得有些奇怪，不过想到可能是刚才做得太猛，所以还没来得及恢复过来，钱鑫也就没放在心上，把手中的安全套扔进垃圾桶后，扶着腰，走进浴室去冲洗。
<br/>　　刚洗完澡走出浴室，钱鑫放在口袋里的手机就响了起来。钱鑫头发还在滴着水，走到椅子边，把手机摸了出来，看了眼上面的来电显示，赶紧接通了电话，“三爷！”
<br/>　　“你小子这次是玩爽了。”三儿手指在方向盘上敲打着，对电话那头的钱鑫说道。
<br/>　　钱鑫赶紧拍马屁道：“这完全是沾了三爷的光，要不是三爷您，这地方哪儿是我能来的，岑姐哪儿能来陪我呀。”
<br/>　　“还算你小子有良心，没忘记我。”三儿拿着手机说道：“以后还想不想继续让你岑姐陪你？”
<br/>　　“岑姐还能再来陪我！？”
<br/>　　钱鑫有些难以置信，他觉得自己能跟陆岑上一次床那就已经是很满足了，哪里还敢想第二次。如果没有三爷，就自己口袋里这点钱，也就只能去街边发廊玩玩。
<br/>　　“能啊。”三儿很肯定的说道：“这还不是我一句话的事情……”
<br/>　　“谢谢三爷，谢谢三爷！”钱鑫不停的道谢。
<br/>　　“先别忙着道谢，你岑姐的身价可不一般，也不能让你白玩是不是？想玩可以，但是你得帮我把事情做好。”三儿对钱鑫说道。
<br/>　　“三爷您说佟医生那事儿啊。”钱鑫这小子不务正业归不务正业，脑子还是很灵活的，三儿一提醒，他就反应了过来，说道：“佟医生那边也没什么事情，哦对了，她男人回来了。”
<br/>　　“她男人回来了？”三儿皱眉问道。
<br/>　　“是啊。”钱鑫回答道：“那天正好是我上班，他自称是佟医生的丈夫，我不让他进小区，他还把证件给我看了，好像……好像还是个中尉！我一开始不相信，最后还是佟医生来把他接进去的。”
<br/>　　“中尉，中尉，这事情倒是有点难办了。”
<br/>　　三儿轻声嘀咕了一句，对钱鑫说道：“你继续给我盯着点，有重要消息记得立刻汇报告诉我，如果还想要你岑姐来陪你，那你就要给我带来有价值的消息，不然上面我也不好交代，毕竟我也是给人当小弟，没办法。”
<br/>　　“我理解，我理解。”钱鑫拿着手机点头哈腰着说道：“您放心，三爷。我可一直都不敢忘记您交待的事情，我这边一有消息，我立刻就通知您。”
<br/>　　三儿挂断手机，想了想最后还是给李霄力去了一个电话，跟李霄力简单的说了几句后，发动汽车，掉头从陆岑所居住的小区门口离开了。
<br/>　　……
<br/>　　钱鑫洗完澡，也没去想之前自己为什么想到了片子里那些花样而自己没反应的事情，躺在床上，闻着陆岑留在这张床上的味道，睡着过去。
<br/>　　第二天一早，钱鑫起床，发现自己兄弟还是软趴趴的没什么反应，想着自己应该是纵欲过度，也没多想，洗漱完，拿了自己的东西离开了房间，昂首挺胸的走出“碧海潮生”，跨上自己的小电驴，离开了。
<br/>　　事情一转眼过去了好几天，渐渐的钱鑫察觉到了不对劲。
<br/>　　不管是自己看片，还是回想在“碧海潮生”那个晚上的事情，自己感觉到十分兴奋，非常渴望发泄，可是自己身体却没有任何的反应。尤其是看到慕容飞陪着佟千雪进出小区的时候，钱鑫觉得自己心头就像是被人用羽毛不断撩拨一样，恨不得立刻冲上前，把佟千雪压在自己身体下，狠狠的蹂躏，可自己的身体却没有任何的反应。
<br/>　　钱鑫觉得自己身体中有着一团猛烈的火焰在不断地燃烧着，烧灼着自己的肉体和灵魂，让自己难以忍受，非常渴望得到发泄，可是发泄口却像是被堵住了一样，让他无法得到舒缓。
<br/>　　以前钱鑫想要的时候，看个片子，或者拿着陆岑的照片，自己动手解决一下也就过去了，可是现在就算他对着陆岑的照片，身体也没有任何反应。
<br/>　　难道自己废了！？
<br/>　　钱鑫被自己这个想法吓得一激灵，心中燃烧起来的火焰没下去了一些，可是很快，那团火焰又烧了起来，这让钱鑫要发疯了。
<br/>　　这种情况一直在折磨着钱鑫，而且一天比一天严重，他晚上睡不着，精神越来越差，他想到了那天陆岑给自己吃的东西，赶紧打了个电话给三儿。
<br/>　　“有消息了？”三儿问道。
<br/>　　“暂时还没有。”钱鑫挂着两个大眼袋，精神萎靡的对三儿说道：“三爷，我问您个事儿，那天您让岑姐给我吃的东西还有吗？我感觉自己好像废了，这、这硬不起来了……”
<br/>　　“你说那个，有啊，我当然还有啊。”三儿看了眼正跪在自己脚边的陆岑，嘴角噙着一抹冷笑，说道。
