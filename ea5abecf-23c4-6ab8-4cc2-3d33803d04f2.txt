        宋云汐抿了抿唇，没再吭声。
<br/>
<br/>        冷晏琛将宋云汐送到病房门口后没打盹就转身离开了。
<br/>
<br/>        他回到傅炎廷的病房里时，傅炎廷的贴身管家林贵叔，给他递了个保温饭盒过来。
<br/>
<br/>        “这是你奶奶刚送过来的排骨汤，你拿去送给宋小姐喝。她早几天才做完骨髓移植手术，需要补补。”傅炎廷嘱咐道。
<br/>
<br/>        冷晏琛不解道：“既然是奶奶送过来的，那么应该是做给祖爷爷您吃的吧？”
<br/>
<br/>        林贵叔“咳”了一声，忍俊不禁道：“老爷这是想让小少爷去追那位宋小姐。”
<br/>
<br/>        “祖爷爷，星儿那丫头没跟您说吗？宋小姐在相亲的时候，就已经拒绝我了。”冷晏琛淡定道。
<br/>
<br/>        “你这臭小子，就该使出你爹地当年追你妈咪的那股子劲来！拒绝了一次，你不会继续追啊？”傅炎廷没好气地训斥。
<br/>
<br/>        冷晏琛尴尬地笑了笑：“祖爷爷，我已经答应过您相亲了，这种追不追女孩子的事情，您就不要再管了！”
<br/>
<br/>        “我不管？我不管我能抱到小玄孙吗？”傅炎廷顿时一脸不乐意。
<br/>
<br/>        冷晏琛一脸无奈地抬起手来，挠了挠后脑勺：“我还在服役了！”
<br/>
<br/>        “那你就先把宋小姐追到手再说。”傅炎廷一本正经道，顿了顿后，又补充了一句，“你放心去追！祖爷爷保证没看错人！宋小姐是个好女孩，你别错过了。”
<br/>
<br/>        “可我对宋小姐……”
<br/>
<br/>        “可什么可，有什么好可的？”傅炎廷直接打断了冷晏琛的话，接着说道，“日久生情，懂不懂？赶紧把汤给宋小姐送过去。”
<br/>
<br/>        冷晏琛咧了咧嘴，一脸无奈地从林贵手中接过保温饭盒。
<br/>
<br/>        另一边，宋云汐拿起手机刚准备给自己点外卖，一个人影从她余光里闪过将她吓了一跳。
<br/>
<br/>        她抬眸时，见来人是郑温韦，脸色立即变得很不好看。
<br/>
<br/>        郑温韦仍旧是一袭高档的西装革履，早已没有了当年她恋慕的那个清新润朗的少年的影子。
<br/>
<br/>        “为什么要瞒着我给雨柔捐赠骨髓？”郑温韦深沉而怜惜地问。
<br/>
<br/>        宋云汐不以为然地笑了笑：“哦？是吗？我的骨髓捐给了宋雨柔啊！这么巧噢！我来医院，只不过是做个爱心捐赠罢了！捐给谁，对我来说，压根就无所谓。”
<br/>
<br/>        “云汐，我会兑现那天我所说的话。你放心，我会娶你为妻！”郑温韦大言不惭道。
<br/>
<br/>        “我为什么要嫁给你？”宋云汐只觉得郑温韦这句话很搞笑。
<br/>
<br/>        不，不是搞笑，是可耻！
<br/>
<br/>        郑温韦自信满满地说道：“除了我有条件娶你以外，其他男人压根就入不了你的眼不是吗？云汐，我知道，你看男人的眼光很高。”
<br/>
<br/>        “哦，是吗？你娶我？然后把宋雨柔养成你的地下情人？”宋云汐忍不住对郑温韦翻了个白眼。
<br/>
<br/>        郑温韦凝了凝眉，抱怨道：“云汐，你看看你，你总是这个样子！我说过，我和雨柔之间是清白的。说真的，你真的不如雨柔。她温柔体贴，善解人意。而你，猜忌、多疑，动不动就爱跟我吵。”
<br/>
<br/>        “呵呵——那你还娶我做什么？三年前，你就说我两不合适，跟我提出分手。是不是将来，我若嫁给了你，你又会对我说，我们不合适，离婚？”宋云汐冷斥。
<br/>
<br/>        郑温韦顿时恼火起来：“云汐，你到底有完没完？”
<br/>
<br/>        “你滚吧！我不想再看到你。”宋云汐闭了闭眼睛，转过身去看向窗外。
<br/>
<br/>        郑温韦此刻却从裤兜里掏出一个玫瑰红的小礼盒，在宋云汐的身后单膝下跪，双手举了起来，将礼盒打开来，真挚诚恳地说道：“云汐，嫁给我！”
<br/>
<br/>        宋云汐身心一怔，讷讷地转过身去看着郑温韦。
<br/>
<br/>        郑温韦见宋云汐愿意为他转身，惊喜不已地扬了扬眉，发自肺腑地说道：“云汐，我们已经错过了三年。未来，我们就不要再错过了好吗？当初，我选择跟你分手，是想让你好好反省自己。现在，我知道。其实，你是相信我的，对不对？不然，你不会为了我，拒绝那么多追求你的男人。”
<br/>
<br/>        而就在这个时候，冷晏琛提着保温饭盒正巧走到病房门口，撞见这一幕。
<br/>
<br/>        他先是一愣，缓过神来后，准备掉头就走时，岂料有人唤住了他。
<br/>
<br/>        “阿琛，你来了！”宋云汐嘴角微扬，无视掉单膝跪地的郑温韦，踱步向冷晏琛走了过去。
<br/>
<br/>        宋云汐走到冷晏琛的跟前时，仰头对上他的黑眸，并向他眨了眨眼睛，使了个眼色。
<br/>
<br/>        冷晏琛剑眉微扬。
