<br/>    等到五朵红云过去，陈明终于是慢悠悠的出来了，顺路来到紫霞仙子他们等待的地方，陈明来到紫霞仙子他们的山头上，朝着紫霞仙子问道：“你们来这儿做什么？”
<br/>    
<br/>        紫霞仙子道：“宗师，我等听闻燕山有难，特地前来助宗师一臂之力，宗门也是想借此机会人，让我们历练一番。”
<br/>    
<br/>        陈明微微点头，原来是不要钱的打手，这感情好啊！
<br/>    
<br/>        陈明笑道：“既然诸位来都来了，我也实不相瞒，此次出兵，我欲一统这十万大山，重整妖族！”
<br/>    
<br/>        紫霞仙子道：“宗师，你堂堂燕山，数万炼丹师的大宗门，只要十万大山这么小一块地盘？”
<br/>    
<br/>        陈明还以为古仙门怕自己扩张，现在才知道，原来这在古仙门眼里，还太小了......
<br/>    
<br/>        果然是古仙门，财大气粗那不是现在的燕山可以比的，陈明笑道：“我燕山此前一直隐世，不过刚刚入世而已，先站稳脚跟吧。”
<br/>    
<br/>        紫霞仙子道：“宗师，依我来看，宗师这一行，最好是直接攻打到铁牛谷附近，正好这一路之上基本上都是鬼仙的势力范围，也好和我们古仙门连成一线，彼此之间，也好有个照应。”
<br/>    
<br/>        陈明思量了一番这个距离，这打过去得有十万八千里吧！
<br/>    
<br/>        你们古仙门都是这么看地图的吗？
<br/>    
<br/>        比不了比不了！
<br/>    
<br/>        就算是一路打过去了，哪儿来那么多的人守？
<br/>    
<br/>        等等，不是还有三千多燕山领主吗？等到这三千多领主兵强马壮，未必不能打过去啊。
<br/>    
<br/>        陈明拱手道：“我们燕山，乃是热爱的和平的一个宗门，平时都只是闭门炼丹不出，若不是此地大妖王，妖族大君肆虐，劫掠我们给你们运送丹药的商队，我也不会攻打他们。”
<br/>    
<br/>        众多天才听到陈明这句话，顿时高山仰止，“宗师大义！”
<br/>    
<br/>        毕竟还不知道燕山喜欢一不小心就把别人抄家的黑历史，他们对于陈明，可谓是敬佩之极，拥有强大的实力，却不欺凌周围的弱小，“宗师，当真是我正道楷模！”
<br/>    
<br/>        而且一听到这些丹药是准备运出去卖给他们的，对于这些大妖王，更是一丝同情都没有了，“这些大妖王，窃据一方，为祸人间，当真是该死，宗师这是替天行道！”
<br/>    
<br/>        对此，陈明也就打了个哈哈，对你们有利的，那就是替天行道，果然，是正道的一贯德行。
<br/>    
<br/>        “诸位既然来了，那我就先谢过诸位了，若是诸位日后得到了还魂水，送到燕山来，我会以九幽寒泉之中的价格，继续为大家炼制。”
<br/>    
<br/>        众多的天才终于是听到了这个消息，齐齐拱手道：“只要宗师一声令下，我等绝对没有二话，直接冲杀出去！”
<br/>    
<br/>        紫霞仙子的眼中露出一抹精芒，“宗师既然是要进攻妖族大君，对方毕竟也是有着一位大君，而且聚兵十万不止，不如和我商议一番？”
<br/>    
<br/>        陈明看了紫霞仙子一眼，这个女王到底又有啥事？
<br/>    
<br/>        跟着紫霞仙子进入一个小木屋，紫霞仙子见四周没人，直接道：“宗师，你这一炉阴阳和合丹直接赚了六枚，你就没有良心不安吗？”
<br/>    
<br/>        陈明道：“不知道是谁，吃着这昧良心的丹药突破的大君。”
<br/>    
<br/>        紫霞仙子也不脸红，直接道：“宗师，我为你保管这个秘密，宗师你看，你要怎么谢我？”
<br/>    
<br/>        陈明道：“那我们来算算我帮你突破大君这笔账要怎么算。”
<br/>    
<br/>        紫霞仙子愣了愣，然后道：“四位老君不是已经谢过了吗？”
<br/>    
<br/>        陈明伸手一拍桌子，大义凛然道：“那是谢我救你们的命，作为你们的长辈，保护后辈是应当的，这个我就不提了。”
<br/>    
<br/>        紫霞仙子思量片刻，然后道：“要不这样，你让我们群星仙门有资格在你这秘密购买阴阳和合丹，如何？”
<br/>    
<br/>        陈明思考了片刻，这样一来，也不是不行，反正这玩意卖得贵，然后道：“那好吧，就这么定了。”
<br/>    
<br/>        紫霞仙子忽然想起了正事，“对了，对战妖族大君的时候，还劳烦宗师在旁替我掠阵，让我熟悉一下大君之间的对战。”
<br/>    
<br/>        陈明对于这个倒是没有任何意见，白送的打手，还是大君级别的打手，不要白不要，要了当白要，反正都是带小孩，多一个少一个对于陈明来说已经没区别了，“可以，这件事，交到我身上。”
<br/>    
<br/>        紫金山，金毛吼和长河对坐，正在对坐下棋，长河道：“这燕山，也未免太目中无人了，居然敢随意斩杀我的儿子！大哥，一定要为我做主啊！”
<br/>    
<br/>        金毛吼也是露出一副义愤填膺之色，“放心，贤弟，我已经查探过燕山了，如今燕山的本部据说已经迁移到了沙海之中，我们根本进不去，只要燕山敢出来，我一定为贤弟报仇！”
<br/>    
<br/>        长河拱手道：“多谢大哥！”
<br/>    
<br/>        就在这时，金犴忽然走了进来，脸上有着急色，“禀报父王，据前方的探子来报，燕山已经出兵了！”
<br/>    
<br/>        金毛吼丝毫不急，从容落下一子，道：“想必是妖族大君连日以来催促，燕山已经忍不下去了，就让他们两虎相争，我们坐山观虎斗，等到他们两败俱伤的时候，我等再发兵，坐收渔翁之利！”
<br/>    
<br/>        金犴道：“父王，燕山的目标，似乎并不是妖族大君？”

<br/>        金毛吼道：“不是妖族大君，那么他们是准备攻打谁？”
<br/>    
<br/>        金犴道：“看他们的样子，似乎是准备对父王动手？”
<br/>    
<br/>        金毛吼顿时一拳轰烂了棋盘，怒道：“好一个燕山，我不去找他的麻烦，他倒是先对我动手了，真当我紫金山是泥捏的不成？速速用我的名义，召集十万大山之中其他的大妖王，前来议事！”

<br/>        金犴道：“父王，现在，我们恐怕做不到。”

<br/>        金毛吼怒道：“怎么回事？以我的威望，这点事都做不到吗？”

<br/>        金犴道：“燕山此次，直接兵分五路，攻打了五位大妖王，他们现在自身难保，哪里还有时间过来会盟。而燕山之主，似乎是在坐镇后方，目前并没有太明显的行动。”
<br/>    
<br/>        金毛吼怒道：“好胆，竟然敢直接进攻我们数位大妖王，想必燕山的兵力也极为分散，便与他们战上一场，让燕山知道，我等也不是好惹的！”
