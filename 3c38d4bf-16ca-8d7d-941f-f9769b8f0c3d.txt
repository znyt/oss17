　　这道巨大的木墙，在迷雾的尽头，和谐地融入到一个更巨大的物体之中，天空之上，斑驳的阳光透过迷雾倾洒而下，随即又消失不见。
<br/>　　慕清霄深吸了口气，终于肯定了先前心中不可思议的猜测，那道巨大的木墙，居然只是一小截树根......
<br/>　　迷雾层重重，飘荡不定。
<br/>　　两人也因为太过惊讶，有些微微喘息，然后霍然抬头，视线穿过重重迷雾，紧接着，出现在他们视线中的，是完全超乎想象的一颗巨树！
<br/>　　树干在迷雾中若隐若现，粗壮的看不到边际，被粗糙的树皮包裹着的树干，如巨大的山丘巍峨耸立，贯穿云霄，一眼看不到尽头。
<br/>　　慕清霄与陆雪琪两人，在这颗巨树前，显得微不足道，宛若蝼蚁一般，若非亲眼看见，他们是绝对不信，世间竟有如此神树，仅仅是树根就高达六丈！
<br/>　　“这天帝宝库的主人，修为估计已经恐怖到了一定层次，现在的自己，恐怕只能仰望！”
<br/>　　微微收回目光，此刻，两人的手牵在一起，陆雪琪似乎有一些不安，慕清霄露出一个自信的笑容，道：“雪琪，不用紧张，目的地就是这里了。”
<br/>　　听到丈夫的安慰，陆雪琪也是安心了一些，道：“你所说的天帝宝库，便在这神树之上？”
<br/>　　“绝对不会有错，上去一看便知。”
<br/>　　说罢，慕清霄就搂住她纤细的腰肢，脚尖轻点，伴随着一阵破风声，化作一道剑光直冲上天，很快就没入重重迷雾之中。
<br/>　　疾风扑面而来，因为速度飞快而显得有些凌厉，在这片迷雾之中，倒是有几分像刚进入内泽时，在瘴气领域中的情景，只是此处没有毒气罢了。
<br/>　　顺着面前这颗不可思议的巨大树木往上疾驰，足足半刻钟的时间，迷雾竟还未消散，慕清霄有些怀疑，该不会就这么和天上的云层互相连接在一起了吧？
<br/>　　同时，慕清霄注意到身前那颗巨树的树杆上，也逐渐有了变化。
<br/>　　先前，在地面的时候，自然就是这巨树的底部，也是粗大到无法想像，而在树身之上，除了粗糙的树皮，就再也没有其他的东西了。
<br/>　　但在向上飞行片刻后，慕清霄忽然发现，这颗巨树上已经开始渐渐出现了分枝，嘴角牵起一丝笑意，看来这巨树也并非没有尽头。
<br/>　　而最引人注目的，却是从刚才开始，缠绕在树杆之上，类似藤蔓的一种奇异植物。
<br/>　　藤蔓纵横交错，叶片极其硕大，在枝叶顶端盛开着五颜六色的花朵，有红有黄，有橙有紫，色彩缤纷，煞是好看，隐隐还有芳香迎面飘来。
<br/>　　可直到此刻，他仍旧无法完全看清这颗巨树的树杆，究竟有多粗。
<br/>　　造化之奇，实在匪夷所思，面前的这一颗奇树，恐怕真是传说中，活了亿年的神木。
<br/>　　“蓬！”
<br/>　　伴随着一道沉闷的声响，剑光自迷雾中冲出，脚下的雾气也随着两人的身影，向上飘动了些许，随后轻轻落下，仿佛海浪慢慢平息。
<br/>　　冲出迷雾的瞬间，天地豁然开阔！
<br/>　　蔚蓝的天际，万里无云，碧空如洗，而脚下白雾茫茫，缠绕在面前这颗巨树的周围，越是接近巨树的地方，雾气就越加浓郁。
<br/>　　此刻，慕清霄与陆雪琪已经身处高空之上，也终于看清了面前这颗巨树。
<br/>　　即使是在如此之高的地方，这颗巨树呈现在他面前的树杆竟然仍是粗达百丈，而联想到从地面飞到现在的距离，在面前的简直不能称之为树，而是一座巍峨耸立的高山！
<br/>　　然而，事实就摆在眼前，这确实就是一棵树，而且依旧向上伸展，那巨大的树干上，除了同样令人惊愕的粗大分枝外，依然笔直地伸向天空。
<br/>　　“看来，我们还需飞行一阵子啊。”
<br/>　　慕清霄微微一笑，搂着陆雪琪化作一道剑光，再次向上空飞去，有种天高任鸟飞，海阔任鱼跃的心理情绪油然而生。
<br/>　　速度越来越快，任凭着疾风刮面如刀，越往上飞行，巨树的树杆也就渐渐缩小，片刻后，已经变成只有数十仗大小，尽管如此，也依然是惊世骇俗。
<br/>　　此时此刻，已经渐渐有了云气，不时飘荡在树身旁边，这颗巨树，似乎就像是上古传说中，那一座登天的阶梯，直上青天！
<br/>　　又向上飞行数仗左右，慕清霄停泄在空中，在他的面前，一条笔直的树杆，在这里突然分开了巨大的两枝，向左右伸展了出去。
<br/>　　慕清霄沉吟片刻，抬头向上看了一眼，看来就是此处了，轻轻落在这颗巨树的分岔地点，陆雪琪美眸看向四周，骇然之色根本掩饰不住。
<br/>　　慕清霄面带微笑，虽然表面平静，但这一路飞行上来，心中震动委实不小，在这之前，根本无法想像，世间竟有如此巨大的树木，不愧称之为亿年神木。
<br/>　　在最初的惊愕过后，他心中也有些惊喜，绝对是这里没错，天帝宝库肯定就在这里。
<br/>　　分岔的两枝，大小相若，几乎都有数十仗粗壮，站上几百人都没问题，凌空横去，犹如两条巨龙横跃在半空中。
<br/>　　从这里开始，枝叶渐渐繁茂，而且看着延伸距离估计不断，站在分岔路口，两边都看不到尽头。
<br/>　　慕清霄思索片刻，先前的惊天异象，莫名的鸟鸣，都预示着天帝宝库即将开启，紧紧牵着陆雪琪的玉手，也不知道待会会遇到何种危险。
<br/>　　“我们走吧。”
<br/>　　慕清霄目光向左右望了一眼，毫不犹豫，带着陆雪琪向左侧树枝飞去。
<br/>　　虽然在高空之上，但这颗巨树的巨枝上，依旧有无数硕大的叶片，繁茂至极。
<br/>　　但不知为何，却没有见到有什么果实花朵，倒是从底下树杆开始就一直缠绕着这颗巨树的无名藤蔓，鲜花盛开，花枝招展。
<br/>　　随着慕清霄的前行，这一侧的树枝渐渐也缩小了下来。
<br/>　　但不知为何，那些藤蔓确越来越是粗大，而那些盛开的花朵也越来越多，到后来简直随处可见，目不暇接，空气中飘荡着莫名的香气。
