
    山坡下面是个与世隔绝的古老藏族村，大概就几十户人家，在这处地方已经安居了大概千年左右，始终都保持着最为原生的生活状态，很多习惯千百年来都没有怎么改变过。<br/>　　<br/>　　    小林昙一是在一个多月前率领登山队抵达这里的，并且还与这里的村民进行了交谈，在藏区雪域高原生活了十年左右，他几乎已经算是个藏通了，并且为此特意学习过很多藏族极为生僻的语言，你真要是换个只能说普通话的人来，都未必能跟这里的藏民交流。<br/>　　<br/>　　    三井美奈他们从山上下到村子里的时候，这里的藏民都保持着警惕和提防的状态，甚至还有村民拿出了原始的弓箭和道具，三井良河和三井小叶顿时就把手伸到了腰间，要把枪给拔出来时，小林昙一慌忙拦了他们一下，低声说道：“别紧张，千万不要动手，他们只是正常的警惕状态，毕竟我们属于外来者打扰了他们的生活，上一次我来的时候也是这样，不过后来我用带的一些食物赠给了他们，敌意随即也就消失了，你们别乱动我去沟通”<br/>　　<br/>　　    小林昙一说完，就双手合十朝着对面走了过去，眼睛在村民中扫了几圈就看见了上一次他过来后曾经交谈过的几个村民，于是走上前去打了招呼，藏民对他明显还有着熟悉的记忆，见状果然放松警惕了不少，小林昙一和他们攀谈了几句后，就朝着身后的人败了下手说道：“把我们带的补给拿出来一部分交给他们”<br/>　　<br/>　　    小林良河说道：“给钱是不可以更直接一点？”<br/>　　<br/>　　    小林昙一无语的摇头说道：“你想多了，你觉得在这种地方有钱了他们有办法花么？他们多少年都不和外界接触一次，哪怕就是有人去山脉外面，也是以物换物，所以钱对他们来说跟废纸差不多，你还不如给他们一些药品和食物了”<br/>　　<br/>　　    三井美奈吩咐他们把补给直接拿出了一半，然后由小林昙一交给了对方，他还随之把这些东西做了详细的介绍，食物什么的对于藏民来说也就是新鲜和好奇，但小林昙一告诉这些人有一些消炎和治疗伤寒，感冒的药是什么作用时，顿时就让这些村民喜出望外了。<br/>　　<br/>　　    “上一次我过来和他们打听过，从这个村子往东南方，就是几十年前马丁鲍曼去过的地方，经过村子后大概四五天的时间就又再次回来了，并且回来的时候他们队伍里的所有物资都消失了，所以我估计马丁鲍曼藏起来的那批珍宝应该距离这个村子不到一天或者两天的路程”<br/>　　<br/>　　    三井美奈有些激动的说道：“那能让这些村民给我们带路么？”<br/>　　<br/>　　    小林昙一皱眉说道：“他们可以为我们领一段路，再远了就不可能了”<br/>　　<br/>　　    “为什么？”三井美奈不解的问道。<br/>　　<br/>　　    小林昙一说道：“村民说，往东南三十公里左右那里是他们的圣地，有保护他们的神明在那，每年除非是在特定的日子过去祭奠，否则他们是不会过去打扰安息的神明的”<br/>　　<br/>　　    三井小叶撇嘴说道：“不过是一些古老的传说罢了，哪里来的神明，这些未开化的愚人们，太盲目崇信自己的信仰了”<br/>　　<br/>　　    小林昙一很认真的摇了摇头，慎重的叮嘱道：“在我们国内也有神社的传说，很多东西你都解释不了，而在藏区就更不能掉以轻心了，这个地方简直太神秘了，甚至我一直都相信这里有很多神秘的力量，我们不至于会崇信，但不能不认真对待，你们在藏区高原的时间太少了，如果像我一样多呆几年见得多经历过一些事之后，就不会说出刚才那番话了”<br/>　　<br/>　　    “那你的意思是……”<br/>　　<br/>　　    小林昙一说道：“我先和他们沟通一下再说吧”<br/>　　<br/>　　    三井小叶回头看了眼他们刚刚下来的方向，疑惑的问道：“吉野他们怎么还没有下来？”<br/>　　<br/>　　    三井良河嗤笑道：“你该不会是以为他们三个带着枪过去，会有什么意外吧？我们下来才不过几分钟，而从山顶往下走至少需要四十分钟以上，再等等的估计他们就会赶过来了”<br/>　　<br/>　　    于此同时，王惊蛰他们在杀了吉野三人后，却没有顺着山坡往下走进入藏族村，而是横向转移，去到了一处较高的山头上，王惊蛰觉得这里的山脉走势略微有些古怪了点。<br/>　　<br/>　　    “这种山脉走势我是第一次见到，以前只在古书上看到过图解，这种走势有个说法叫‘白驹过隙，青鱼落潭’是一种很难得的风水脉络，内地基本见不到，除非是在一些名山大川里”<br/>　　<br/>　　    “又是驹，又是马的，你说的太专业了，大哥通俗一点解释呗”<br/>　　<br/>　　    王惊蛰深深的吐了口气，手指着东边的方向说道：“如果我没看错的话，顺着山脉的走势从这里一直往前，应该会另有一番天地，于喜马拉雅山脉海拔六千米的高度格格不入的一番天地，那里可能是另外一个世界”<br/>　　<br/>　　    陈三岁和孔良听闻，都半天没回过神来，之前王惊蛰就曾经说过也许在这片山脉里有一处春暖花开的地方，两人都是将信将疑，但这时他又说山脉里可能还有另外一个世界，他俩就更没办法领悟了。<br/>　　<br/>　　    孔良咽了口唾沫，说道：“你要是没开玩笑的话，那可就太惊人了”<br/>　　<br/>　　    王惊蛰抬起手，指着自己的脑袋说道：“我的经验告诉我，所看到的不是玩笑，而是真正的‘白驹过隙，青鱼落潭’的风水山脉走势”<br/>　　<br/>　　    王惊蛰所说的，是一个很古老的风水现象，用通俗易懂的话来解释就是，群山环绕下将一整片区域完全于山外隔绝开来，将这一片的中间地带，独成一处风水格局。<br/>　　<br/>　　    打个最简单易懂的比方，就是在寒冬腊月的北方，屋外可能大雪纷飞，但屋内由于供暖却异常的暖和，基本就是这个意思了。<br/>　　<br/>　　    “那头獒王，我想应该就是一直生活在那里，然后可能每隔一段时间就出来游荡一次，这回可能正是它返回老巢的时候了……”<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
