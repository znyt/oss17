　　第704章
<br/>　　汉子持枪而立，挠了挠头：“为何要闯宫门，因为他们不让我进呐？”
<br/>　　一众人面面相觑，这汉子说得理所应当，错得仿佛是大家。
<br/>　　“来人，拿下去，打进大牢。”
<br/>　　“慢。”李乐天伸手拦住了鱼龙卫，看着面前这汉子，道：“喏，你叫什么名字？”
<br/>　　“西凉，马超马孟起。”
<br/>　　李乐天眯起了眼睛，凉州来的，莫非是程大雷暗中派来的。这种又蛮又强的性格，当真很有蛤蟆寨的气质。
<br/>　　“你为何要闯宫门？”
<br/>　　“我要见皇帝。”
<br/>　　“喔。”李乐天道：“见皇帝为何？”
<br/>　　马超眯起眼睛：“你是什么人，我为何什么事都同你说？你是皇帝么？”
<br/>　　李乐天不怒反笑，道：“偏偏你还是猜对了，朕正是帝国天子。”
<br/>　　马超上下打量李乐天，这眼神其实十分无礼，不过李乐天偏偏是个很有度量的人，对这汉子的眼神也不以为意。
<br/>　　在马超眼中，李乐天身着八分绣龙袍，头戴紫金冠，玉树临风不怒自威。
<br/>　　天子天子，天下唯其一子。
<br/>　　马超也收起顽劣表情，慢慢道：“我想要出人头地，要大富贵。”
<br/>　　“倒也是坦诚。”李乐天点点头：“要大富贵须有大本事，你，有这个本事吗？”
<br/>　　马超将枪扛在肩头，睥睨道：“应该有。”
<br/>　　李乐天目光转了一圈，无论如何，把守宫门的卫士都千挑万选出来。击败七八名卫士，跨过宫门十步，这已足以证明这莽撞汉子的本事。
<br/>　　李乐天心中一动，仰头遥望天际，呐呐道：“冥冥之中，当真有天数么。”
<br/>　　……
<br/>　　于此同时，程大雷也见到了宋慈。
<br/>　　年纪三十岁左右，模样蔫蔫的，一副混得不如意的样子。据其所说，来自江南，本来是个小衙门的仵作。也不知抽了哪门子风，一路逃到这里，要给程大雷做事。
<br/>　　反正无论他说什么，程大雷都会相信，也是系统安排的。
<br/>　　“宋大人来得正好。”程大雷道：“我恰恰有个监察的差事与你做，先休整一下，熟悉熟悉这里，然后便开始工作。”
<br/>　　宋慈大概属于混得不行的那种人，在衙门里做一个刀笔小吏，勉强能混个温饱。却是没想到，刚刚到了这里，程大雷便委以重任。这监察官的职责，是一个管官的官，要辖制程大雷手下的文官武将。
<br/>　　程大雷给自己这样一份差事，对自己的信任是否太过分了。
<br/>　　士为知己者死，君以国士待我，我将以国士报之，现在这个时代的人，或多或少都有这种心思。
<br/>　　宋慈感激涕零的走了，看这意思，恨不得当场把脑袋剁下来送给程大雷。
<br/>　　对这件事情最为震惊的却不是宋慈，乃是崔白玉。她原先就在琢磨，程大雷会把这监察官的位置交给谁，在她看来，程大雷手下并没有谁能胜任。偏偏这个时候，就来了个宋慈，程大雷几乎毫不犹豫就将这重任交给了他。目前宋慈并没有表现出任何本事，那么程大雷又是凭什么确定他的能力。
<br/>　　而除了这件事外，她还帮着处理另外一件事。帮着整理凉州城的农田数目，城外已经开始种植，是程大雷自己拿出的麦种。
<br/>　　这种小事本用不着程大雷做，但他对这件事却格外关注。
<br/>　　崔白玉心中填满困惑，虽然朝夕相处，经常能见到程大雷，可是，崔白玉依旧看不透程大雷。
<br/>　　他露出的破绽太多，多到程大雷根本不去隐瞒。可当自己想顺着这些破绽顺藤摸瓜寻找真相的时候，却发现一点痕迹都无。
<br/>　　关于这件事，程大雷也有他的说法。类似于神仙梦中传艺，师父是南极仙翁，和太白金星拜把子之类的话。
<br/>　　这些话，蛤蟆寨还颇有信徒，不过靠这些哄骗普通人还行，至于崔白玉根本不相信这些。可她又没有能力看透程大雷身上的迷雾，渐渐地，崔白玉也开始想……
<br/>　　莫非程大雷背后当真有仙人传艺。
<br/>　　这成了崔白玉每天忙碌工作中一项困扰，除此之外，她还有另外一个困扰。
<br/>　　便是长安城发生的事情：当今天子准备对相府动手。
<br/>　　这件事属于极大的机密，整个凉州城都没几人知道，偏偏是冲崔白玉说了。还详细询问了相府的情况，估计李乐天有多少胜算。
<br/>　　以崔白玉的估计，李乐天几乎是没有胜算的，然而，凡事都怕万一。假若崔白玉能将这消息传递到相府，李乐天的垂死挣扎便没有多少意义。
<br/>　　可问题是，崔白玉没有能力将这消息传递到相府。如今她大概是养在城主府的金丝雀，虽然是能和程大雷朝夕相处，问题是只能和程大雷朝夕相处。
<br/>　　程大雷现在也关注着长安城的情况，长安一变，牵一发而动全身，也将影响到凉州这边。
<br/>　　程大雷命令喜字队严密关注长安城的情况，一有消息便立刻禀报回来。然而，这也是无用之功，两地相隔甚远，一来一去需要十数天的时间。便是消息传到这里，怕也只有结果，程大雷并不能做什么。
<br/>　　如此，程大雷倒是动了心思，实在不行自己走一趟长安城。有什么情况可以及时做出反应，但这个想法却也是想想而已。
<br/>　　一来，凉州城本身就是一团乱麻，还有许多事情需要处理。二来，自己只要敢在长安城露面，便必定是人人喊打的局面，无论是崔相或者是李乐天，都以弄死自己为目的，说不得他们会放下两家仇恨，先对付自己。
<br/>　　如此，程大雷只能做个袖手旁观的角色，只能静等着长安城的结果。
<br/>　　时间一天一天的过去，凉州城这边是一如既往的忙碌，长安城也十分平静。但在长安表面的平静下，却是暗中波涛汹涌，有许多事情发生。
<br/>　　一队贩草药的客人混入长安城，几名挑柴的大汉进入长安……这些人表面上都是普通的商人，看上去并没有异常。
<br/>　　但在不知不觉间，却有八百这样的人潜入了长安城。
<br/>　　李乐天坐在未央宫中，静等着自己需要的力量到来，虽然不是很多，却足够令他做一次亡命徒。
<br/>(本章完)
