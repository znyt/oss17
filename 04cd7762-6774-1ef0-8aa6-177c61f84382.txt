        “大哥，如果你真的跟筱颍在一起，我会恨你。”傅昼景从沙发上站了起来，双手握拳，冷冷地说道。
<br/>
<br/>        傅夜沉没搭理他的话，而他撂下这话后，便自顾自地只身离开了。
<br/>
<br/>        要恨，就恨吧！
<br/>
<br/>        反正这世上，恨他傅夜沉的人，不止他傅昼景一个。
<br/>
<br/>        楼上，童筱颖黯然神伤地垂下了眼帘，手下抓着扶手的力度，情不自禁地紧了紧。
<br/>
<br/>        如果不是他们提到爷爷傅炎廷，她还真没意识到，大哥跟她领证到现在，都未带她去见过他的爷爷和母亲。
<br/>
<br/>        其实，大哥也是担心这件事情，才不带她回临海城的傅家大庄园的吧！
<br/>
<br/>        爷爷和婆婆，一定不会接受她……
<br/>
<br/>        还有昼景……
<br/>
<br/>        傅昼景走后没多久，落以柔牵着小爱回到了总裁办。
<br/>
<br/>        她一个劲地当着傅夜沉的面，不停地夸小爱又乖又可爱，她很喜欢小爱。
<br/>
<br/>        起初，一大清早，傅夜沉醒来的时候，刚打完电话让人给筱颖送衣服过来，小爱这个时候也跟着醒来了。
<br/>
<br/>        然后它那狗屁股端坐在地上瞪着两只黑溜溜的圆眼睛，盯得他浑身不自在。
<br/>
<br/>        傅夜沉总觉得小爱好像在对他期待着什么？
<br/>
<br/>        但是，他没有养宠物的经验，而筱颖当时还在睡觉，又不忍心吵醒筱颖，所以他只能将小爱牵下了楼。
<br/>
<br/>        刚巧，落以柔进了总裁办看到对一只白色博美犬束手无策的傅夜沉，于是好心帮他解了这困惑。
<br/>
<br/>        落以柔说，狗狗每天早上都要出去上厕所。
<br/>
<br/>        傅夜沉便让落以柔将小爱牵了出去。
<br/>
<br/>        不得不说，落以柔真的是傅夜沉的得力帮手，不管是工作上还是生活上，她懂得察言观色，懂得他需要什么，不需要什么。
<br/>
<br/>        就连童筱颖都能感觉得出，这个落以柔对傅夜沉应该是有那方面的意思，就是不知道傅夜沉他心里是怎么想这件事情的。
<br/>
<br/>        童筱颖心里估摸着，大哥可能对其他女人爱慕他的这事不大敏感，那么大哥每天心里都在想些什么？
<br/>
<br/>        当落以柔也离开后，傅夜沉的总裁办里才彻底安静下来。
<br/>
<br/>        童筱颖这才去洗漱，换上了傅夜沉不知何时为她准备好的另一套干净的衣服。
<br/>
<br/>        就在她准备下厨做早餐的时候，傅夜沉忽然走了上来，说道：“筱颖，我们俩得回一趟临海城。”
<br/>
<br/>        “就现在？”
<br/>
<br/>        “嗯。”
<br/>
<br/>        “那小爱也一起带去？”
<br/>
<br/>        “小爱交给小落就好。”他云淡风轻地说。
<br/>
<br/>        童筱颖倒是一点都不反对。
<br/>
<br/>        虽然知道落以柔对傅夜沉别有用心，但落以柔至少不会做伤害傅夜沉的事情。
<br/>
<br/>        临近中午时，她跟着他赶回来临海城的傅家大庄园，下车前，他特意拿了一双崭新的平底鞋给她换上，她还以为是自己扭伤脚的缘故，心里暖暖的，觉得大哥很贴心。
<br/>
<br/>        下车后，童筱颖忽然发现，哪怕阔别三年，傅家大庄园仍旧是老样子，一点都没变。
<br/>
<br/>        府邸大门口那对石狮，威武霸气。
<br/>
<br/>        只是她这次进这张大门的身份，不再是傅家的二少奶奶了。
<br/>
<br/>        偌大的餐厅里，华丽典雅的烛火吊灯下，是一张超级气派的大圆桌。
<br/>
<br/>        主位上坐着爷爷傅炎廷，以及傅夜沉的母亲冷太太，桌上放着精致丰盛的食物，但食物的香气，并未缓和餐桌前这异样的僵硬气氛。
<br/>
<br/>        今天的重点，自然不是吃团圆饭这么简单。
<br/>
<br/>        童筱颖一看到这爷爷傅炎廷皱着眉头，心里就“咯吱”着很不舒服，但是为了傅夜沉，她又不得不装出自然的态度来。
<br/>
<br/>        “爷爷，筱颖的事，我想阿景一定告诉您了，来龙去脉，我就不多说了。”傅夜沉淡淡地开口，餐桌底下，他紧紧地牵着童筱颖的手，示意她不要紧张。
<br/>
<br/>        童筱颖抿了抿唇，看了看爷爷傅炎廷，又看了看冷太太，弱弱地唤了声：“爷爷、婆婆。”
<br/>
<br/>        “……”冷太太默不吭声，没应话。
<br/>
<br/>        傅夜沉瞥了一眼自己母亲的反应后，在爷爷傅炎廷未动筷子之前，主动拿起了筷子，夹了一块宫廷苏造肉放到了童筱颖跟前的碗中，温和地说道：“筱颖，别饿着肚子了，你现在是两个人吃饭。”
<br/>
<br/>        两个人吃饭？！
<br/>
<br/>        一时间，童筱颖还未反应过来。
<br/>
<br/>        冷太太却最先发出惊怔的质疑声：“筱颖怀孕了吗？”
<br/>
<br/>        “是的。”傅夜沉嘴角微扬。
<br/>
<br/>        爷爷傅炎廷倒是十分淡定盯着傅夜沉，严肃地反问道：“是你的，还是阿景的？”
<br/>
<br/>        这话问得也太怼人了吧！
<br/>
<br/>        “……”这回反应过来的童筱颖，不得不冲着爷爷傅炎廷和婆婆冷太太尴尬地咧嘴一笑，同时也在桌底下悄悄地拉了拉傅夜沉的衣角。
