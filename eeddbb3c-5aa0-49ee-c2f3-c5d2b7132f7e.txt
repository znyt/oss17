陆寒霆没有说话，他只是抿了一下薄唇，然后缄默的迈开长腿离开了幽兰苑，他高大挺括的身影很快就与外面清冷的黑夜融为了一体，直至消失不见。
<br/>
<br/>        陆茵茵收回了目光，她看向陆老夫人，“妈，我一直不明白的是夕绾怎么嫁给寒霆的，我查过了，夕绾是代嫁的对吗，当时整座海城都知道夏夕绾是从乡下回来的土包子，妈，我太了解你了，你坐镇陆家这么多年又宠爱寒霆，他的婚姻大事你一定会替他全权把关，你竟然会放任一个乡下的土包子替嫁给了寒霆？妈，你是不是有什么事情瞒着我？”
<br/>
<br/>        陆茵茵可是陆老夫人的亲闺女，母亲那缜密的心思和雷霆手段她会不清楚吗，她派人去查了陆寒霆和夏夕绾的这段婚姻，这段婚姻疑点重重，一点都经不起推敲，今天她过来就是想好好问一问老夫人的。
<br/>
<br/>        陆老夫人动作优雅的切着盘子里的牛排，并没有看陆茵茵，“茵茵，既然你已经派人去调查过了，那你还来问我干什么？”
<br/>
<br/>        陆茵茵一僵，“妈，你是不是怪我在你背后动小手脚，那…那都是因为你太偏心了，你一直在为寒霆筹谋，就连这段替嫁都是经你的手办的，就算我不查，哥哥和嫂子来了也会查的，现在你也知道夕绾就是子羡一见钟情的那个女孩了，他们才是天造地设的一对，而寒霆的身体你我都清楚，这种心理疾病是无法治愈的！”
<br/>
<br/>        “说够了吗？”老夫人静静的听陆茵茵说完，然后幽幽的问了一句。
<br/>
<br/>        “妈，我…”
<br/>
<br/>        “寒霆从来没有跟你们抢过任何东西，自他离开陆家的那一刻起，陆家的东西，他一样都不要，绾绾是他的妻子，是他的陆太太，你们现在是什么个理，跑到海城来抢人了，还这么雄赳赳气扬扬？”
<br/>
<br/>        陆茵茵顿时语结，在老夫人不喜和严厉的目光里，她也不敢真去顶撞，虽然陆寒霆七年前已经离开了帝都的陆家，陆家的东西他一样没有要，这些年哥哥陆司爵对这个有病的儿子也一直不闻不问，两个人好像已经断绝了父子关系一样。
<br/>
<br/>        但是，如果不是嫂子柳招娣无意之间看到了哥哥立下的那份遗嘱，谁会知道陆司爵的遗嘱里竟然将陆氏那个庞大的商业帝国和基业都交给了陆寒霆，在陆寒霆和陆子羡这两个儿子之间，哥哥还是选择了陆寒霆作为自己的继承者。
<br/>
<br/>        陆茵茵知道，哪怕十年过去了，二十年过去了，柳璎珞那个风华无双的女人依然活在所有人的记忆里，她的哥哥，她的前夫…都是！
<br/>
<br/>        三十年前的帝都城异常的繁华，那个时代的人都知道那时出了两个少女，一个俏丽无邪，一路升级打怪成为传说，一个清冷绝色，创下Fly珠宝经典，成为一代时尚教母，当时的豪门子弟对这两个少女趋之若鹜，帝都四大豪门厉陆徐夜更是席卷其中。
<br/>
<br/>        只要想到这些，陆茵茵的胸膛里就窜起一股怨恨的火气，“妈，我几乎能笃定这场替嫁都在你的谋划里，你比我们之间的任何一个人都先知道了夕绾的先机，作为奶奶，你给予寒霆最好最周全的爱就是给了他一个好妻子，好太太，对吗？”
