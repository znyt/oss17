        “大少奶奶？！”属下？接她？

<br/>        为首的男人会心一笑，接着解释道：“因为您脖子上戴着的，是大少爷留给您的祖传玉坠！”

<br/>        祖传玉坠？

<br/>        难道，面前的男人，不知道吗？

<br/>
<br/>        “傅氏集团？！”苏漫雪震惊，喜上眉梢得有些支支吾吾，“就是那个产业扩及娱商政三界，净利润称霸全国连续十年第一的傅氏集团？！”
<br/>
<br/>        “是的，大少奶奶。”刘管家微笑着点了点头，傅氏集团在国内外都很出名，几乎无人不知。

<br/>        苏漫雪连忙激动的接着问：“那你家大少爷是？”

<br/>
<br/>        苏漫雪听得心花怒放了，她没有在做梦吧，傅氏集团的大少爷，竟然要娶她！
<br/>
<br/>        如果能嫁给傅家大少，那么她将再也不会住在这么寒酸的房子里，她会过上她梦寐以求的荣华富贵的生活！
<br/>
<br/>        “好，我跟你们走。”苏漫雪立马点头道，早已忘了这块玉坠真正的主人。
<br/>
<br/>        苏漫雪一件自己的行李都没有收拾，就跟着刘管家走了出去。
<br/>
<br/>        反正嫁给了傅大少，她就有花不完的钱，这些廉价的破烂货，已经配不上她！

<br/>        下了楼，苏漫雪第一次坐上价值上千万的豪车，心潮澎湃，更对那位傅家大少爷充满了无限的期待。
<br/>
<br/>        傅大少爷会是一个什么样的男人？
<br/>
<br/>        苏漫雪既紧张又激动，坐在驾驶座后，想到诺筱颖，不由地有些心慌，试探性地问：“你家大少爷，知道我叫什么名字吗？还有，你家大少爷，为什么突然要娶我？”
<br/>
<br/>        “这个嘛……”刘管家顿时一时间也答不上来，因为大少爷并没有告诉他缘由，只是吩咐他，让他去希望公寓1808号房接一个拥有傅家祖传玉坠的女人，而这个女人，就是未来的傅家少奶奶！
<br/>
<br/>        刘管家摇了摇头说：“大少爷也没有告诉我您的名字。”
<br/>
<br/>        “这样啊……”苏漫雪假装可惜的说道，但心里早已心花怒放，哈哈，真是老天爷都在帮她！
<br/>
<br/>        苏漫雪连忙坐正，装作一副大家闺秀的模样说道：“刘叔，你好，我叫苏漫雪。”

<br/>
<br/>        ‘大少爷，已经成功接到大少奶奶，苏漫雪。’

<br/>        漫雪，满天飞雪，真是个冰清玉洁的美丽名字！
