　　将陨落心炎炼化，他不仅感觉自己力量有所提升，同时，心中也生出一抹非常奇异的感觉。
<br/>　　这种感觉来自于脚下，岩浆世界的更深处，仿佛遥相呼应，受到某种召唤一样，感觉非常的奇妙。
<br/>　　难道是因为自己吞噬了陨落心炎，所以能感受到另一枚异火的气息？
<br/>　　想到此处，慕清霄眼眸中闪过一丝惊喜。
<br/>　　原本，在他的计划中，寻找另一枚陨落心炎，可能需要不少时间，现在看来，这段时间倒是省了下来。
<br/>　　感受到若有若无的召唤，慕清霄不再迟疑，向岩浆底部掠去。
<br/>　　召唤传来之地，距离颇远，越是往下，温度与压迫力就越高。
<br/>　　粘稠岩浆所带来的压迫力自四面八方涌来，不过，随着阴阳灵气化作防御，压迫力也随之减轻。
<br/>　　岩浆底部有些什么，慕清霄隐约清楚一些，虽危险了点，但他非去一趟不可。
<br/>　　...
<br/>　　...
<br/>　　茫茫无尽的岩浆世界中，映入眼中的是一片赤红，一道身影穿梭其中，婉若游龙。
<br/>　　此时，阴阳灵气护罩中，慕清霄脸色平静，距离他下潜，已经有十分钟了，距离地面，他也不知道有多远。
<br/>　　依旧保持着原本的速度，不是很快，却也不慢。
<br/>　　虽然，情绪不表现在脸上，但慕清霄心中却保持着警惕，他可不想阴沟里翻船。
<br/>　　由于下潜的颇深，乃至于周围的压力也越来越恐怖，对于寻常斗皇，在下潜恐怕就祸福难料了。
<br/>　　哪怕是斗宗，想要继续往下，都需要一颗坚定的心。
<br/>　　因为，谁也不知道，下面会遇到什么，稍有不慎，恐怕就会陨落于此。
<br/>　　陡然，先前的召唤变得更加清晰，使得慕清霄身形一滞，目光环顾四周，暗道：“距离不远了。”
<br/>　　目光环绕间，映入眼中的尽是千篇一律的赤红，若非召唤变得清晰，他都怕是以为，一直在原地徘徊。
<br/>　　心神守一，慕清霄不禁加快下潜的速度，向着下方掠去。
<br/>　　半刻钟后，速度缓缓慢了下来，慕清霄面露凝重之色，谨慎的打量起四周。
<br/>　　这里依旧是一片赤红色的世界，看上去并非是岩浆世界的尽头。
<br/>　　但是，在慕清霄的脚下，却是一片暗红色的岩浆，可仔细看去，岩浆中隐隐参杂着一抹令人余悸的黝黑。
<br/>　　相比头顶，脚下更显得诡异莫名，仿佛存在着某些东西。
<br/>　　最重要的是，若有若无的召唤，在这里显得格外清晰，恐怕就在这附近了。
<br/>　　就在慕清霄打量周围的同时，眼中却是出现了奇异的一幕，只见脚下的岩浆缓缓蠕动起来。
<br/>　　随后，一道暗淡的乳白色光芒升起，旋即浮现在了眼前。
<br/>　　在慕清霄的视线中，只见乳白色光芒自岩浆中凸显而出，露出了本来的面目，令得他一阵愕然。
<br/>　　愕然过后，眼眸中闪过一丝精芒，这熟悉的气息，只不过较弱而已。
<br/>　　此时，乳白色的光芒中，正躺着一具已化作白骨的骨骸，光芒就是从骨骸上散发而出。
<br/>　　而发出乳白色荧光的物体，并非其他东西，通体透明，宛如无形之物，与先前吞噬的陨落心炎，完全一样。
<br/>　　就在慕清霄沉吟之际，忽然一道炽热的劲风，诡异的向他射来，虽有岩浆的高温掩饰，依旧被敏锐的察觉到。
<br/>　　并没有躲闪，只见他随手一挥，劲风就被化解。
<br/>　　放眼望去，约莫十几米外，只见一道红色身影游荡在岩浆之中，身体赤红，与周围岩浆颜色一样。
<br/>　　若是不仔细看，恐怕真的难以发现，在这岩浆领域中，竟然有着生物的存在。
<br/>　　赤红色身影双脚直立，通体赤红，其上布满着红色鳞片，犹如天然的保护色，约莫半丈长的红色尾巴微微摆动，脚掌大的出奇。
<br/>　　圆滚滚的脑袋上长着一对细小的瞳孔，毫不掩饰的透露着凶光，嘴巴微微咧开，露出两排尖锐的牙齿。
<br/>　　整个看上去，就像是能够直立行走的蜥蜴人一样，而且气息上来看，居然是斗灵级别。
<br/>　　慕清霄面露古怪之色，当真是世界之大无奇不有，若是换做以前，鬼才相信有这种东西存在。
<br/>　　“吼吼！”
<br/>　　就在慕清霄打量岩浆生物的时候，后者也凶狠的盯着他，对峙过后，岩浆生物瞳孔中凶光大盛、
<br/>　　旋即，狰狞的最终发出两声沉闷的嘶吼，然后尾巴一甩，锋利的兽爪就划破岩浆，径直向慕清霄冲了过去。
<br/>　　“不知死活。”
<br/>　　望着朝自己冲来的岩浆生物，慕清霄面无表情，弹指之间，一道锐利的剑气自指尖疾射而出。
<br/>　　“铮...！”
<br/>　　好似是察觉到剑气的恐怖，岩浆生物尾巴一摆，想要躲开，但为时已晚。
<br/>　　剑气自岩浆生物的身躯上划过，切开岩浆，最后消失的无隐无踪。
<br/>　　岩浆生物凶狠的表情凝固在脸上，最后腰间出现一条整齐的剑痕，伴随着一道细微的声音，上半身与下半身彻底分离。
<br/>　　身躯被切成两半，岩浆生物嘴中发出一道凄厉的嘶吼声，身躯如水波一般，在岩浆中扩散，直至消失。
<br/>　　岩浆生物消失，其上却浮着一颗赤红色的珠子，普通石子大小，正闪烁着淡淡的炽热光芒。
<br/>　　“反应倒是挺快。”
<br/>　　说罢，慕清霄随手一卷，赤红色珠子就出现在手中，好奇的打量起来。
<br/>　　珠子刚入手，其中就传来一股狂暴的火属性能量，极其精纯，有些类似于魔核，其中蕴含的能量，堪比斗灵修炼两三天的量。
<br/>　　对于专修火属性的人来说，绝对是不可多得的好东西，只是能量太过狂暴，普通修炼者不能直接服用。
<br/>　　不过，对于慕清霄来说，并无大碍，反倒是补品，异火他都能直接吞噬，何况一枚小小的火属性珠子。
<br/>　　就在岩浆生物死后，在岩浆领域的西边方向，那里的岩浆，忽然间传来一阵剧烈的骚动。
<br/>　　将珠子收进储存空间，视线向着岩浆领域西边看去，只见一大群红色身影，犹如鱼群一般，蜂拥而来。
<br/>　　慕清霄嘴角牵起一丝笑意，道：“四百多只，九成都是斗灵，六只斗王，两只斗皇级别，真是豪华的阵容。”
