　　“吾，上清灵宝天尊玉宸大道君通天，今日成圣！”通天那宏大的声音传遍八荒。
<br/>　　轰！
<br/>　　那消散不到一天的惊天圣威异象再次出现在洪荒之中，剑气凌天，剑光纵横，紫气浩荡，天音阵阵，龙飞凤舞，飞虎踏空……
<br/>　　各种各样的异象不断出现，出现在洪荒各处。
<br/>　　巨大的通天剑柱横贯洪荒，天地各处出现巨大的剑道之路，大道法则动荡，宛若实质般显现在众人眼前。
<br/>　　“靠，又是昆仑山方向，能不能一次性地搞完，非要一次又一次地打击我们。”
<br/>　　“还能不能愉快的玩耍了。你们这样玩，我们怎么办？”
<br/>　　“成圣都选在了同一天，这样真的好吗？”
<br/>　　“货比货，该扔，人比人，气死人！这人与人之间，简直没法比。”
<br/>　　“哎，三清都成圣了，只剩下西方二人与红云了，他们怎么还没有成圣？”
<br/>　　“要是能够拜入圣人门下，那该多好啊！”
<br/>　　“……”
<br/>　　一日成两圣，惊天动地，洪荒各处都变得不平静了，他们纷纷将目光投向昆仑三清，同时也在暗暗揣测，剩下的三人何时成圣。
<br/>　　昆仑山中，不管别人如何去猜测，三清并没有去混沌开辟道场，也没有去别处开门收徒，而是直接回去闭关了，此次突破他们收获很大，需要静心下来，自己参悟，还需要将自身的法力转化为圣元，成圣后，修炼的法力较之之前，已经有了根本的区别。
<br/>　　法力转化为圣元，不仅带有天道之力，而且带有圣人威压，一身实力比之前强大无数倍。
<br/>　　天庭太阳神宫中，妖帝帝俊与东皇太一对视良久，突然，帝俊开口道：
<br/>　　“二弟，我妖族的实力还是不够啊，对比这些圣人，我们妖族可是没有能力抵挡啊。”
<br/>　　“是啊，大哥，你说的对，但形势就是如此，我们又该怎么抵挡呢？”
<br/>　　“二弟，你觉不觉得我们天庭还缺点什么？”帝俊突然向太一问道。
<br/>　　“呃？不缺啊，我们这里什么都有，缺什么啊？”太一微微一愣，不知帝俊想表达什么意思，但他还是微微一想，认真答道。
<br/>　　“你不觉得天庭却少一个女主人吗？”帝俊沉声道。
<br/>　　“哦！原来如此，大哥你是看上那里的女神了，看上了就娶回来啊。”太一大笑道。
<br/>　　“你也知道，我们金乌一族先天至阳，很难诞生子嗣，但若是能够找到一位身具太阴之力的女子，与之交合，便可能出现后代。”帝俊沉声道。
<br/>　　“大哥说得在理，天庭不能无后。身具太阴之力的女子，太阴星上不就有两位吗，大哥是看上谁了？”
<br/>　　“上次去昆仑山论道，我看见太阴之主的弟子嫦羲便是太阴之体。”
<br/>　　“哦？大哥是看上她啊，没事，这件事包在我身上。太阴星是至尊星辰之一，最为重要，也是时候将它归为我们妖族所有了。”太一拍了拍胸膛大声道。
<br/>　　“二弟，不可鲁莽，这件事还需要从长计议。”
<br/>　　“嗯。大哥，还有太阳星上的那个羲和，她也是金乌化形，不过，我们代表的是至阳，她却是阳中有阴，对大哥应该很相配吧，要不，大哥考虑考虑？”太一突然说道。
<br/>　　“羲和啊？”
<br/>　　帝俊听到此名，眼神突然间变得有些柔和了，他其实对羲和也是有感觉的，之前住在太阳星上，两人已经见过多次，他心里觉得羲和十分适合自己，但之前因为事情太多，自己一时间也没有往那方面去想，此次说起嫦羲之事，才忽然间想起她。
<br/>　　其实他心里更觉得，自己对羲和那才叫爱，对嫦羲更多的是至阳与至阴之间的吸引，还有她师傅那太阴之主的身份。
<br/>　　“大哥，既然你心中有她，那干脆一并娶了就是，何必纠结那么多？”太一见此，大声道。
<br/>　　“也是，我帝俊不就是看上一个女子吗，怎么变得婆婆麻麻了，这可不像我啊！”帝俊突然醒转，大声道。
<br/>　　之后，两人商量此事，伏羲听说此事后，便推荐自己的妹妹去提亲，他知道自己的妹妹有红绣球，那是一件可以定天、地、人三婚的特殊先天灵宝，若是此事有成，自己的妹妹也能得到一些功德。
<br/>　　两人都知道他的心思，不过，这件事的确是女娲去最为合适，所以，他们稍微一想，便答应了。
<br/>　　之后，伏羲直奔混沌，来到娲皇宫，见到自己的妹妹，并说明来意。
<br/>　　女娲沉吟了片刻，觉得此事不管是对妖族，还是对自己都是有利的，就答应了。之后，她直奔太阳星而去。
<br/>　　……
<br/>　　太阳星上，深处一片山谷之中，有一座红色的宫殿，红色本是象征着喜庆与热烈，但此宫殿之中，却充满冷清与孤寂。
<br/>　　这天，一位绝色女神从宫殿之中款款而出，她蛾眉柳腰，朱唇秀目，长长的头发，纤纤玉手，翩若惊鸿，婉若游龙，一身红色衣裙，顾盼之间，风情万种。
<br/>　　此女名为羲和，与帝俊、太一一样，都是金乌所化。金乌本是至阳的代表，按理来说，并不会出现女的，但凡事也有例外，太阳本就是至阳之力的汇聚之地，阳极生阴，羲和便是那太阳星上的阳极生阴之地所孕育出来的。
<br/>　　这种几率很小，小到数万只金乌之中才有可能出现一只雌性金乌。而羲和却能在三只金乌中占据那唯一的女性，不得不说，这是极为幸运的。
<br/>　　这一天，羲和正如往常一般观看太阳星上的烈火，突然间太阳星外进来一位威严的丽人，正是女娲圣人。
<br/>　　“这位可是羲和妹妹！”
<br/>　　女娲一进太阳星，便知道羲和的住处，一步踏出，便来到羲和面前，见到羲和，便眼前一亮，立刻打招呼道。
<br/>　　“原来是女娲圣人，小妹正是羲和，在此见过圣人，不知圣人找我所谓何事？”
<br/>　　羲和看见女娲，便认出她来，顿时起身行礼，但心中却极为疑惑，不知大名鼎鼎的女娲圣人为何会来此？
<br/>　　女娲见羲和困惑，她自己也不想拖拖拉拉的，便直接开口，表明此行目的：
<br/>　　“羲和妹妹，我此次前来是为人做媒的。妖帝帝俊想来妹妹也不陌生吧，此次正是那帝俊看上了妹妹，想要招你做他的妻子。”
<br/>　　对于帝俊，羲和自然是知道的，两人之前都住在太阳星，见过多次面，他对帝俊的印象也极为深刻，知道那是一个威严霸气，有勇有谋的男子。
<br/>　　说起来，她对帝俊也是有好感的，但这种好感绝对不到谈婚论嫁的地步，她还从没有想过此事，所以她也并不急着回答，一时之间，大殿陷入沉默之中。
<br/>　　女娲也知道此事急不来，所以也就没有催她，只是让她好好想想。
