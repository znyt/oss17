　　天焚炼气塔中的炙热温度确实恐怖，若是没有准备，钻入体内的热量，足以将人体自内而外烤熟。
<br/>　　当然，对于慕清霄来说，塔内的温度恰恰刚好。
<br/>　　他不仅继承了凤凰该有的特性，又保留了人族的喜好，对于炽热的东西，虽不至于喜欢，但也不讨厌。
<br/>　　望着自小医仙身躯中冒出的毒气，慕清霄抬起手，轻轻放在她的香肩上，闭上双眼，查看起她的情况。
<br/>　　此时，在他的视线中，小医仙的心脏部位，冒出一簇诡异的火焰，出乎常理。
<br/>　　火焰无形无状，并不算太强大，但心脏永远都是人体最脆弱的部位，稍有不慎，可能就会导致严重的后果。
<br/>　　火焰散发着极高的温度，炙烤着小医仙的经脉，骨骼，各种器官，若是这样下去，再过不久，她体内的经脉器官都会坏死。
<br/>　　最重要的是，在极高的温度下，小医仙体内的毒气，瞬间狂暴起来。
<br/>　　甚至，有一丝丝从普通封印中窜出，逃也似的离开她的身体，不敢多呆。
<br/>　　“不能如此下去了，这是最后一次加固封印，得找机会到塔底去一趟，否则，事情会很难办。”
<br/>　　说罢，慕清霄双手结印，阴阳灵气渡入小医仙的体内，游走在经脉中，逼迫着毒气，最后汇聚在小腹处。
<br/>　　“封！”
<br/>　　封印刚设下，阴阳灵气形成包围网，将她体内的毒气包裹起来，普通的封印形成。
<br/>　　阴阳灵气封印毒气的同时，毒气也在慢慢侵鉵灵气，小医仙终究不是阴阳灵气的载体，厄难毒体也在滋生毒气。
<br/>　　若是慕清霄自己，他体内的阴阳灵气源源不断，永远将毒气封印下去，都不是问题。
<br/>　　“现在没有凝成毒丹，你根本不可能以自己的实力，将毒气凝聚在一起，虽然实力在提升，可厄难毒体，依旧不是你现在能控制的。”
<br/>　　小医仙抿了抿红唇，脸色通红的望着他，玉额上都冒出一层香汗，刚才毒气的溢出，确实将她吓的够呛。
<br/>　　说着，慕清霄伸出右手，隔空对着小医仙身体一吸，随即一缕有些扭曲的无形火焰，被强行扯了出来。
<br/>　　望着在掌心跳动的无形火焰，黑色神火一闪即逝，无形火焰已经被吞噬殆尽。
<br/>　　待心火泯灭，小医仙脸色也渐渐恢复了正常。
<br/>　　“不必担心，很快就能找到封印毒体的办法，有我在你身边，你安心修炼就可以。”
<br/>　　闻言，小医仙微微点头，上前一步，臻首轻轻靠在他的胸膛上，只有他温暖的怀抱，才能使得她静下心神，摆脱厄难毒体的困扰。
<br/>　　两人相拥，令得练气塔中的老生，纷纷投来异样的目光。
<br/>　　...
<br/>　　...
<br/>　　片刻后，慕清霄与小医仙就在天焚炼气塔中闲逛起来。
<br/>　　这座深埋地底，仅仅只露出一截塔尖的天焚炼气塔，但塔内的面积，却宽敞得有些出乎意料。
<br/>　　随着这一路走下来，他便发现，仅仅是这第一层塔内空间，便足够五百名学员同时修炼了。
<br/>　　在呈圆形分布的塔内空间中，还搭建着一间间修炼室，其中大小不一。
<br/>　　当然，此时的修炼室，几乎已经被学员给占满，而且这些修炼室的门口上方，都是挂着一些红木牌。
<br/>　　高级，中级，低级三种，对应的就是塔内修炼室的等级。
<br/>　　学员进入等级越高的修炼室，火能的费用是不变的，唯独需要相对应的实力。
<br/>　　否则，哪怕冲进去，也会快速被别的学员给丢出来。
<br/>　　所以，想要获得更好的修炼条件，就需要拼命的努力。
<br/>　　实力差的学员，就只能在外围的低级修炼室修炼，而实力强大的，就能在中级，或者高级修炼室中修炼，效果截然不同。
<br/>　　比如，低级修炼室中，能够让学员的修炼速度增幅两成，而中级修炼室却是四成，高级就是六成。
<br/>　　长期修炼下来，学员们就会发现，他们早就被同期甩了几十条大街。
<br/>　　正是这种局面，促使他们更加拼命。
<br/>　　不面临困境，人的潜力永远不会彻底爆发出来，而学院所提供的，就是这种另类的压抑。
<br/>　　只要压抑到了极致，总会有一些彻底爆发的学员，以及快的速度，追赶上以前难以触摸的强者。
<br/>　　...
<br/>　　...
<br/>　　随着慕清霄与小医仙，由东向南走动，果不其然，越是高级的修炼室，就越接近练气塔中央位置。
<br/>　　而在外围，则是分布着中级修炼室，还有更外围的低级修炼室，划分整齐，一目了然。
<br/>　　气息越来越接近，神火已经饥渴难耐。
<br/>　　几分钟后，慕清霄脚步顿住，顺着视线向前看去，两人已经来到了塔内最中央的位置。
<br/>　　“是这里吗？”
<br/>　　望着映入眼中，深不见底的黑洞，慕清霄缓缓向前走了几步，来到黑洞的边缘。
<br/>　　向下望了一眼，漆黑一片，直到视线的尽头，没有一丝的光线，仿佛是一张噬人的巨口，黑暗诡异。
<br/>　　眼前的黑洞，似乎贯穿了整座天焚炼气塔，若非有着塔尖的遮盖，站在上空，便能将这深不见底的黑洞尽收眼底。
<br/>　　站在黑洞的边缘，明显能感觉到，这里的空气要比外面炽热许多，而给天焚炼气塔提供如此热量的，唯有异火！
<br/>　　小医仙紧紧跟在他的身后，倒也没有感到什么不适，只是小心翼翼的看了一眼黑洞，便收回了目光。
<br/>　　“小子，这里可不是你们新生该来的地方，赶快离开这里。”
<br/>　　就在慕清霄思绪万千之际，身后却响起一道苍老的声音，将他拉回现实，声音有些熟悉，正是先前在塔外听到的声音。
<br/>　　转过身来，一位老者正负手站在两人身后，老者白发苍苍，浑浊的双眼，看向中央处的黑洞时，闪过一丝忌惮。
<br/>　　慕清霄微微点头，拉着小医仙离开黑洞，来到老者面前，道：“请问您是？”
<br/>　　“我是天焚炼气塔，第一层的守塔长老，你们可以叫老夫杨长老，看你们的样子，是今年的新生吧？”
<br/>　　杨长老捋了捋白花花的胡子，语气平淡的说道。
<br/>　　原来是守塔的长老，难怪有四星斗皇的实力，看来越是往下，守塔的长老实力也越是强大，真不愧是迦南学院。
<br/>　　“慕清霄，见过杨长老。”
<br/>　　“嗯？你就是新生中，大名鼎鼎的慕清霄？”
