　　第13章
<br/>　　黄三元一直很紧张，因为只有他知道，自己在蛤蟆寨见到的山贼不过两人。看韩玄之这劲头，出动一千余人，一旦被他发现这件事，等待自己的只能是死路一条。所以黄三元一直在想，怎么在路上找个机会逃跑。
<br/>　　可当队伍开始上路，黄三元渐渐觉得，也许自己一直考虑的问题，其实算不上问题。因为队伍中有更大的问题。
<br/>　　道路都不平坦，崎岖坎坷，草木丛生，人们走在这种路上，简直就是一种折磨。一千人的队伍中，有人感冒，有人拉肚子，重甲兵晕了好几个，监军只能骑马在队伍中来回折腾，用鞭子抽打着士兵前进。
<br/>　　偏偏在这样的队伍中，还有一辆豪华马车，两匹马拉车，车厢内铺着厚厚的锦被。韩玄之就坐在车厢内，轻摇羽扇，神色淡然安定。
<br/>　　黄三元憋了好久，终于把肚子里的问题问出来：“少城主，咱们不是要急行军么，可为何您还坐在马车上？”
<br/>　　“我不是士兵，不负责冲锋陷阵。”韩玄之眉毛一挑，英气逼人：“我的责任是指挥他们打仗，吃苦和受罪并不能发挥出我的任何价值，这个道理你懂不懂？”
<br/>　　“我懂，我懂。”黄三元慌忙点头，他又有那种『你说得好有道理，我竟无言以对，可总觉得那里不对』的感觉。
<br/>　　“你不会懂的。”韩玄之目光看向车窗外，感慨道：“古有名将陈玄之，行军时足不出营帐，却杀敌与千里之外，谈笑间敌军灰飞烟灭，前辈风姿，令人敬仰呐。”
<br/>　　合着您是没有名将的命，偏偏还得了名将的病。黄三元心中腹诽，现在他渐渐明白韩玄之是个怎样的人。这世上有酒囊饭桶的二代，也有锐意进取的二代，也有自以为锐意进取，其实酒囊饭桶的二代。而自己眼前这位爷……毫无疑问就属于最后一种。
<br/>　　黑石城，韩玄之带着大军走后一天，薛半川来都城主府。
<br/>　　天气很热，韩虎踞光着上身喝酒，即便如此还热得浑身冒汗。花儿草儿这一对姐妹花伺候着他，姐妹身上只穿着纱衣，透过薄纱能看到少女曼妙玲珑的身体。
<br/>　　薛半川进入城主府，就看到这一幕，他慌张低下头去，不敢直视。
<br/>　　“什么事？”韩虎踞十分不满。
<br/>　　“城主，公子已经走了一天，我心里种觉得不安，不如我领一百骑兵去看一看，以防不测。”
<br/>　　“能有什么不测……好啦，去吧，去吧……”韩虎踞不耐烦的挥挥手。
<br/>　　薛半川倒退着离开，直到走出房间，他才站直身子。
<br/>　　半个时辰后，一百轻骑在点兵场汇集。薛半川曾经练兵五百，这一百轻骑便是其中精锐。都是二三十岁的年纪，人人身强体壮。他们排成五人方阵，身后弓箭，手中长枪，腰上还佩着短刀。
<br/>　　没有人说话，便是战马都保持着安静，一百精壮儿郎用沉默的目光注视着薛半川。
<br/>　　他们不需要大脑，薛半川就是他们的大脑，他们不需要思想，薛半川就是他们的思想。
<br/>　　薛半川顿了顿，翻身上马。
<br/>　　“走！”
<br/>　　唏律律骏马长嘶，就连马蹄落地声几乎都是一致，奔腾如虎，跟在薛半川身后。
<br/>　　以百人心为一人心，以百人手脚为一人手脚，这才是真正的精兵。
<br/>　　……
<br/>　　队伍慢慢行在路上，是在一个黎明接近蛤蟆岭的，一百五十里路程，队伍整整走了一天一夜。
<br/>　　“少城主，咱们马上要到蛤蟆岭，不如让士兵歇一歇，他们已经赶了一天一夜的路。”黄三元道。
<br/>　　“歇！”韩玄之难以理解的看着黄三元：“黄管家，兵贵神速的道理你不懂么，我们歇息，晚上贼人来偷营怎么办？”
<br/>　　“这……”黄三元道：“可士兵们都太累了，他们撑不住啊……”
<br/>　　“我的兵，就要有逆难而上的勇气，绝境求生的意志，黄管家，咱们是在打仗，是要死人的，这小小的疲惫又算的了什么呢。”
<br/>　　妈的，那种感觉又来了，你说得好有道理，我竟无言以对，可总觉得那里不对。
<br/>　　哒哒马蹄声响起，有监军来到马车外：“公子，前面出现一个岔路，一条是官道，一条是羊肠小路，途中还有一段峡谷，请问少城主，咱们走哪一条路？”
<br/>　　“混账，当然是走官道！这还用问。”黄三元破口大骂，心中暗暗祈祷：我的亲爷爷，千万不要再出幺蛾子。
<br/>　　“慢！”果然，韩玄之又是那种『你们都不明白，就我自己明白』的表情，他道：“黄管家都觉得我们该走官道，所有人都以为我们走官道，山贼难道不知道我们走官道？他们定会在路上设下埋伏，可我偏偏出其不意，让他们的如意算盘落空。”
<br/>　　黄三元一口老血差点喷出来，出其不意你麻痹啊，山贼就两个人伏击什么。罢了罢了，反正山贼只有两个人，走那条路都一样，只是士兵多折腾一下罢了。
<br/>　　队伍开始转向，向那条羊肠小路进发。
<br/>　　黄三元心中郁闷自不必提，而他也并不知道，在周围的草林中，有一双双眼睛正注视着队伍。
<br/>　　徐神机在这几天走遍青牛山所有山寨，说的都是同一件事：黑石城要发大兵来攻打青牛山一地的土匪。
<br/>　　对于他的话，没有谁会相信，但心里总会怀疑，总会派人来看看。当韩玄之的队伍来到青牛山附近，已经被人看到，飞速回报给各个山寨。
<br/>　　一千人的队伍呐，所有山寨都如临大敌。
<br/>　　飞虎寨，杏花岭，狗头岭，野猪林，秃头坡……各个山寨的头目都藏在官道两侧的山林中，人人如刺在背。
<br/>　　其中有一人身穿黑色皮甲，手中一柄大刀烁烁放光，他正是飞虎寨的大当家高飞虎，身侧还有一人，是他的弟弟高飞豹。
<br/>　　“大哥，他们好像不是冲我们来的，这条路是冲蛤蟆岭去的？”
<br/>　　“蛤蟆岭不过四五人，还值得这么多兵丁来打？”高飞虎皱紧眉头，也想不明白其中道理：“哎，若是军师在此就好了，先看看吧，今天究竟是什么事。”
<br/>(本章完)
