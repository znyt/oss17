　　最令小兰芳心荡漾的是，慕清霄身为有史以来最年轻的凝丹长老，却与其他的长老不同，为人随和，不摆一丝的架子。
<br/>　　虽然只相处了一周的时间，但慕清霄未曾要求过她做任何事，也并没有觊觎过她的美色。
<br/>　　每天除了准备早中晚三餐，其余的时间都非常的闲，但短短一周获得的好处，却比以往半年都多。
<br/>　　如此年纪就身居高位，性格极好，想不动心都难，更何况长得如此俊逸，总比服侍某些老东西来的好。
<br/>　　小兰的一些小心思，慕清霄多少能猜到，但姿色有限，他真没兴趣，只能选择无视。
<br/>　　午膳过后，慕清霄就离开宫殿，朝偏向北边的妙法堂走去。
<br/>　　五个堂口分部在中央宫殿四周，形成阴阳五行之势，将掌门居住的主殿包围在中央，任何一处，都是天墉城重地。
<br/>　　行走在天墉城的路上，慕清霄一路走来，自然吸引了许多的视线。
<br/>　　不时有弟子从他身边走过，都恭敬的朝他行礼，眼中满是尊敬与崇拜，甚至不乏漂亮的女弟子眉目传情。
<br/>　　几分钟后，慕清霄就来到北边一座高楼前，仔细打量了一番，高楼的形状有些古怪，呈现六边形状。
<br/>　　抬头望去，偌大的匾额上，刻着三个略显沧桑的字体‘妙法堂’，从气息上来看，妙法堂的存在，似乎比凝丹堂历史更悠久，
<br/>　　微微点头，慕清霄便走进其中，妙法堂内空间很大，这跟高楼怪异的形状是分不开的。
<br/>　　中央处摆放着一个木桌，木桌前坐着一名青年，穿着弟子的服饰，认真的登记者妙法堂中来往的弟子信息。
<br/>　　除却中央，周围偌大的空间中都很空荡，唯有靠近墙角的地方，摆放着一排排干净整洁的书架。
<br/>　　书架上规则的放满了各类功法，而且区域都划分的很清楚。
<br/>　　望着周围琳琅满目的书架，慕清霄微微一笑，天墉城毕竟是天下第一大派，若是这点底蕴都没，传出去岂不是让人笑话。
<br/>　　他的来到，自然是引起了周围弟子的注意。
<br/>　　并不在意这些弟子的目光，慕清霄迈步来到中央的登记处，登记信息的弟子就连忙站起来行礼。
<br/>　　“长老，有什么能帮您的吗？”
<br/>　　“先给我说一说妙法堂的信息与注意事项吧。”
<br/>　　说着，慕清霄在周围扫视一圈，周围的修炼功法，似乎都是初级功法，妙法堂掌管着整个天墉城的功法，不可能只有这点。
<br/>　　闻言，登记信息弟子道：“是，长老......”
<br/>　　随后，这名登记信息的弟子就用十分钟的时间，简洁的为他介绍了一下妙法堂的信息。
<br/>　　妙法堂，其中共有四层，第一层摆放的都是一些初级功法，其中包含天墉城功法，也包含其他门派的功法。
<br/>　　第二层摆放的自然就是中级功法，能上去的都是一些精英弟子。
<br/>　　至于第三层，摆放的都是高级功法，非真传弟子，长老，或则得到高层允许的弟子，不得擅自入内，否则重罚。
<br/>　　至于第四层，只有掌门与五堂长老能够进入。
<br/>　　大致了解了妙法堂的信息后，慕清霄满意的点了点头，取出一颗培元丹递给了这名弟子。
<br/>　　子接过培元丹，受宠若惊，在周围羡慕嫉妒的目光中，不断的行礼。
<br/>　　慕清霄摆了摆手，语气平淡，道：“行了，登记一下信息，我需要阅览一下修炼功法。”
<br/>　　“是，长老您随意，弟子马上帮您登记。”
<br/>　　“嗯。”
<br/>　　弟子登记信息的时候，慕清霄已经来到初级功法区域，他作为第十一代凝丹堂长老，妙法堂就没有他去不得的地方。
<br/>　　至于登记信息，不过是走过形式罢了。
<br/>　　初级功法，都是给刚入门的弟子修炼，以便给他们打下扎实的基础，一般长老都不会来此，而是直奔更高层。
<br/>　　于是，慕清霄就在周围弟子诧异的目光中，在书架前走动起来，甚至每一本功法都会触碰一下。
<br/>　　半刻钟后，将整个初级区域走完，慕清霄脸上露出遗憾之色，正当他准备前往第二层的时候，脑海中就响起系统的提示声。
<br/>　　“叮，《三才剑》收录中，奖励500掠夺积分。”
<br/>　　微微一愣，慕清霄一喜，随后就将指尖所触碰到的功法从书架上抽出，《三才剑》天墉城初级剑法之一。
<br/>　　此剑法为天墉城后辈弟子入门必习剑术，凝气挥剑，产生三道剑影交错刺穿对手，此技威力较小，适合初入门派的弟子修习。
<br/>　　果然，系统收录功法依旧有效，虽然只有500点掠过积分，但蚊子再小也是肉啊。
<br/>　　“叮，《沧海剑》收录中，奖励500点掠夺积分。”
<br/>　　“叮，《叠浪剑》收录中，奖励500点掠夺积分。”
<br/>　　“叮，《法剑降魔》收录中，奖励500点掠夺积分......”
<br/>　　当慕清霄逛完整个初级功法区域，总共收获了四千点掠夺积分，积少成多，四千点掠夺积分已经算不错了。
<br/>　　对于这些初级剑法，慕清霄并不感兴趣，根本不需要修炼，只要扫一眼，直接就能施展出来。
<br/>　　离开初级功法区域后，慕清霄脸上挂着一丝笑意，向第二层走去。
<br/>　　至于他脸上那喜悦的模样，自然是令得周围的弟子二丈摸不着头脑，面面相觑，你一个凝丹长老，学习初级剑法有啥用？
<br/>　　当然，他们也只好在心里想想，自然是不敢说出来的，哪怕是普通长老，都容不得他们指手画脚，更何况凝丹长老。
<br/>　　来到第二层，其中的弟子只有寥寥数位，相比第一层也是清静的多，中央位置处，依旧有一名登记信息的弟子。
<br/>　　第一层负责登记信息的弟子是练气修为，而第二层则是筑基修为，但见到慕清霄后，依旧是毕恭毕敬。
<br/>　　别说是这些弟子，整个天墉城，除了掌门和五堂长老外，任何人见着他都必须恭恭敬敬的行礼。
<br/>　　等弟子登记完信息后，慕清霄就开始在中级功法区域探索起来。
<br/>　　第二层的功法，相比第一层明显要少的多，只有数个书架，上面也并非摆满功法，甚至有许多地方都空着。
