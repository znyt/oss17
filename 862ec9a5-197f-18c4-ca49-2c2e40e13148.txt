　　第554章
<br/>　　夜黑风高，但程大雷仍一眼就发现了对方。
<br/>　　对方的动作很快，迅捷却不失灵敏，轻飘飘的落地，不发出一丁点声音，往前一跃，却又是数丈距离，感觉像一只生着肉垫的猫。
<br/>　　程大雷阅读对方的信息。
<br/>　　姓名：燕不归（来自草原的绝世高手）
<br/>　　技能：重剑术
<br/>　　隐藏属性：无
<br/>　　如此，对方在程大雷眼中已没有秘密。既然来自草原，很大概率就是戎族，也暗合了先前的猜测。
<br/>　　他就是盗书贼，盗书贼就是他。
<br/>　　不过程大雷还是有一件事想不通，既然对方已盗走山水经，那么今夜探山又是为了何事。难道是偷出兴趣来了，觉得偷一次不太过瘾。
<br/>　　当然，这对程大雷来说无关紧要。程大雷只要知道，跟着对方能夺得山水经也就够了。
<br/>　　他没有打扰对方，也没有故意向书院警报对方的到来。一个纵身跃出藏书楼，选择一高地藏身，第一是搞懂对方来干嘛，第二就是在对方离开时跟好对方，查清楚对方的落脚点。到时候能偷就偷，不能偷就抢，一定要将山水经搞到手。
<br/>　　燕不归现在的心情很急躁，所谓关心则乱，因为事关黑贞的生死，如他这般镇定的人，行事也莽撞起来。
<br/>　　潜入万卷山，他才意识到这一点。书院是大的，找一座藏书楼容易，可自己怎知道华家人住在哪里。
<br/>　　自己就连对方是男是女，是老是少都不知道。
<br/>　　根据过往的经验判断，外来客都住在后山厢房。但因为这次伏魔大会的事情，江湖豪杰来了不少，后山就是高手窝子。
<br/>　　就算燕不归再豪横，也没自信能以一人之力对抗整个东海的高手。
<br/>　　一来是艺高人胆大，二来是这件事关乎黑贞的生死，于是燕不归决定铤而走险，潜入后山。
<br/>　　他的动作极其灵巧，如夜蝙蝠轻飘飘潜入后山，趴在一间屋子的房顶上。
<br/>　　周围有许多房子，有的亮着灯，有的已经暗下来，并看不出来那间屋子里住着自己想要找的人。
<br/>　　正这时候，两个小童从屋檐下走过，边走边聊天。
<br/>　　“那位就是华神医么，没想到生得那般模样。”
<br/>　　“嗳，人不可貌相，连夫子都敬重他，一身本事肯定非同小可。夫子已经吩咐了，让我们好生伺候，万不可有半点懈怠。”
<br/>　　趴在房顶上的燕不归听到这番话，眼睛顿时一亮。这当真是踏破铁鞋无觅处，得来全不费工夫。他顺着童子来的方向望过去，见一栋屋子的门刚刚关上。
<br/>　　燕不归立刻展开行动。
<br/>　　华正则在房中打了个哈欠，将手中的书合上。
<br/>　　华家不是一个人，而是一整个家族。以医术传世，为避战乱来到东海。多年传承下来，也算一个颇大的家族，而华正则算这一代的代表人物。
<br/>　　这次书院搞什么伏魔大会，华正则被家里的老人派了过来。其实伏魔大会什么的，华正则并不关心，医者只负责救人，杀人的事情和他们有什么关系。
<br/>　　这次与他而言，算是出趟公差，等应付完了，他还要回到村子里去。
<br/>　　一阵又一阵的涛声吵得他不得安宁，他索性也不读书了，吹灭灯准备睡觉。
<br/>　　就是这一瞬，只感觉身后寒风袭来，华正则刚回过头，就有一柄剑压在他脖子上。
<br/>　　身后的窗户洞开，面前站着一个大汉，黑紫脸膛，整个人如一块黑铁。
<br/>　　华正则表现得很镇定，说出了那句经久不衰的台词：“好汉饶命。”
<br/>　　“你是大夫？”
<br/>　　燕不归也很困惑，他没见过所谓的华神医，所以华正则长什么样，都该是理所应当。可关键是，对方长得也太丑了。
<br/>　　当然，燕不归也远远谈不上英俊，但以燕不归的相貌，都觉得对方丑陋，那华正则的模样就实在有些有碍观瞻了。
<br/>　　“算是吧。”华正则。
<br/>　　“你会治病？”
<br/>　　“也会一些。”华正则顿了顿道：“但不包括把自己的脑袋接上。”
<br/>　　“油腔滑调。”燕不归冷哼一声：“我要你去治一个人，你若答应，我到时候会饶你一命。”
<br/>　　“这个……我好像也没有说不的权力。”华正则道：“能否让我带上些东西，治病救人总不能空手去。”
<br/>　　燕不归点点头，稍稍将剑松开：“敢耍花招，我会立时杀了你。”
<br/>　　华正则没有要耍花招的意思，将大概需要用到的东西收进医囊，道：“怎么走，有车么？”
<br/>　　“我带你走。”
<br/>　　说着，燕不归一计手刀敲在华正则脖子上，将他敲晕过去，然后抓起床上的帷步，将他绑在自己身上。
<br/>　　如此，他跃出窗户，纵身上房，沿着来时路下山。
<br/>　　程大雷一直缀在他后面，此刻找了个高处的山石隐蔽身形。燕不归在房间内做了什么他不知道，但走出房子的一举一动，都没有逃过程大雷的眼睛。
<br/>　　程大雷顺势阅读了华正则的信息，他也没见过华正则，可看到对方的身份信息，就明白对方正是自己千里迢迢来东海要寻的人。
<br/>　　啧啧，你要我的神医，我要你的山水经，看来，咱们还真是对上了。
<br/>　　程大雷心底叹了一声，视线一直没有离开燕不归。现在不是动手的时机，程大雷必须确认山水经的下落，才会出手。
<br/>　　现在的他不仅不会出手，反而祝福燕不归可以平安下山。
<br/>　　而这个时候，不管程大雷还是燕不归，他们都不知道，今夜鬼鬼祟祟，夜闯万卷山的不仅他们二位。
<br/>　　还有一人，轻身功夫比程大雷还要轻盈，速度比燕不归还快。如果程大雷是生着脚垫的猫，燕不归是无声无息的夜蝙蝠，那么这第三人就是一只鬼影。
<br/>　　飘飘忽忽，既不见人，也不见影。
<br/>　　程大雷盯着燕不归，想做捕蝉的螳螂，却不知道身后已立着黄雀。
<br/>　　从黑影眼中照出两道冰冷，像是要冻住一切。但是他没有对程大雷下手，而是转身去了另外一个方向。
<br/>　　原来你已经来了，那么很好，琅琊城就将成为你的埋骨地。
<br/>　　鬼影飘忽不见，无声无息。
<br/>(本章完)
