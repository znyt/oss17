　　“不是我们的人。”几人中一人也拉下面罩，喘着气开口。
<br/>　　领头的一愣：“还有别的队伍前来暗杀？”
<br/>　　“我们的人都没有行动，我来之前还特地查看了一边，也没有人受伤。”一人说道：“一定还有别的队伍。”
<br/>　　“跟我们一样？”领头的人有些惊讶：“我们可是……”
<br/>　　“别的家族也暗中培养了死士，这些并不是秘密。”一人思索道：“只是哪一家族的人前来，我们却不能肯定。”
<br/>　　“看那帮人的行·事作风一点不讲究，想来也不是我们的老对手高家。”领头的人冷笑：“怕是罗家那些人，他们刚从外面回来，没经历过多少次家族之间的大战，上来就跟高家来了场血统成员的较量，死士之间的斗争恐怕丝毫没有了解过。”
<br/>　　“那现在怎么办？”一人问道。
<br/>　　“既然有炮灰，我们何尝不坐等好处送上门？”领头的人笑着，套上了面罩。
<br/>　　此时在已经变成一片空地的木屋周围，狼牙的队伍相互间靠的很近，所有人都安静无声，几名嗅觉和听觉敏锐的人不断感受着四周。
<br/>　　“四点钟方向！”忽的一人大喊，紧接着——
<br/>　　刷刷刷！
<br/>　　无数道攻击猛然朝着那一方向攻去，漫天都是密密麻麻的武器交织。
<br/>　　嘭！
<br/>　　一声重响，一人被打出现形，口吐鲜血的匍匐在地，身上那件诡异的外罩就这么慢慢的褪去颜色，变成了纯黑。
<br/>　　狼牙军中气势大振，但同时一声惨叫也从队伍后方传来。
<br/>　　噗！
<br/>　　一道血花飞溅，一名防守队成员的胸口一柄刀剑破出体外，而后便是猛然的倒抽而去，留下一股喷涌的鲜血，和直愣愣倒地的身影。
<br/>　　其余人只能这么看着这名同伴死亡，而那名暗杀者也再一次隐身，消失的干干净净。
<br/>　　当狼牙找到了解决办法时，战斗也就这么发展成了拉锯战，有时能发现对方一两个暗杀者的踪迹将其秒杀，有时却是一场空，而同时狼牙队伍中的人也在不断减少，总会被暗杀者趁机而入。
<br/>　　一来一回的战斗持续了很久，双方似乎就这么杠上了，暗杀者不断想要攻破防线靠近禁地，狼牙就死守不退。
<br/>　　——————
<br/>　　当狼牙基地爆发了暗杀的同时一时间，安罗城的城门处正有一支队伍排队入城，自从南方的战事爆发以来，期间就不断有人往安罗城投奔，哪怕此时安罗城的居住地价格已经高的离谱，也依然挡不住人们想要住进这座梦幻城市的热情。
<br/>　　这时候的队伍已经排到中段，城门处的人依照流程检查这些人的身份，以及简单的问话。
<br/>　　“兄弟俩？”一名守卫惊讶出声，目光在眼前两名身材和相貌都相差巨大的人身上打转。
<br/>　　“是啊。”排在前面的人笑了笑，露出一口洁白的牙齿，皮肤白·皙的不像是末世纪元的幸存者。
<br/>　　而他身后的男子则人高马大，在队伍中尤其扎眼，长相更是带着一股凶煞之气，皮肤有种说不出的古怪颜色，总之怎么看怎么与前面的这名男子不像有血缘关系。
<br/>　　守城门的人还想继续问，前方的那人却先一步开口：“结拜的嘛！同姓真是巧合，我跟我弟有过命的交情。”
<br/>　　问话的人顿了顿，此时另一名城门防守队的人小跑着前来，在此人耳边小声道：“查过了，两人的相貌都不在情报部有记录，应该是普通幸存者。”
<br/>　　守门人点点头，转身对着两兄弟摆了摆手：“进去吧，记得……”
<br/>　　“交钱嘛，我懂的！”走在前方的年轻男子又一次一笑，随手拿出一沓大面额的回炉币。
<br/>　　队伍中顿时响起一阵倒吸凉气的声音，几名防守队成员也愣了愣，忍不住嘴角一抽不再关注这两人，显然把他们当成了那个大人物的亲友类人物，年轻不出名却有钱的，在这末世纪元里也就只有这样的公子哥。
<br/>　　对于这样的人，狼牙向来来者不拒，坑起他们的钱来也一点不手软。
<br/>　　成功进入城内后，年轻的男子心情很好，面上一直带着笑容，一路走走逛逛对什么都很好奇。
<br/>　　跟在他身后的凶煞大汉此时看起来，到更像是保镖，路人这么认为，狼牙暗中观察的人也同样这么认为。
<br/>　　两人一路走着，直到最后一名暗中观察的防守队成员撤离消失在转角。
<br/>　　这时的年轻男子终于停下了脚步，看着路边的一家店铺，像是自言自语的开口：“你为什么没有杀楚涵，是恢复的记忆让你摇摆不定？”
<br/>　　站在他身后的凶煞大汉面无表情的回答，声音带着一股浑厚气息：“我已经与他的身份和立场完全相反，这几年的杀戮也让我根本不再是当年那个木天，我没那么多的想法特地去保他命。”
<br/>　　说话的凶煞大汉，正是消失已久的木天，同时也是以类尸身份登上丧尸首领的丧尸王！
<br/>　　此时他长相与人类无异，一切都像个正常人类，甚至混在人类的城市里，除了身材像是个战力超绝的存在外，其他地方一点都不突兀。
<br/>　　而一直走在他前方的人，正是神秘莫测的异种王，木叶！
<br/>　　听到木天的话，木叶笑容一瞬间的诡异，回身道：“这么说来，是你没能杀掉他？”
<br/>　　木天的眉头皱起，整张脸都看起来很凶恶：“他身上有其他东西，我破不进去，给我一种哪怕杀死了他的肉体，也杀不死他灵魂的感觉，所以我及时收手，改成了让他沉睡。”
<br/>　　“那么现在你着急的想要过来又是为何？”木叶再次问道，同时腥红的舌头不经意的舔·了舔嘴唇，就像是一只贪婪的动物，身处满是食物的中心。
<br/>　　木天望向远方，眼底有一丝凶狠一闪而过：“我设下的屏障在变薄，有人在试图打破我对楚涵的精神限制。”
<br/>　　此番话让木叶想要吃光这一整座城市之人的心思拉回，神态也认真了起来：“这世上竟然还存在与你一样等级的精神类强化者？”
<br/>　　“不，那个人比我弱很多。”木天摇头中语气肯定，紧接着又有些不解：“但为了救出楚涵，似乎不怕死。”
