        &lt;h3&gt;第三卷第五十二章  闫欢的决定&lt;/h3&gt;
<br/>
<br/>        闫欢来不及告诉陈子昂甸城的暗黑点，陈子昂已经踏上了回北京的行程。
<br/>
<br/>        当闫欢给陈子昂打电话的时候，陈子昂已经在飞机上。
<br/>
<br/>        闫欢看着手机深深的叹了口气，陈子昂是见过谷强的，谷强还活着，而谷强的身份她是知道，甸城是非常凶险之地，谷强依然踏足，为什么他还要将陈子昂也拉过去呢？
<br/>
<br/>        陈子昂这么多年一直不结婚是不是对张函还抱有希望呢？
<br/>
<br/>        她当初被陆玉送到甸城的时候，如果没有甸城的人接应，她是不会相信的，那么陆玉是不是和那边的人有往来呢？
<br/>
<br/>        闫欢又想起了谷强，那个帅气、温文有礼却又冷冰冰的男人，他生活在黑暗的世界，却有一颗善良的心，在那么多黑暗的日子里，是谷强带给了她温暖和希望。
<br/>
<br/>        他如今还好吗?他什么时候能完成他的使命回归正常的生活呢？她已经回来好几年了，他还在那个地方，他的大生意做成了吗？也许是做成了吧？
<br/>
<br/>        色拉朵说，如果做成了，他就会活着回来，做不成他就回不来了，那么他一定是做成了。
<br/>
<br/>        可是，他将如何面对他死去的战友呢？这几年他还保持着他的初心吗？
<br/>
<br/>        闫欢有泪滑落。
<br/>
<br/>        她已经脱离了苦海，可是谷强还在苦苦挣扎，只有深入其境的人才了解挣扎的苦痛。
<br/>
<br/>        她们的超市已经在西省遍地开花了，经济的浪潮催生了很多产业的发展，她们算是抓住了机遇，而且她们做的都是于民生息息相关的事情，可以说，目前是高枕无忧的，只等着收钱就好。
<br/>
<br/>        她和陈子悦商量了一下，她们要抓住机会，在别的地方拓展业务。最近她准备去别的地方考察一下，陈子悦这个女人心大着呢。
<br/>
<br/>        想到陈子昂要去甸城，想到谷强，闫欢心里一动，她可以跟着陈子昂一起去甸城。
<br/>
<br/>        她对甸城充满了恐怖，那个地方是她一生的噩梦，但是那里还有一个人，一个她心心相念的一个人，虽然她知道她和他不可能。
<br/>
<br/>        那里还有她的妹妹，她的妹妹可是秦少卿之妻，秦家的儿媳妇。
<br/>
<br/>        她去甸城，有了秦少卿这层关系，做事情也方便很多吧？
<br/>
<br/>        说干就干，她对陈子悦说：“子悦，子昂一个人去甸城让人有点不放心呢，不如我跟着一起去？”
<br/>
<br/>        “你去干什么啊？家里的事情你不管了吗？”陈子悦有点反对，子昂是去做生意，她闫欢去做什么？
<br/>
<br/>        “你听我给你说，子昂对甸城不太熟悉，而我的妹妹在甸城，而且是当地二把手老总的老婆，我过去也将我们的超市开过去，那边地区荒远，过去肯定也有市场的。子昂做她的高大上，我做我们的接地气。这样我们俩个人都有个照应，你说呢？”
<br/>
<br/>        “你说你怎么会突然想要和子昂一起出去？”
<br/>
<br/>        “你都说了，你有现在这样的境地，都是子昂的功劳，而我有现在的这种境地也是你的功劳，吃水不忘挖井人，子昂又是我们最大的股东，没有子昂也没有我们的现在，我陪着她一起过去，有我妹妹的那层关系，好歹她也能轻松点不是？”
<br/>
<br/>        陈子悦听闫欢这样一说，心里豁然开朗：
<br/>
<br/>        “那你过去吧，有你陪着子昂也好，只是我怕我一个人应付不过来这边的事情。”
<br/>
<br/>        “哎呀，我的好子悦，这个世界上还有你应付不来的事情？再说，都同在一个国家，同一片天空，现在都有手机和电脑了，有啥事一个电话就解决了，没事的。等我将那边的地盘拿下来了，请我们的大老总过去指导。”
<br/>
<br/>        “那你要不要给子昂打电话说一声？”
<br/>
<br/>        “也好，我早上给她打电话关机，估计是在飞机上，你晚上再给她打电话说一声。你就说是你决定的啊，不要说是我。”
<br/>
<br/>        “为什么？”
<br/>
<br/>        “你是她姐姐啊，你说了她肯定会同意的。”
<br/>
<br/>        “就你事情多。心眼多。”
<br/>
<br/>        “陈子悦！”
<br/>
<br/>        “心眼多，怎么了？”
<br/>
<br/>        “好了，我心眼多，你是奶。对了，子悦，我走之后你一定要照顾好自己，另外，虽然现在我们手头上都有钱了，但是，你还是注意一点王老师，不要给他们家很多钱，够吃够用就行。”
<br/>
<br/>        “好的。”陈子悦并没有问闫欢为什么会这样说，她知道闫欢已经知道有些事情了。
<br/>
<br/>        “你呀，就是心肠太软，这个世界上光凭心肠软是解决不了问题的，有心人会利用你的心软得寸进尺的。”
<br/>
<br/>        “好的，我知道啦，怎么就没有发现你还有做老娘的潜质呢？”
<br/>
<br/>        “我只做你的老娘！我说的是真的，你一定要记住！”闫欢正色的对陈子悦说。
<br/>
<br/>        “嗯，我知道了，闫欢，你说你要出门，我咋就觉得心里这么难过呢。”陈子悦突然就抽泣起来了。
<br/>
<br/>        闫欢看着流泪的陈子悦，她了解陈子悦的过去，她们是一路人，这几年在苦苦的奋斗着，只是为了成为更好的自己，也是为了自己在乎的人。
<br/>
<br/>        她们努力的成长，努力的想要和过去做告别，努力的好好活着，期间的挣扎和辛苦只有她们自己清楚。
<br/>
<br/>        她和陈子悦是在这个世界里相互取暖的俩个人。
<br/>
<br/>        她没有太多的情感纠葛，高中的时期只是年少的懵懂，青春的必然，却葬送了她的美好前程，让她尝尽心酸，受尽折磨。
<br/>
<br/>        遇到谷强，那是可望不可及的一种存在，谷强虽然身处泥潭，他确实带着一种光，照亮了闫欢黑暗的世界，让闫欢看到了希望，他给了闫欢尊重，给了闫欢做人的自尊。
<br/>
<br/>        他看似冰冷、桀骜不驯，看似霸道让人害怕，其实他的内心是柔软的，善良的。
<br/>
<br/>        闫欢叹了一口气，无论如何她要去甸城看看。
<br/>
<br/>        “子悦，想哭就大声哭出来吧，在我这里哭，好过在别人面前哭。哭过之后还是要满面笑容的生活，因为我们有自己在乎的人。”
<br/>
<br/>        陈子悦不敢放声大哭，她怕吵醒孩子了，只能抽抽噎噎的小声的哭着。
<br/>
<br/>        这么多年，她一直都是笑着的，和闫欢一起奋斗，一起骂账，一起笑，一起闹，突然闫欢要离开她，她的心中就空了，和王浩楠之间的那道裂痕似乎一下子就放大了很多倍。
<br/>
<br/>        闫欢一直看着陈子悦流泪，她静静的等着，等着她哭个够。
<br/>
<br/>        陈子悦哭了一会，感觉好多了。
<br/>
<br/>        “好了？”
<br/>
<br/>        “好了！”
<br/>
<br/>        “好了，明天就该各司其职了，我走了所有的事情你都要亲自打理，不必要事事都过问，但是一定要管理好几个店长。”
<br/>
<br/>        “嗯”
<br/>
<br/>        “我会在这一段时间内将所有的事情都理顺之后交给你。”
<br/>
<br/>        “好。”陈子悦还没有从忧伤里完全出来。
<br/>
<br/>        “那就这样了，你抽时间给子昂打电话。”
<br/>
<br/>        闫欢心意已决，她准备重返甸城，当初是别人送过去的，现在是她自己过去的。
<br/>
<br/>        当初，她有多狼狈，现在她就有多风光，她将会是以一个投资人的身份去甸城！。
