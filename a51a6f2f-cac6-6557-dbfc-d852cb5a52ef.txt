
<br/>　　<br/>　　    宝贝还没见到，疯的疯死的死，十来个人的队伍就剩下了三井小叶，小林昙一和三井美奈，怎么看都有一种崩溃到了极致的感觉，令人难以接受，这一趟寻宝之旅肯定是彻底夭折了，因为面对那漆黑，诡异的两扇石门里的宫殿，他们谁也不敢再上前一步了。<br/>　　<br/>　　    人若崩溃，多少都是会有些丧失心智的，所以当王惊蛰他们来到宫殿前的时候，这三人都没有看见也没有听到一点动静，心神仍旧处于失常下，六个人错落有致的站在宫殿门前，陈三岁茫然不解，王惊蛰眉头紧皱，孔良则是一脸诧异，然后说了一句话。<br/>　　<br/>　　    “是苯教的护法尊者像，千手忿怒佛母，辛拉哦噶还有南巴嘉瓦……这，不是宫殿，是神庙？是苯教的神庙？”<br/>　　<br/>　　    陈三岁张了张嘴，无语的说道：“好高深，感觉好像很难听懂的意思”<br/>　　<br/>　　    孔良很严肃的看了他一眼，然后双手合十朝着神庙的方向施礼，等他再抬起脑袋的时候，就轻声解释道：“在藏区有个规矩，就是喇嘛庙很多，多到能有1700多座，但是神庙却不允许随便修建，到如今也不过才寥寥几座而已，是因为想要建神庙的话，得需要有佛陀降临才行，上师不行，法王不行，就连转世的活佛可能也不够，得是真正的法脉传承者才行”<br/>　　<br/>　　    陈三岁问道：“什么是法脉传承？”<br/>　　<br/>　　    孔良愣了愣，摇头说道：“我也不太清楚，可能大部分人都不太清楚，这是神庙的隐秘，特别是苯教的法脉传承就更没人知道了，他们很神秘，你所看到的绝大部分喇嘛都可能不是苯教的”<br/>　　<br/>　　    他们在这闲聊的时候，终于引起了三井美奈那边的注意，初时三人有些奇怪和惊异，接着就回过神来了，他们能来那就说明之前去杀人的吉野肯定死了。<br/>　　<br/>　　    “你们？是你们捣的鬼是不是？”三井小叶这时精神状态已经不正常了，他颤巍巍的抬起胳膊，哆嗦着手里的枪，咬牙吼道：“是不是你们在捣鬼，说啊，那个宫殿里有什么，良河君为什么会疯，纳粹的珍宝到底在没在里面，说啊”<br/>　　<br/>　　    王惊蛰皱眉看着这小鬼子，眼神里很冷的闪过一道精光。<br/>　　<br/>　　    “说啊，不说我就开枪了啊……”三井小叶晃了晃枪口，在他们三人身上来回的指了几下，颤抖的手指放在扳机上。<br/>　　<br/>　　    眼看着他似乎马上就要开枪之际，王惊蛰没动，陈三岁的身影晃了晃，就跟划过一道残影似的“嗖”的一下就落在了三井小叶的身前，姿势很平常而普通的伸出胳膊就抓住了他的手腕，然后轻轻向上一抬。<br/>　　<br/>　　    “嘭！”三井小叶扣动了扳机，子弹射向了半空中。<br/>　　<br/>　　    陈三岁鄙夷的说道：“之前与你们一路，只是因为我们懒得管你们，有自己的事要忙，可你们还真不知好歹，居然起了杀人灭口的心思，真是让人好笑，也不想想自己的脚下踩着哪一方的土地呢，能容得了你在此放肆？”<br/>　　<br/>　　    三井小叶的手腕上就跟上了一道紧箍咒似的，被陈三岁都给抓红了，他冷汗直冒的呵呵笑着说道：“我们又不是没放肆过，几十年前不是一样在这里烧杀抢掠么……”<br/>　　<br/>　　    陈三岁眼神当即一冷，手下忽的就加大了力道，你很难想象的是一个人的手腕，竟然如此轻易的被另一个人用手给捏折了。<br/>　　<br/>　　    王惊蛰淡淡的说道：“天下大事每合久必分，中华大地也是如此，不经历一场生灵涂炭，哪里来的浴火重生？你们那个弹丸小国真当自己有几把刷子呢？当年，要不是我们都看清了走向，这片土地你们来了，就没有回去的可能了，你真以为老虎打盹的时候不咬人啊，天真”<br/>　　<br/>　　    “嘎巴”陈三岁一把扭断三井小叶的胳膊后，他手里的枪就掉落下来，被陈三岁左手接住，只见他也没怎么动动手指，一把枪就被拆卸成了一堆的零件然后随意的扔在了地上。<br/>　　<br/>　　    “啪”陈三岁再伸手掐上三井小叶的脖子，说道：“你们这些人来，真不够看的，要是传说中那些什么上忍一类的狗头狗脑的玩意儿来还能试试，你啊连炮灰都是算不上”<br/>　　<br/>　　    陈三岁说完三井小叶的身子就软软的倒在了地上，他甩了甩手腕就要朝着小林昙一走去，他顿时惊慌的连连后退，反应极快的说道：“这个时候我们应该互相抛开成见才是，这个宫殿太神秘了，我们要合作啊，不然，不然谁都不可能活着出去了”<br/>　　<br/>　　    陈三岁讶然的说道：“它在神秘，你不进去不就得了，非得一脚踩进火盆里试试烫不烫脚，你脑子里都是浆糊么？”<br/>　　<br/>　　    小林昙一：“……”<br/>　　<br/>　　    三井美奈不解的问道：“你们不是为了纳粹的宝藏而来的么？”<br/>　　<br/>　　    陈三岁撇嘴说道：“钱财如粪土，都是身外物，谁会在乎那些玩意啊？”<br/>　　<br/>　　    三井美奈不解的问道：“那是为了什么？”<br/>　　<br/>　　    “大道朝天，各走一边，人活一世有的为财，而有的人就麻烦了，比如为生，想要活下去啊”王惊蛰戚戚然的叹了口气，要不是为了龙骨草他何至于来这神秘兮兮的天坑底下。<br/>　　<br/>　　    因为，到这时王惊蛰已然发觉这里太神秘了，面前的那个小鬼子会死，至于这个女人就不太好说了，他没算出来，那剩下的就是陈三岁和孔良了，依旧处于横祸临头的征兆。<br/>　　<br/>　　    陈三岁要杀人，小林昙一惊慌不已，但你不得不佩服这个人的脑袋，简直灵的不要不要的。<br/>　　<br/>　　    “你们不想知道那座宫殿里发生了什么嘛……”小林昙一语气急促的说道：“我可以先为你们进去观望一下”
