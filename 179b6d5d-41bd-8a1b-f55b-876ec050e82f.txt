<br/><br/>　　    jeangeorges不愧是顶尖的法国餐厅，菜品水准绝对一流，特别是鱼子酱，绝对是无与伦，搭配着法国出产的黑桃a香槟，让人都有一种融化了的感觉。<br/>　　<br/>　　    吃了一会儿之后，傅华看了看谈红，笑了笑说：“谈红，谢谢你这美味的晚餐，不过我明天要去洛杉矶了，你是不是可以告诉我你到底决定回不回国帮我打理金牛证券啊？”<br/>　　<br/>　　    谈红笑了笑说：“诶，傅华，你有点情调好不好，对着这么美味的食物，不要谈这些俗事，等我们吃完晚餐之后，我自然会给你一个答复的。”<br/>　　<br/>　　    傅华笑了，说：“好，那我们先吃晚餐。”<br/>　　<br/>　　    谈红笑着举起酒杯，说：“来，傅华，为我们在纽约相会干杯。”<br/>　　<br/>　　    两人边吃边聊了起来，席间谈红显得略微有些兴奋，回忆起她在顶峰证券跟傅华打交道的一些情形，回忆让两人都有些兴奋，然后她接连的端起酒杯敬傅华的酒，到结束的时候，她已经有些微醺了。<br/>　　<br/>　　    从餐馆出来之后，谈红陪着傅华走回了西屋酒店，在酒店的门口，谈红笑着说：“傅华，请我进去坐一下吧，关于回不回国帮你打理顶峰证券，我还有些话需要跟你说说。”<br/>　　<br/>　　    傅华晚倒也没事，带着谈红进了他的房间，开门进去之后，他直接走向沙发的茶几那里，笑着说：“谈红，你今天晚可是喝的有点多啊。你先坐，我倒杯水给你喝。”<br/>　　<br/>　　    “别忙了，傅华，”谈红从后面一把拉住了傅华的胳膊，不让他继续往茶几那里走去。<br/>　　<br/>　　    傅华转头看了一眼面若桃花的谈红，刚说了一句，没事，……<br/>　　<br/>　　    谈红的另一只胳膊揽住了他的脖子，红艳的嘴唇堵住了他的嘴，然后开始有些疯狂的吻了起来。错愕之下，傅华一开始还想要推开谈红，但是谈红的胳膊死死地箍住了他的脖子，不让他有机会挣脱。<br/>　　<br/>　　    傅华看了一眼谈红，谈红神色黯然，泫然欲泣，心不禁泛起了无限怜爱，他知道谈红在当初他们刚相识的时候，对他情愫暗生了，于是他不再去推开谈红，而是积极地去回吻起谈红来……两人喘息着瘫软在了一起，这时，谈红开口说：“傅华，我想告诉你……”<br/>　　<br/>　　    傅华苦笑了一下，说：“好了，谈红，我知道你想说什么的，你想告诉我你要留在纽约是吧？”<br/>　　<br/>　　    谈红娇喘着笑了一下，说：“原来你都看出来了。”<br/>　　<br/>　　    傅华说：“是啊，从我们见面开始我看出来你是有些犹豫的，但那个时候我还无法确定你要拒绝我，但刚才你吻我的时候，我意识到你心基本已经决定要拒绝我了。”<br/>　　<br/>　　    谈红笑了笑说：“是啊，实际我一开始是不太想回国去帮你的，我更习惯于纽约这边的工作环境，这里的规则明确，只要你按照规则去做了，别人不会找你的麻烦。但是在国内，是另外一番情形了，国内的潜规则实在太多了，我根本无力应付的。”<br/>　　<br/>　　    傅华看了谈红一眼，笑了笑说：“既然你心里清楚你是不愿意回国的，你该早一点告诉我的，我的性格你不是不了解，我是不会因为被你拒绝而生气的。”<br/>　　<br/>　　    谈红笑了笑说：“我之所以没有一开始拒绝你，是因为我那时还有一些另外的想法的。首先是我担心你那个金牛证券目前是不是确实需要一个像我这样的人去帮你撑住场面，你曾经几次帮我渡过难关，你现在如果是有困难了，我义不容辞的也要去帮你解决的。”<br/>　　<br/>　　    傅华笑了笑说：“所以你才会问我金牛证券现在有没有什么危机？”<br/>　　<br/>　　    谈红点了点头说：“是啊，可是你告诉我金牛证券现在并没有什么急于解决的危机，这让我觉得没有必要非回国去帮你的。不过那个时候，我还是没有下定决心拒绝你，因为我心对你还存在着一丝幻想，想说我是不是可以利用这一次的机会跟你有更近一些的发展啊？结果我看到了你不接那个女孩子的电话的情形。”<br/>　　<br/>　　    谈红说到这里，抬头看了一眼傅华，笑笑说：“那个女孩子是你现在的女朋友吧？”<br/>　　<br/>　　    傅华摇了摇头说：“还真不是，她其实是一个长辈介绍给我的相亲对象，很年轻，并不适合我的。”<br/>　　<br/>　　    谈红笑了笑说：“不管是不是，都让我认识到了一个事实，那是虽然过去了这么长时间，你好像却是什么都没改变一样，依旧是风流潇洒，依旧是有那么多的女孩子喜欢你的。而如果我回国帮你的话，会成为你的属下，以你的个性，肯定是不会跟我谈什么办公室恋爱的，那我还是没机会得到你。所以我死心了，决定还是留在纽约，不回国帮你了。”<br/>　　<br/>　　    说到这里，谈红亲了傅华的嘴唇一下，笑了笑说：“其实刚才我是有点借酒壮胆的，想说此一别，恐怕这一辈子我们都是难以再见面的，如果再不跟你好好疯狂一次，恐怕这辈子没有机会疯狂了。幸好，你没拒绝我，也算是为我们这段情缘画一个完美的句号。”<br/>　　<br/>　　    谈红这么说让傅华心不禁有些伤感了起来，但是他心也知道他是给不了谈红想要的东西的，虽然刚才那一刻他确实是意乱情迷的，但那是一种短暂的迷失，并不足以让他跟谈红长相厮守的。<br/>　　<br/>　　    傅华抱紧了一下谈红，苦笑着说：“对不起啊谈红……”<br/>　　<br/>　　    谈红笑了，摇了摇头说：“傅华，对不起什么啊，你已经给了我想要的了。我已经很满足了。”<br/>　　<br/>　　    当晚谈红留宿在了傅华的房间，第二天一早，她送傅华了去洛杉矶的飞机，临分手的时候，谈红拥抱了一下傅华，在傅华的嘴唇轻轻地吻了一下，然后说：“再见了傅华，你不要再跟我联络了，从这一刻开始，我会让自己把你给彻底忘掉，开始我的新生活了。”<br/>　　<br/>　　    傅华的心是很惆怅的，他没想到他来邀请谈红回国帮他，不但人没请得到，最终却变成了谈红跟他了断过往情缘这种情形，不过谈红能够往前看还是让他心有些欣慰的。<br/>　　<br/>　　    傅华笑了一下，说：“祝福你了谈红，你一定会找到你想要的幸福的。”<br/>　　<br/>　　    几个小时之后，飞机降落在了洛杉矶国际机场，傅华不得不收拾起惆怅的心情了，洛杉矶是他此行的主战场，危机四伏，他必须要打起十二分的精神应付才行的。<br/>　　<br/>　　    傅华很快看到了周安强，他和一个健壮的年轻男子站在一起，那个健壮的男子拿着写着傅华名字接机牌的。<br/>　　<br/>　　    周安强本人照片看去更黑更瘦一些，不过也显得更健康一些。傅华走过去，笑着伸出手来，说：“你好，周会长，我是傅华。”<br/>　　<br/>　　    周安强是洛杉矶华人商会的会长，因此傅华称呼他为周会长。<br/>　　<br/>　　    周安强笑着跟傅华握了握手，说：“你好傅先生，欢迎你来洛杉矶。”<br/>　　<br/>　　    周安强跟傅华握完手之后，介绍了身边的那位强壮的年轻男子，说：“这是我的司机马勇。”<br/>　　<br/>　　    傅华笑着跟马勇握了握手，说：“你好马先生。”<br/>　　<br/>　　    马勇笑了笑说：“您好傅先生，跟我不用这么客气了，你叫我小马好。行李给我吧。”<br/>　　<br/>　　    马勇把傅华的行李接了过去，三人了车，离开了机场。在车，傅华看了看周安强，笑着说：“周会长，这一次我来洛杉矶可什么都靠你了。”<br/>　　<br/>　　    周安强笑了笑说：“傅先生太客气了，我跟黄董那是过命的交情，帮他这点小忙理所当然，你说谢谢，那是见外了。”<br/>　　<br/>　　    傅华笑了笑说：“感谢还是应该的。周会长，这两天有什么异常情况没有啊？”<br/>　　<br/>　　    周安强笑了笑说：“没有什么异常情况，我让人盯着楚歌辰和那对母女，他们什么都跟往常一样。”<br/>　　<br/>　　    傅华的眉头皱了一下，说：“是这样的啊，怪了，好像事情不该这样平静才对啊。”<br/>　　<br/>　　    周安强笑了笑说：“也许他们并不知道你来洛杉矶了吧？”<br/>　　<br/>　　    傅华说：“那不可能，我来之前跟齐隆宝见过面的，我跟他说过我要来洛杉矶的。”<br/>　　<br/>　　    周安强说：“那我不知道了，反正我派去盯着他们的人跟我说他们没什么异常。傅先生，你也不要太着急了，这种事情急也是没什么用处的。”<br/>　　<br/>　　    傅华点了一下头，说：“我知道不能急的。不过我总要找个点展开调查才可以的。周会长您对这边的情形较熟悉，您认为我该怎么做才好？”<br/>　　<br/>　　    周安强笑了笑说：“傅先生，你稍安勿躁。跟你说，黄董把你这件事情交代给我之后，我已经着手做了一些工作了。在你来之前，我已经收买了一个楚歌辰身边的人，他答应我说会在这一两天把能够证明楚歌辰和齐隆宝之间有往来的证据提供给我。”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
