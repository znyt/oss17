        童筱颖有些抱怨道：“当你的傅家大少奶奶好累啊！每天不是陪爷爷下棋品茶就是陪婆婆刺绣插花跳舞，还得去参加什么画展啊！时装周啊！音乐会什么的！你赚钱好辛苦的样子，要养这么多的闲情逸致。”
<br/>
<br/>        傅夜沉忍俊不禁道：“我爷爷名下有茶庄，以及老年人俱乐部，还有古玩店在为我爷爷赚钱。我妈妈名下则是绣庄、上流名人会所、以及赞助投资一些时尚产物带来的盈利。要说我的赚钱，基本上是只养你一个了，所以，我并不是很辛苦。”
<br/>
<br/>        他现在在广招人才，基本上每个子公司都有总经理或总裁任职替他把关，还有第三方的审计公司帮他盯着，他只要决策重大的问题，以及按时召开总部会议即可，集团的运作正向顺轨方向前进。
<br/>
<br/>        而且，他现在还有地下势力。
<br/>
<br/>        夏阳枭命令他解散地下势力，他并未解散，而只是把他们安排到明面上去工作，他这么做，也是为了防止夏阳枭过河拆桥。
<br/>
<br/>        傅夜沉也不知道是不是自己多心了，他隐约有种不祥的预感绕在心头，总觉得夏阳枭不会那么轻易地放过他和筱颖。
<br/>
<br/>        如今他几乎与夏阳枭的势力抗衡，只不过是筱颖，成了他唯一显露出来的弱点。
<br/>
<br/>        童筱颖听着傅夜沉这话，小脑袋立起来，双手托腮，直勾勾地盯着他的眼睛，笑道：“原来我的老公家里这么有钱啊！”
<br/>
<br/>        “有钱不好吗？你想买什么，就买什么。”傅夜沉单手抵着自己的脸颊，与童筱颖四目相对。
<br/>
<br/>        童筱颖忽然有了个想法，双手抓着屁股下的圆凳，一起跟着身子往傅夜沉面前挪了挪，俏皮地问：“大哥，我有个想法，要不，你给我做投资？”
<br/>
<br/>        “什么想法？”傅夜沉嘴角微扬。
<br/>
<br/>        童筱颖接着说道：“我想开一家卖衣服的网店，还有做一款与服装有关的手机游戏。然后，游戏里的服装，如果玩者喜欢，就可以在线上预定，再批量生产。”
<br/>
<br/>        她这个想法，是跟着冷夫人参加时装周想到的。
<br/>
<br/>        傅夜沉笑了笑：“有想法不错，但你得写份策划案给我，让我看看，我值不值得投资。”
<br/>
<br/>        “策划案？什么是策划案？要怎么写？”童筱颖纳闷地问。
<br/>
<br/>        傅夜沉双手一摊，有板有眼的比划道：“策划案了，就是你对自己想要做的事情所进行的一些策划与计划，里面的内容包括，这件事情，你要怎么做，主题是什么，涉及到的人员，你怎么执行，最终得到的目的，以及所能带来的利润等等。”
<br/>
<br/>        “这么复杂啊！”童筱颖难以置信。
<br/>
<br/>        傅夜沉忍俊不禁道：“你把计划理清楚了，就不复杂了。”
<br/>
<br/>        “还是好复杂。”童筱颖瘪了瘪嘴，抱怨道。
<br/>
<br/>        傅夜沉剑眉微扬：“你这是打算改行吗？”
<br/>
<br/>        “改行？我原来是做什么的？”童筱颖不禁好奇地问道。
<br/>
<br/>        傅夜沉却苦笑：“建筑设计与室内设计。”
<br/>
<br/>        “怪不得我会画画。”童筱颖笑着说，顿了顿后，又问道，“我为什么会学建筑设计啊？”
<br/>
<br/>        “你是为了……”傅夜沉欲言又止。
<br/>
<br/>        难道要告诉她，她是为了阿景吗？
<br/>
<br/>        她为了阿景，付出过很多很多，但最后，阿景负了她的一片真心。
<br/>
<br/>        “嗯？”童筱颖在等待着傅夜沉的下文。
<br/>
<br/>        傅夜沉半晌后，才接着回答：“是为了你自己的梦想。而且，你还是滨江城体育馆外形的设计者，还拿了奖。”
<br/>
<br/>        “我以前有这么厉害？！”童筱颖惊喜不已。
<br/>
<br/>        傅夜沉忍不住刮了刮这小女人的鼻梁，一脸宠溺地说道：“是啊！你很棒！”
<br/>
<br/>        “那你是喜欢以前的我，还是喜欢现在的我？”童筱颖眨了眨眼睛，明眸里像星星一样在放光，问话的语气略带一丝调皮的意味。
<br/>
<br/>        傅夜沉没想到她会突然这么问，怔了一下，才回答道：“都喜欢。”
<br/>
<br/>        “大哥。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “婆婆跟我说，我还是个小婴儿的时候，你就喜欢抱着我亲。”童筱颖粉唇微扬，意味犹尽地说。
<br/>
<br/>        这几天和冷夫人待一起，冷夫人倒是跟她讲了许多关于他傅夜沉小时候的事情。
<br/>
<br/>        “我不记得有这种事了。”傅夜沉略显尴尬。
<br/>
<br/>        小时候的事情，他哪记得那么多。
<br/>
<br/>        不过，他倒是记得，那次筱颖和苏漫雪第一次被爷爷接来傅家大庄园里住，母亲就跟他提起过，说苏漫雪长得像干妈的女儿，还跟他说，他干妈那个女儿刚出生的那天，他就爱不释手，又亲又抱，还想着小妹妹长大以后给自己当老婆。
