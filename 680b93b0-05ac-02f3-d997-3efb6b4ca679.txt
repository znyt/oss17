<br/>    对于燕山本部的消失，十万大山之中并没有什么特殊的反应，现在乃是多事之秋，加上每个大妖王对于自己的领地都看得很重，一般也没谁大举去查探周围的大妖王，也是没有人注意到这一点。
<br/>    
<br/>        陈明将燕山本部安排好，每天除了给弟子讲道，便是在剑峰之上将万剑炼化到万道天竹之中。
<br/>    
<br/>        时间就这样缓缓的过去，风妖老祖，是一种很奇怪的生灵，和朱雀的情况有些类似，都是灵魅，不过是强大之极的灵魅，像是风妖老祖这种，极为仇视其他的生灵，所以进入他领地之中的一切生灵吗，都会被他玩弄至死。
<br/>    
<br/>        风妖老祖并没有答应保护燕山，但是实际上，效果是一样的，杀死其他进入他领地的生灵，就像是他的本能。
<br/>    
<br/>        在风妖老祖的保护下，燕山得以安稳的发展，转眼，已经是一年之后。
<br/>    
<br/>        燕山本部之外万妖之城。
<br/>    
<br/>        万妖之城是众多归降燕山的妖王建立起来的一座城市，刚开始得到炼丹房，大妖们也被培养出来之后，他们的爱好是蹲坐在炼丹房里，看着一颗颗可爱的丹药出炉。
<br/>    
<br/>        丹药越来越多之后，他们的爱好是两三天就去燕山钱庄看一眼自己的灵石数量。
<br/>    
<br/>        看到那可怕的灵石增长速度，总是觉得有些不真实，在来回取出来，存回去几次后，他们终于是相信了。
<br/>    
<br/>        平常大家都是自己在家数灵石，数完灵石然后存到燕山钱庄。
<br/>    
<br/>        不过今天，他们却是汇聚在了一起，因为最近，越来越多出去卖丹药的商队，大部分都遭到了劫匪。
<br/>    
<br/>        众多妖王浑身金缕玉衣，一个个吃得膀大腰圆，满脑流油，最近日子舒坦起来了，灵石源源不断，自然是身宽体胖，拜月也是肥了一圈，刚开始还有妖王觉得太胖不好看，但是拜月说，这才像一个大商人嘛，然后大家基本上都胖了。
<br/>    
<br/>        妖王们齐聚一堂，拜月终于是到了，朝着大家挥手道：“诸位，下午好啊！”
<br/>    
<br/>        妖王们也是齐齐拱手道：“拜月大当家的，下午好啊！”
<br/>    
<br/>        拜月整了整身上的衣袍，继续道：“现在的日子，可是比以前提心吊胆的日子好过多了，而这些，都是拜山主所赐，所以，这第一杯酒，我们首先敬山主！”
<br/>    
<br/>        拜月端起酒杯，邀众人同饮，“敬伟大的燕山之主！”
<br/>    
<br/>        众多妖王也是齐齐举杯道：“敬伟大的燕山之主！”
<br/>    
<br/>        饮下一杯酒，拜月道：“好，现在我们开始说正事，有了山主准许我们经营炼丹房之后，大家的日子都好过了，但是现在，有些人眼红我们的灵石，眼红我们辛苦炼制出来的丹药，他们又不会炼制，所以，他们只能抢我们的丹药，抢劫我们的商队，若是这么继续下去，那么大家都没有好日子过！”
<br/>    
<br/>        不少妖王抱怨道：“是啊，他们这简直就是在断我们的命根！”
<br/>    
<br/>        “这么下去，绝对不是办法，我可不想回到以前那种日夜担惊受怕，每天辛辛苦苦，一年还存不下一万灵石的日子！”
<br/>    
<br/>        “我们越是隐忍，劫匪便越是猖獗，现在的丹药行情，可是前所未有的暴利，我们燕山炼制出来的气血丹，放在外界，可是以前的十倍价格！我们绝对不能继续隐忍下去了！”
<br/>    
<br/>        “拜月大当家的，你是我们之中最富有的，你必须要给大家拿个主意啊！”
<br/>    
<br/>        拜月狠狠道：“我早就想好了，只是背后牵扯的太多，需要很大的魄力，这也是我为什么将大家都召集到这里来的缘故。”
<br/>    
<br/>        有妖王笑道：“不错，要反击，就要给他们一个狠的！”
<br/>    
<br/>        “只有这样，我们才能通行在这个十万大山！”
<br/>    
<br/>        “拜月大当家的，什么主意，说出来吧，算我一个！”
<br/>    
<br/>        “也算我一个！”
<br/>    
<br/>        一时之间，众多妖王群情激涌，纷纷表示他们也要参一脚。
<br/>    
<br/>        拜月挥手示意大家肃静，然后道：“现在的燕山，分为沙海之内与沙海之外，沙海之外已经撤掉了灵兵的管制，想必诸位都是按时交税的，已经得到了在燕山之外配给灵兵的资格。”
<br/>    
<br/>        其他妖王纷纷响应道：“不错，我等在沙海之外，已经拥有了佩戴灵兵的资格！”
<br/>    
<br/>        拜月继续道：“那么你们可知道血衣剑卫最新的战力评估？”
<br/>    
<br/>        不等拜月说出来，已经有妖王先回答了，“据最新的消息，一百血衣剑卫，可以轻松围杀妖王，而且几乎所有的血衣剑卫，都是全身披甲，上好的灵甲，妖王根本奈何不得！不过血衣剑卫，我们可没有任何机会，拜月大当家的，染指血衣剑卫，可是在自寻死路。”
<br/>    
<br/>        拜月哈哈大笑道：“我怎么可能有染指血衣剑卫的心思，你们应该知道，燕山的镖师，也足足有着一万多，而且，他们使用的灵兵，是血衣剑卫更新灵兵之后换下来的，虽然没有血衣剑卫那么精锐，不过同样也是改修了血衣剑卫的功法，镖师，可是准许我们花灵石雇佣的！”
<br/>    
<br/>        有妖王问道：“那不知拜月大当家的，这一次准备雇佣多少燕山镖师？”
<br/>    
<br/>        拜月道：“我想要雇佣全部的燕山镖师，但是我一个人的财力，也是有些支撑不住，所以我才请了大家一起商议。”
<br/>    
<br/>        拜月本以为还要废一番口舌，才能说服所有人，但是还没等他开口，就有妖王道：“不就是雇佣所有的燕山镖师吗？我们这一年也赚了不知道多少灵石了，这才多少，交税都不够，这一票，干了！”
<br/>    
<br/>        “不错，一定要给其他觊觎我们丹药的贼子一个血的教训！”
<br/>    
<br/>        “犯我燕山者，虽远必诛！”
<br/>    
<br/>        拜月大喜，计划着所有妖王雇佣的名额，按照炼丹房的数量平摊下来，于是，一个庞大的商队离开了沙海，朝着他们出售丹药的地点赶去。
<br/>    
<br/>        拜月亲自带领着商队在大道之中前行，在他们的背后，是隐藏埋伏起来的一万多镖师。
<br/>    
<br/>        夜色深沉，在离开沙海之后，他们沿着自己以前的商路继续向前。
