        &lt;h3&gt;第三卷第一百七十二章  欢喜冤家&lt;/h3&gt;
<br/>
<br/>        凌辰和刘诚之各自拉着自己的行李箱走出房门，在走廊里看到了董亚子和董巧奕。
<br/>
<br/>        “凌总，我和亚子来送送你们。”董巧奕看着拉着行李箱的凌辰。
<br/>
<br/>        “谢谢，我们打车去机场。”凌辰没有想到董家姊妹会来送他们。
<br/>
<br/>        刘诚之看到董亚子也是吃了一惊，他往凌辰的身边靠了靠，凌辰嫌弃的看了他一眼。
<br/>
<br/>        “我们来都来了，难不成你还将我们赶走？”董亚子伸手想拿过刘诚之的背包。
<br/>
<br/>        刘诚之缩了缩手：“你干嘛？”
<br/>
<br/>        “我帮你拿啊。”董亚子看着陈诚之那种躲避三舍的样子来了气，一巴掌拍在了刘诚之的胳膊上。
<br/>
<br/>        虽然是冬天，但是董亚子的那一巴掌有些重，刘诚之疼得裂了一下嘴：“我是个大男人，怎么可能要小女人帮我拿东西，好意领了，你还是帮你凌辰哥哥拿吧。”
<br/>
<br/>        “凌辰哥哥，你身边怎么会要这种人？”董亚子给了刘诚之一个白眼。
<br/>
<br/>        “不要我，难道要你吗？”刘诚之生气了，他是什么样的人?
<br/>
<br/>        “你......”董亚子语塞，凌辰和刘诚之都是榆木疙瘩，她怎么就会碰到这样的人呢？
<br/>
<br/>        凌辰看着吵吵闹闹的俩个人对董巧奕说：“一对欢喜冤家。”
<br/>
<br/>        董巧奕也笑着看着俩个吵闹的人，没有想到董亚子那活泼的性子在凌辰和刘诚之的面前都会吃瘪。
<br/>
<br/>        凌辰她也是见过几次，知道凌辰的个性，但是刘诚之她是最近才见，倒是感觉那就是一个没有长大的小直男，萌得可爱。
<br/>
<br/>        董亚子虽然有点娃娃脸，但是人也漂亮着，是那种可爱型的美女，加上身世，在学校里是被人追着跑的那种，到了凌辰和刘诚之这里就是被嫌弃的对象了。
<br/>
<br/>        想想她都忍不住好笑：“确实！这样的人欢乐很多。”
<br/>
<br/>        “巧奕小姐快要毕业了吧？”凌辰也不管后面的俩个人就和董巧奕聊起天来。
<br/>
<br/>        刘诚之和董亚子看了一眼，刘诚之：“果然，美女的魅力还是很大。”
<br/>
<br/>        董亚子懒得理刘诚之了，和刘诚之并排走着。
<br/>
<br/>        前面俩个人聊得热火朝天，后面俩个人别别扭扭。
<br/>
<br/>        到了停车场，俩个人将旅行箱放到后座上，凌辰看着董巧奕开车，直接拉开前车门坐在了副驾驶上。
<br/>
<br/>        刘诚之郁闷的看了一眼凌辰，拉开后车门，请董亚子上车。
<br/>
<br/>        上车后，刘诚之无视做在身边的董亚子，直接对凌辰说：“凌总，去北京的机票已经定好了，我已经给小雅姐说过了，她说她在家等你。”
<br/>
<br/>        “人家凌辰哥哥的女朋友，为什么要你去说？”董亚子没等凌辰说话她就接过了刘诚之的话。
<br/>
<br/>        “董总，我和凌总说话和你也没有关系吧？”这个董亚子就知道怼他，看他不顺眼，再说了，人家凌辰明明是有女朋友的人了，还要追着人家不放。
<br/>
<br/>        “你.......”董亚子语塞。
<br/>
<br/>        “别你呀我呀的，女孩子需要矜持一点。”刘诚之总是感觉心里不爽，直接给董亚子怼回去了。
<br/>
<br/>        董亚子的手扬起来了，刘诚之将胳膊伸过去：“想动手是吧？来，趁现在还有机会，让你摸一摸。”
<br/>
<br/>        “你，无耻！”董亚子脸红了。
<br/>
<br/>        “大家都在车上坐着，我怎么就无耻了？”
<br/>
<br/>        “我们坐前面什么都没有看到，你们继续。”凌辰坐在前排听见后面吵吵闹闹的俩个人心情竟然莫名的好。
<br/>
<br/>        “噗嗤。”董巧奕捂住嘴笑了。
<br/>
<br/>        “巧奕小姐，开好车开好车，让他们闹腾去吧。”凌辰看着一只手开车的董巧奕。
<br/>
<br/>        董巧奕忍住笑，一本正经的开着车。
<br/>
<br/>        到了机场几个人挥手道别，董亚子说：“凌辰哥哥明年见。”
<br/>
<br/>        “明年见，新年快乐。”凌辰也微笑着回答，这一次的贵州之行虽然有坎坷，不过还算是圆满。
<br/>
<br/>        五年，他的总代位置都不会动。
<br/>
<br/>        “刘诚之。”董亚子看着刘诚之：“我春节去杭州给我老师拜年，到时候找你啊。”
<br/>
<br/>        “哦，那要看我在不在杭州。”刘诚之现在是不怎么想和董亚子说话。
<br/>
<br/>        “没事，你在或者不在，你都得在。”董亚子露出两排白白的牙，笑得刘诚之有些恍惚，又想到了他的first  kiss，脸莫名其妙的红了。
<br/>
<br/>        “董总说的是，那欢迎董总到杭州，有可能我在山西。”刘诚之挥了挥手：“两位美女老总再见，欢迎到杭州参观。”
<br/>
<br/>        凌辰和刘诚之在飞机上，看着窗外远处那金碧辉煌的云层，俩个人都不约而同地叹了口气。
<br/>
<br/>        “你叹什么气？”凌辰问刘诚之。
<br/>
<br/>        “你叹什么气？”刘诚之反问凌辰。
<br/>
<br/>        “我是松了一口气，这一次比任何一次都感觉到累。董老头那次请大家吃饭，其实是考验大家的，很多人都没有醉，其实我也没有完全醉到那个样子，只是接了小雅的电话，感觉气血翻涌，上头了。”
<br/>
<br/>        “哈哈哈，你这是无心插柳柳成荫了啊？走了狗屎运，就这还不好好和小雅姐道个歉？”要不是在飞机上旁边还有人，刘诚之简直要开怀大笑了。
<br/>
<br/>        “第二次真是走了狗屎运，本来打赌那个事情，我也是可以参加的，不过何少华太心急了，他一心急我就抓住机会了，后来他一着急就开始赶董巧奕，这个就让人不齿了。”凌辰长长的松了一口气继续说：“董亚子原本对我有意思，还好有你在，她的视线转移了，要不然还真是个麻烦事情。”
<br/>
<br/>        刘诚之：“你意思我不在你就答应了？”
<br/>
<br/>        “不是，太明确的拒绝会伤了大家的和气，从我嘴里说出来的话他们是不会相信的，他们会认为我敷衍他们，但是从你的嘴里说出来就不一样了，你想一想啊，助理都知道我和宁雅的感情有多好，那就证明是真的了。”
<br/>
<br/>        “你的意思你不是真的？”刘诚之有点气愤。
<br/>
<br/>        “我是认真的，只是不知道小雅是不是认真的，希望这一次到北京能看到一个结果。无论结果好坏，我都认了，你说得很对，爱情不是暖昧，我要拒绝暖昧，勇敢的去面对。”
<br/>
<br/>        “这才是凌总哦。其实你去北京的事情我并没有给宁雅姐说，只是故意在路上那样说，让董亚子死心而已。”
<br/>
<br/>        “现在董亚子的心已经不在我这里了哦。年轻女孩子的心思来的快也去得快啊。”凌辰感叹着。
<br/>
<br/>        “什么意思？”刘诚之表示不懂。
<br/>
<br/>        “别装了，难道你没有回味你和董亚子的first  kiss？”凌辰鄙视的看了一眼刘诚之。
<br/>
<br/>        ”那个，你到北京去了也刚好和小雅姐姐对接一下，将投到陈子昂公司的货款拿回来，年后还要给厂家准备一大笔资金呢。“刘诚之换了话题。
<br/>
<br/>        “小滑头！”凌辰不再和刘诚之说话，闭上眼睛休息，他和宁雅，刘诚之和董亚子，嗯，很不错的感觉。
