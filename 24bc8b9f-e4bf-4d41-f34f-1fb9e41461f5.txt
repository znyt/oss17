　　第275章
<br/>　　中午时分，高飞豹引着几个兄弟推着独轮车走在路上，每辆车上都堆着麻袋，里面装满盐巴。
<br/>　　山寨初建，一切都缺，程大雷就撒出人手去采购生活必需品，如粮食，布匹，铁锅，被褥等物。当然，如盐巴这样的重要物资，还得交给高飞豹这样有本事的人办。
<br/>　　当然，这话是程大雷说的，高飞豹觉得他说的是真的。
<br/>　　盐铁是朝廷专营，大批量的采入，价格昂贵不说，官府还要盘查你的来历。所以，高飞豹找到一个私盐贩子，从对方手中买了十车盐。
<br/>　　其实只要你胆子够大，又不怕掉脑袋，贩卖私盐才是真正的一本万利。不过，在青牛大家没有做这生意的基础，一是幽州靠海，二是幽州人少。过程中，高飞豹还特意问了问这生意怎么做，看以后大家有没有入行的机会。
<br/>　　“二爷，走得太累了，要不要歇歇？”有弟兄道。
<br/>　　“歇歇，歇歇。”高飞豹摆摆手：“这路也不好走，大家歇一阵再走。”
<br/>　　“二爷，要不喝点酒？”手下把酒囊打开。
<br/>　　“嗳，开车不喝酒，喝酒不开车，大当家吩咐的，大当家还说：司机一杯酒，亲人两行泪。”高飞豹。
<br/>　　“呃……有时候大当家是会说些稀奇古怪的话，也听不太懂。”
<br/>　　“什么都让你听懂了，大当家还是大当家么。”高飞豹撇撇嘴：“好啦，大家好好歇一会儿，抓紧时间赶路。”
<br/>　　正这时候，身后路上骑马冲过来一股人。高飞豹看着他们，露出警惕的神色，大家的手也都搁到车底，那里藏着兵器，如果这帮人来者不善，大家也只能放手一搏。
<br/>　　高飞豹示意大家稍安勿躁，人家未必是冲着自己来的。
<br/>　　却是没料到，这队人很明显就是冲大家来的，他们快马奔来，将高飞豹等人团团围住。
<br/>　　“是你们从杜黑子手中买的盐？”为首一个大胡子道。
<br/>　　杜黑子就是和高飞豹交易的私盐贩，见对方人数超出己方四五倍，真动起手来，没有什么把握。所以高飞虎脸上打着哈哈：“兄弟，这中间是不是有什么误会？”
<br/>　　“去你娘的，谁和你是兄弟。”大胡子根本没将高飞豹看这眼里，道：“这条路是你老子罩着的，谁让你们从这里过。”
<br/>　　手下十几个弟兄示意高飞豹，直接动手吧，已经有人开始悄悄小范围移动，各自盯住一个目标，只要高飞豹一声令下，定是击杀的局面。但高飞豹已经不是一年前那个愣头青，真动起手来，己方并没有任何优势。
<br/>　　“哈，我们兄弟的确初来乍到，还不懂这里的规矩，不知道哪里得罪了各位好汉，还请好汉赐教。”
<br/>　　“哼，看你也是个明白人，这条路是云中龙龙爷罩着的，你们竟敢在这条路上贩盐，是嫌命长了么。”大胡子道。
<br/>　　“三哥，别给他们废话，直接宰了不就完了么！”大胡子的手下已经有些着急。
<br/>　　高飞豹打了个哈哈：“我以为是谁呢，原来云龙中龙爷呐，我们兄弟刚来，还正打算去拜龙爷的山门。我特意给龙爷准备了一件礼物，还请这位给送一下。”
<br/>　　“你认识龙爷？”大胡子在马上伸出手来：“什么东西啊，拿过来给我看看。”
<br/>　　“这不正往外拿么，您可得在龙爷面前美言几句……”
<br/>　　高飞豹手搁在怀里，脸上堆满微笑，这个时候，骤变突生，他突然暴起，抓住大胡子的手腕，硬生生将对方拉下来，然后蹂身而上，跃到马背上，双手勒住马缰。
<br/>　　其他人也没闲着，抽刀杀人夺马一气呵成。
<br/>　　这突然而来的变故是谁也没有想到，短短时间，已经有七八人折了性命。
<br/>　　“弟兄们，别恋战，撤。”
<br/>　　高飞豹一声呼啸，十几个兄弟舍了盐车，纵马而去。
<br/>　　……
<br/>　　程大雷正在和大家一起翻修山寨，因为山寨缺少人手，程大雷也只能亲自上阵。不过，有建筑精通的技能，倒也不是太麻烦。建筑精通带来两项属性，其一是建筑消耗时间减半，这一项好理解，就像游戏中派出一个农民修建，本来需要一个小时才能完成的建筑，现在只需要半个小时就够了。而所有建筑属性提升10%则是指，山寨的厨房作出来的饭更香一些，在医馆养伤效果更好。
<br/>　　现在，建筑精通发挥的作用还不是很大，可要等后期，铁匠铺，医馆等都建立起来，这项技能带来的提升才是恐怖。
<br/>　　如今程大雷正忙碌着修建居住的房屋，伐木为梁，垒土为墙，一切都在有条不紊的进行着。
<br/>　　“大当家，大当家……”
<br/>　　这时候，高飞豹带着十几个弟兄跑过来，两眼泪汪汪。
<br/>　　“大当家，我受委屈啦。”
<br/>　　咦，程大雷还是很少见到高飞豹这副表情。他忙问道：“究竟发生什么事了？”
<br/>　　“哎，大当家你可是不知道……咱们不能让人这样欺负啊……”
<br/>　　高飞豹鼻涕一把，眼泪一把，程大雷渐渐把事情过程听明白了。
<br/>　　“你是说，你杀了对方七八个人？”
<br/>　　“嗯。”
<br/>　　“然后安然无恙的脱身？”
<br/>　　“嗯。”
<br/>　　“还抢过来十几匹马？”
<br/>　　“嗯。”
<br/>　　“很好，那么问题来了。”程大雷打了个响指：“你究竟委屈在哪里？”
<br/>　　“呃……”高飞豹扮可怜本来是想让程大雷给他出气，被程大雷这么一问，他还真有些无言以对。愣了半晌，他试探着问：“精神损失算么？”
<br/>　　“既然你精神受了创伤，那就好好歇一歇。”程大雷撇撇嘴，又叫来两个兄弟：“你们出去，查一查这云龙中的底细，看对方究竟什么来头？”
<br/>　　说罢，程大雷嘴里自己嘀咕了一句：“不是说琴川这里没有强盗么？”
<br/>　　夜里，派出去打探消息的弟兄已经回来了。
<br/>　　琴川的确没有强盗，云龙中也不能说是强盗，他是官，是莫鸣米封的官，官职是校尉还是什么。在这一代，他属于路霸，走陆地的，不管什么生意，都要拜他的山门，有他的标志就能通行，不然寸步难行。
<br/>
