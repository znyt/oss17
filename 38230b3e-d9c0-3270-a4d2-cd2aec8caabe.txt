　　“好东西。”萧逸自语一笑。
<br/>　　这双兽手，以及兽脚，完全可当控火重宝来用。
<br/>　　嘭…嘭…嘭…
<br/>　　萧逸双手齐出，一道道紫炎游龙凝聚半空。
<br/>　　心念所动间，游龙翱翔。
<br/>　　上百道紫炎游龙的火光，将整个高空映衬得一片紫色。
<br/>　　火焰游龙的数量，还在增加。
<br/>　　不多时，渐渐达到千条之数。
<br/>　　要知道，以前的萧逸，顶多控制数十上百条紫炎游龙，已经是极限了。
<br/>　　毕竟这些紫炎游龙由他的紫炎凝聚而成，威力惊人。
<br/>　　现今，竟是轻松操控千条。
<br/>　　高空中，千条游龙，栩栩如生，宛若嬉戏云间，狂猛而美丽。
<br/>　　但在这美丽之下，萧逸绝对相信，这千条游龙，足以爆发出骇人的威力。
<br/>　　萧逸再度看回双手，眉头微皱，“到底是什么妖兽，竟有如此可怕的控火能力。”
<br/>　　在他的对比中，这般可怕的控火能力，就连火儿赠予的火族血脉都远远比之不上。
<br/>　　“散。”萧逸低喝一声，散去了兽手与兽脚。
<br/>　　感知了一番，两只兽手与兽脚，已重新出现于火焰空间内。
<br/>　　他已大致尝试了一番威力。
<br/>　　至于真正的威力如何，还是留待日后战斗时再去体会吧。
<br/>　　现今，还是需要处理这些金光力量，还有八龙焚火炉能冲破多少道封印的问题。
<br/>　　萧逸等待着。
<br/>　　半个时辰后。
<br/>　　八龙焚火炉上，再次传来冲击一般的轰鸣声。
<br/>　　萧逸感知了一番，点了点头，“填满了。”
<br/>　　八龙焚火炉内，再次填满了金光力量。
<br/>　　也就是说，金光力量的再次涌入，将会挤压空间，准备冲击封印。
<br/>　　萧逸脸上虽带着喜色与期待之色，但同时，一抹凝重也霎时涌现。
<br/>　　额头上，一滴冷汗，悄然涌现。
<br/>　　一双锐利的眸子，直直地盯着八龙焚火炉。
<br/>　　手中握紧了冷焱剑以及天机圣盘。
<br/>　　刚才第四道封印解开，尚且爆发火焰惊人，差点没吓他一跳。
<br/>　　如今第五道封印，明显更强，那么爆发的火焰将恐怖到什么地步？
<br/>　　萧逸不敢想象，也无法想象。
<br/>　　他能做的，只有做好万全准备。
<br/>　　在这冰玄封界阵内，萧逸还是比较有把握的。
<br/>　　时间，渐渐过去。
<br/>　　十数分钟后。
<br/>　　萧逸的眉头，开始皱了起来。
<br/>　　因为，八龙焚火炉上，此时毫无要突破封印的征兆。
<br/>　　虽然仍旧剧烈抖动，却无半分火焰气息溢散。
<br/>　　又是十数分钟后。
<br/>　　萧逸面露疑惑之色，连忙放出感知。
<br/>　　感知之下，八龙焚火炉内，此刻的金光力量早已填满。
<br/>　　甚至于，继续涌入的金光力量，已呈压缩之势。
<br/>　　可火焰空间内的封印，仍无半分松动。
<br/>　　整整一个时辰后。
<br/>　　萧逸摇了摇头。
<br/>　　在他的感知下，八龙焚火炉内，早已填满金光力量。
<br/>　　可是，在第五道封印面前，这些强悍至极的金光力量，根本奈何不得。
<br/>　　哪怕它们数量再多，力量再浑厚，也奈何不了封印分毫。
<br/>　　不难想象，第五道封印，必是极强。
<br/>　　萧逸也未纠结，毕竟是天地至宝，里头封印，怎可能是等闲力量能破的。
<br/>　　看来，金光力量的冲击，到这里便结束了。
<br/>　　现今，萧逸也该思索回之前的问题了。
<br/>　　八龙焚火炉内，到底有什么秘密。
<br/>　　当年易老到底是从哪里得到这东西的。
<br/>　　还有那些妖兽尸骸，第五道封印之内，如无意外，必是妖兽右脚。
<br/>　　这只妖兽，到底是何等强悍的存在？
<br/>　　若是八道封印尽皆解除，又是否会有意外或者变数？
<br/>　　萧逸的眉头，紧紧皱着。
<br/>　　半晌，他仍旧想不出些什么。
<br/>　　毕竟这八龙焚火炉他虽掌控着，却并不太了解。
<br/>　　不过，他倒是想到了‘实力’二字。
<br/>　　无论八龙焚火炉内有什么秘密，或者之后有什么变数。
<br/>　　只要有足够的实力，便不惧这些。
<br/>　　特别是日后的第八道封印，他不靠外物冲击，只靠自己实力便是。
<br/>　　若是自己的实力强大到足够打开第八道封印，那么，无论里头有何变数，他都不必担心。
<br/>　　思索至此，萧逸暂时缓去了脸上凝重。
<br/>　　在炎龙大陆，之所以实力为尊，便是因为实力二字，便代表着一切。
<br/>　　萧逸收敛了思绪，看向八龙焚火炉。
<br/>　　现今，既八龙焚火炉的第五道封印已无法冲击。
<br/>　　那么，待得金光力量吸收到极致时，便可收回封炉了。
<br/>　　萧逸暂时等待着。
<br/>　　大概半个时辰后。
<br/>　　金光力量的涌入，渐渐停止。
<br/>　　萧逸点了点头，“看来到极点了。”
<br/>　　事实上，八龙焚火炉内，早已填满了金光力量。
<br/>　　后面的继续涌入，不过是压缩。
<br/>　　现在，已到极点，即便再压缩也无法继续吸摄金光力量了。
<br/>　　“大阵，起。”萧逸低喝一声。
<br/>　　接下来，封炉即可。
<br/>　　萧逸手中一个个手印打出。
<br/>　　刚要彻底封炉，眼眸，忽然一冷。
<br/>　　“嗯？”萧逸脸色一冷，直视高空。
<br/>　　那里，一道身影忽然凭空而现。
<br/>　　来人，是个老者。
<br/>　　那张狞笑着的面庞，有些熟悉。
<br/>　　“北隐宗大长老。”萧逸双眼一眯。
<br/>　　“呵呵，萧逸小贼。”北隐宗大长老冷笑一声。
<br/>　　不错，老者正是北隐宗大长老。
<br/>　　“还真是冤家路窄。”
<br/>　　“老夫不过是恰好来金光险地办些事，却遇高空异象，千道游龙齐舞。”
<br/>　　“本以为是重宝出世，或者有什么异状。”
<br/>　　“没想到这些异象竟是你萧逸小贼弄出来的。”
<br/>　　北隐宗大长老冷笑着。
<br/>　　一双苍老的眼眸，直视金光坑洞之上漂浮着的八龙焚火炉。
<br/>　　“竟在吸收金光力量？”
<br/>　　“好澎湃的气息，这东西，起码是上品圣器。”
<br/>　　北隐宗大长老的眼眸中，闪过一丝贪婪之色。
<br/>　　萧逸明显注意到他的目光和脸色，手中冷焱剑紧了紧。
<br/>　　北隐宗大长老注意到萧逸的动作，看向了萧逸手中之剑。
<br/>　　“嗯，那把剑也不错，起码是中品圣器。”
<br/>　　“还有你那阵法，老夫这般眼力，竟一时认不出是何等阵法。”
<br/>　　“啧啧，看来你身上的宝贝可不少。”
<br/>　　“怎么，想拿？”萧逸冷笑一声。
<br/>　　“你说呢？”北隐宗大长老同样冷笑。
<br/>　　“真以为你身后站着两个巅峰剑修，老夫就不敢动你？”
<br/>　　“这里可是金光险地，而你又身处金光禁地。”
<br/>　　“桀桀，每年死在金光禁地的狂傲天骄，可不少。”
<br/>　　说到最后一句，北隐宗大长老脸上，杀意凛然。
