　　第98章
<br/>　　程大雷六人闻得风动，也急匆匆赶到城门。
<br/>　　城门倒着几具尸体，鲜血涂在城墙上，一群人聚在一起，议论纷纷。
<br/>　　“过来就杀人，一斧头就把脑袋砍成两半！”
<br/>　　“这蛤蟆寨的山贼真够狠的，可不敢招惹他们！”
<br/>　　鲜血溢在空气中，恐惧在弥漫，程大雷的恐惧值蹭蹭增长。
<br/>　　程大雷和身边几人对视一眼，心中都明白一件事：正主找到了！
<br/>　　“那是老子的斧头！”程大雷将牙磨得咯吱咯吱响。
<br/>　　“估计冒我们名字，屠杀贾家满门的也是这伙人。”刘悲。
<br/>　　双方人马都躲在城里，城里官兵展开大搜捕，没抓到程大雷一伙，却是把这伙人逼了出来，所以他们才会冒险破门而出。
<br/>　　马蹄声急响，卷起烟尘，城门拥挤的人群纷纷散开一条道路。只见一支队伍快马而来，是陆哼统帅的乡勇军到了。
<br/>　　程大雷慌忙低下头，却也偷眼打量着陆哼。今天陆哼好俊的扮相，铜盔铜甲骑在高头大马上，手持一杆熟铁棍。上次把兵器丢了，看来这次是又打了一把。
<br/>　　他身后的部队穿得却不是统一的制服，有穿皮衣的，又穿棉衣的，也有手持长剑着公子儒袍。
<br/>　　“陆将军，一定要凯旋回来，奴家会等你的。”
<br/>　　“朱公子，路上要保重喔！”
<br/>　　……
<br/>　　红袖招摇，一群红粉从街头巷陌钻出来，脂粉飘香。美目湿润，依依惜别之意，尽在不言之中。
<br/>　　“敏儿，等我杀贼回来，就到你家提亲！”
<br/>　　“珠珠，你放心，今天我一定诛杀了程大雷那狗贼！”
<br/>　　“我知道你是个英雄，我从没有看错你！”
<br/>　　英雄在歧路，儿女共沾巾。
<br/>　　程大雷瞅着这一幕：“这是打仗，还是干嘛呢？”
<br/>　　“城里组织了乡勇军，都是慷慨激昂的仗义之士，目的就是伸张正义，杀了你这个恶人。”徐神机。
<br/>　　程大雷有些无语，倒也不是没有好处，此刻他的恐惧值已经增长到七十万，远超刚来黑石城的目标。
<br/>　　黑石城有十万人，一人贡献一点恐惧值都能达到十万，而贾家满门被灭后，大部分人心惊胆战，如临倒悬，所以程大雷的恐惧值也不停往上涨。
<br/>　　嘟，正义的少年喔，恭喜你的恐惧值接近一百万，当恐惧值达到一百万可随机兑换顶级品质1 。
<br/>　　忽然收到一条系统提示，程大雷愣了愣，心里琢磨着这件事。一百万恐惧值能连抽一百一十次，且必然能获得优秀品质10 。那么是一百名喽啰加十名优秀等级值钱，还是一名顶级品质值钱。
<br/>　　这笔账，不太好算。
<br/>　　此时，陆哼已经带领乡勇军出城，城门乱哄哄一片，有慷慨激昂者，有捂帕抹泪者，有长吁短叹者。
<br/>　　“大当家，人都走了！”徐神机唤道。
<br/>　　“咱们也走！”程大雷回过神来：“把我的斧头抢回来！”
<br/>　　白茫茫雪地，几匹快马狂奔，老天在白面上洒下几粒黑芝麻。在他们身后五里处，便是陆哼率领的三五百乡勇军。
<br/>　　这伙人里也是什么都有，有舞文弄墨的才子，有看家护院的壮汉，有打猎的猎户。身下骑得马、驴、骡子……还有骑牛的。
<br/>　　“小王子，咱们的人什么时候到？”
<br/>　　“算时间就该是今天到了。”呼尔勒骑在马上，手中提着一柄大斧：“陪他们在城外耍耍。”
<br/>　　旷野积雪颇厚，马的速度被拖慢，身后追兵渐近。
<br/>　　“赤虎，放他们近些，给他们点颜色看看。”
<br/>　　“好嘞！”
<br/>　　卸下身上铁弓，抽出一柄铁箭搭在弓弦上，赤虎两条腿夹紧马，在马背上转过半个身子。
<br/>　　几人骑马的速度都慢下来，三里，二里……三百步，二百步……双方的距离越来越短，连对方的枪缨子都快能看得清楚。
<br/>　　弓开若满月，箭走似流星，一箭所发击中一人的胸口，踉跄跌在马下。
<br/>　　陆哼倒抽一口凉气，此人当真是好俊的骑术。
<br/>　　一百步外，马上开弓，竟然能一箭取人性命。
<br/>　　正应了那句话，这些人的骑术当真是恐怖如斯！
<br/>　　蛤蟆寨的山寨能有这么俊的骑术么，陆哼和程大雷照过面，对方本事怎么样，他清楚得紧。
<br/>　　这些人绝非程大雷率领的山贼，倒很像是……戎族。
<br/>　　戎族生在马背上，便是十岁顽童都有出神入化的骑术。陆哼的心脏猛地被攥紧，相比程大雷，戎族可要恐怖多了。
<br/>　　几人在马上开弓引箭，竟无有不中，乡勇军中纷纷有人倒下，恐怖的氛围在蔓延。
<br/>　　疾奔之中，呼尔勒突然勒马停下，他的同伴也在同一时间刹住马。
<br/>　　如此控马之术，当真是恐怖如斯！
<br/>　　拨转马头，两军对垒，相距大概一百步。
<br/>　　“北蛮部的勇士们，陪他们耍耍！”
<br/>　　“让俺来！”野德拍马出阵，铁枪横直：“那个不想活的，速速上来送死！”
<br/>　　陆哼道：“谁去会他！”
<br/>　　“陆将休急，让我去会会他，三合之内必取他人头。”
<br/>　　一骑灰马的壮汉拍马出阵，此人姓张名勇，乃是城内一武馆的师傅。擅使铁枪，城中很多人都是他的徒弟。
<br/>　　“张师傅出马，定能诛杀这恶贼，为民除害。”
<br/>　　“今日我们且看张师父如何跃马扬威。”
<br/>　　野德和张勇已经战在一处，打得难解难分，两马相交之时，野德使了一招镫里藏身，一枪从背后将张勇挑在空中。
<br/>　　鲜血撒在雪地上，也淋在野德身上，野德浑身浴血，骑在马上哈哈大笑。
<br/>　　“来呀，还有谁！”
<br/>　　野德连战三人，连战连胜，最后因力亏退下。这次换了呼尔勒亲自上阵，他更加豪横，一柄铁斧使得赫赫生威，接连几人倒在他的马下。
<br/>　　这斧头陆哼是认得的，可这人看着有些眼生。当然，这斧头在人家手中和在程大雷手中根本不能相提并论。
<br/>　　难道说，程大雷这恶贼和戎族也有勾结，不然他的斧头怎么在对方手中。
<br/>　　心中咯噔一下，突然冒出个名字。
<br/>　　“戎族小王子……寒地苍狼！”
<br/>　　呼尔勒骑在马上，血顺着斧刃流淌在地上。
<br/>　　“是我，怎么了？”
