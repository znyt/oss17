　　此火山内成空心状，倒斗形，周围洞壁都是漆黑色，这是以往火山喷发后留下的痕迹。元始顺着火山口做自由下降运动，控制自己的身体向下，不过，渐渐地，周围温度慢慢变高开始感到灼热！
<br/>　　如此下降数十里，到此位置，空气已变得灼热难忍，空气之中都带有一种焦灼的味道，像是空气都在灼烧，还有浓烈的硫磺味。
<br/>　　元始的神色依旧很轻松，尽管此处的温度极高，但对于他来说，这并不算什么。先不说他的肉体真身强悍，足以应对比这更高的温度，而且他自开辟了心之神藏道宫，孕育出赤帝火皇神邸来，所有火焰，都已对他无效了。
<br/>　　周围洞壁一片赤红，像是在燃烧一般，要是金仙以下的修士到此，身体早就融化了，就算是金仙强者，也不敢在此久留，因为此处的温度实在是太高了，难以忍受。
<br/>　　随着下降越深，元始就感觉温度越高。最后，下降了近百里，元始落到了地上，那是一块凸起的大石头。元始环顾四周，洞壁都是赤红一片，在他脚下，是一片流动的赤红液体，这是岩浆。在他的脚下，不断地冒着热气翻滚，周围都被照得赤红。
<br/>　　如果不小心掉进此处的岩浆之中，恐怕就是太乙金仙都要饮恨，因为温度太高了，超出了他们的承受极限。就连一般的大罗金仙到此，都需要忍受着极高的温度。
<br/>　　在这岩浆之中，竟然还有大石头，可就这是多么不一般的石头，其强度已经可以堪比大多数大罗金仙的肉身强度了。这里的每一块石头，对于一般的修士，都是不可多得的器材。
<br/>　　可惜，这对于元始来说，没用。既然没用，元始对此也就毫无所动了。
<br/>　　这片空间很大，地下像是空心的，赤红的岩浆流淌，像是一片赤色的海洋，元始站在岩浆中凸起的石头上，就像是岩浆之中一粒比较显眼的沙尘。
<br/>　　在元始的头顶之上还有许多倒挂垂下来的巨大钟乳石，只不过是赤红色的，还有一滴滴赤红的岩浆滴下来，极为壮丽。
<br/>　　不过，元始并没有心情看这些，他稍微感应一番后，就直接钻进脚底的岩浆之中，不溅起一滴岩浆，只是冒了几个水泡，就消失不见了。
<br/>　　元始在岩浆之中行走，就好像是在普通陆地之中行走，他所过之处，岩浆既不暴动，也不翻浪，如履平地。温度极高的岩浆一靠近他的身体，连一丝头发都不能烧焦，就被吸走。
<br/>　　元始的身体之内好像有一个恐怖的黑洞，将他身体周边的岩浆全部吸走，这些岩浆的去处，正是心之神藏道宫之中，一旦进入，全部被镇压吞噬。
<br/>　　不过一会儿，元始就到了岩浆深处，停在一座火红雄壮华丽的宫殿之前。这座宫殿极为豪华，全身赤红，散发着赤金色火焰，透露着一股贵不可言的高贵气势，让人不得靠近。
<br/>　　“轰！
<br/>　　轰！
<br/>　　轰！……”
<br/>　　就在此时，岩浆剧烈翻滚，变得沸腾，赤红色液体暴动，一同向元始狂涌而来，一股恐怖的气势锁定元始，这气势极为恐怖，像是一尊远古沉睡的神魔复苏，周围空间出现剧烈抖动现象，这片空间像是不稳，马上就要崩溃，岩浆变得狂暴沸腾，如同火山喷发之前的沸腾！
<br/>　　“嗡嗡嗡嗡……”
<br/>　　这一刻，焚炎谷外也变得极不平静，大地似乎在震动，而且还伴随着嗡嗡作响。
<br/>　　而岩浆之中，岩浆化为巨浪向元始卷来，滔天的恐怖威压也压向元始，像是要将他毁灭。
<br/>　　元始不为所动，岩浆一近他身，就被吸走，不管多少岩浆，也不管他有多狂暴，多炙热，多热烈，也威胁不了他。
<br/>　　那恐怖的威压，到了元始身前三丈，便停了下来，元始心念一动，一道同样带着无比恐怖威压的不周印出现在他头顶，将火焰之中的威压全部挡在三丈之外，不得寸进。
<br/>　　“啾！”
<br/>　　就在此时，一道嘹亮清脆的鸟鸣，仿佛来自荒古，划破了无尽虚空，从巨大宫殿之中传出。这声音太过嘹亮，而且仿佛有一种魔力，摄人心魂，让人有一种来自灵魂深处的颤栗，要是一般大罗金仙听到这声鸟鸣，都会变得不安。
<br/>　　只是这对于元始来说，鸟鸣传到他的灵魂之中，不能掀起一道波纹，就好像从来没有来过一样。
<br/>　　岩浆变得更加狂暴了，整个岩浆空间，像是一片炼狱，虚空在剧烈抖动，空间变得脆弱不堪，一道道裂缝出现又消失。
<br/>　　“啾！”
<br/>　　最后，那道摄人心魄的鸟鸣再次响起，在元始的眼中，出现了这样一幅场景：一头巨大的生灵出现在宫殿之上，双翼张开，若垂天之云，足足数千丈，遮天蔽日，像是整个岩浆空间都被这庞大生灵覆盖。
<br/>　　生灵浑身散发着无尽令人心悸的金红色火焰，两者眼睛虽然在冒火，但神色却是十分冰冷。
<br/>　　鸿头、麟臀、蛇颈、鱼尾、龙纹、龟躯……
<br/>　　神威如狱，神恩如海！
<br/>　　这生灵的身份自然不用多说，元始第一眼看到，就知道这是四象神兽之一的朱雀神兽无疑了。仅仅朱雀就如此神威，真不知道其余三大四象神兽白虎、玄武、青龙又将如何？
<br/>　　但紧接着，元始便想到需要怎样向这头朱雀索要神兽之力？难不成要强取豪夺？元始摇了摇头，这样不行，神兽一般都是十分高傲的，有些事宁愿死也不会妥协。想必这头朱雀也不例外。
<br/>　　那要怎么办呢？商量一下？要怎么商量呢？元始回想着关于朱雀的信息，顿时，就有了主意。不过，在此之前，还要看人家愿不愿意？
<br/>　　这不，元始这里还没有开始行动呢，朱雀就已经开始攻击了，显然，他将元始当成是入侵他领地的敌人了。
<br/>　　“啾！……轰！”
<br/>　　巨浪滔天，岩浆之中掀起无边巨浪，朱雀口吐神火，神火临世，天地暴动，这是朱雀神火，朱雀的本命神火，足以堪比像太阳真火那样的十大真火，攻击性极强，无物不焚，普通大罗金仙在这样的神火之下，片刻就会被烧化为灰烬。
<br/>　　看着朝自己轰来的朱雀神火，元始心里苦笑，得了，这仗看来是不打不行了，本来还想着用什么办法来和平交易的，但如今……
<br/>　　也罢，就让本座来看看这开天就存在的四大神兽之一的朱雀神兽到底有多强！
<br/>　　况且，自己若是不显露一番实力，朱雀也不会那么容易就答应你。
<br/>　　看着火龙一般向自己攻来的朱雀神火，元始不慌不忙，淡然处之，还不等神火近身，元始就张开大口，一口就将那令人色变的恐怖神火吞入腹中。
