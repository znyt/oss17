　　“轰隆隆！”
<br/>　　一阵轰响，只见后土的新肉体快速凝聚，霎时间，变得无比强横，她的气势一路飙升，大罗金仙、准圣中期……准圣圆满，当后土的气势达到准圣圆满时，微微一滞。
<br/>　　“轰！”
<br/>　　在洪荒诸圣以及所有大神通者的紧张注视下，在万众瞩目之下，天地间渐渐响起仙乐，霞光漫天，瑞丝千条，地涌金莲，天坠金花，紫气浩荡，灵气化形。
<br/>　　借助开辟轮回的无量功德之力，后土一举打破瓶颈，成道混元大罗金仙。
<br/>　　开辟轮回乃造化天地之大事，千秋万载之无量功德，堪称开天之后的第一功德之力，这比女娲成圣的功德还要多，毕竟女娲也只是创造出一个后天种族，若不是这后天种族正好关乎人道，她得到的功德也没有那么多。
<br/>　　而后土开辟轮回之事，却是对洪荒万灵，天道进化，世界进化有着无比重要的关系，所以她得到的功德也堪称无量。
<br/>　　“嘶！”
<br/>　　一阵抽气之音响起，洪荒诸神纷纷大吃一惊。这是洪荒第一个，没有用鸿蒙紫气而成道，证得混元大罗金仙的存在。当然，那是因为他们此时还不知道元始证道混元一事。
<br/>　　这怎能不令人惊讶，怎能不令人震惊？
<br/>　　后土此举，为洪荒后来人开辟了一条先河，坚定了许多人的道心，让众神不至于那般绝望，他人能行，那么，自己为何不行？
<br/>　　有了第一个，后面的人自然也就看到了希望。
<br/>　　是以，看到这一幕，洪荒众多准圣，临近证道的大神通们，纷纷激动不已。
<br/>　　其中，以冥河道人为最。他之前创造阿修罗族，建立幽冥教，得到无数功德，成为准圣大圆满强者，要说距离圣人最近，那便属于他了。
<br/>　　“后土娘娘，恭喜了！”
<br/>　　“恭喜道友，证得混元！”
<br/>　　洪荒诸圣与众多大神通们纷纷降临，一个个面带微笑，向着后土开后恭贺道。
<br/>　　“诸位不必多礼！”
<br/>　　后土脸色平淡，面对诸圣，不起丝毫波纹，淡然道：“诸位请回吧，轮回初开，本宫还有许多事要忙，这里难以招待各位！”
<br/>　　“如此，我等便告辞了！”
<br/>　　诸圣见此，也识趣的离开了。此时三生道人才有机会上前与眼前的后土说话。
<br/>　　他看着眼前的后土，这个他既熟悉又有些陌生的后土。熟悉那是因为他在她身上感受到了后土祖巫的气息，陌生是因为她在他身上感受到了另一股苍茫的气息。虽然那股气息很轻微，就算圣人也难以察觉，但这还是让三生道人发现了。
<br/>　　当然，最主要的原因还是因为，那股陌生的气息他也有些熟悉，因为那是属于混沌的气息，确却的说，那是属于混沌魔神的气息。
<br/>　　对于混沌魔神那独特的苍茫的气息，三生道人自然不会陌生，因为他当年也是混沌魔神。
<br/>　　当后土元神刚刚出现的瞬间，三生道人就发现了它。同时，三生道人也终于知道了，为何后土的血脉之中除了土之法则之外还有一些神秘的法则碎片。
<br/>　　原因只有一个，那就是轮回魔神转劫到后土身上。
<br/>　　至于为何轮回魔神之前一直没有觉醒，那估计是因为后土为盘古血脉，有盘古血脉的压制，以至于她没有能够及时觉醒。当然，也可能是其他原因。
<br/>　　不过，那都不重要了，重要的是，如今的后土已经不再是原来的后土了，而是后土与部分轮回魔神的结合。对此，三生道人不知道该如何面对。
<br/>　　其实，三生道人的猜测很正确，当年轮回魔神的确是被盘古劈死了，然而，她因为执掌轮回大道法则，逃过了一缕残魂，之后转劫到新开辟的洪荒之中。
<br/>　　但因为时间的缘故，再加上盘古血脉的消磨压制，以至于她这缕残魂被弱了再弱，如今已经不剩多少力量了。
<br/>　　若不是此次后土化身轮回，那缕残魂也不会觉醒，而后土也不会因此有了元神，能够证道混元大罗金仙。
<br/>　　但，就算觉醒了，如今的后土也只是接受魔神部分记忆的后土，并没有出现哪个吞噬哪个的现象，因为经过无数年的磨合，她们早已合一了。
<br/>　　看着眼前那漂亮的女子，三生道人眼中闪过复杂之色，叹了一声，道：
<br/>　　“你是后土，还是轮回？”
<br/>　　三生道人知道后土能够明白他所说的话。果然，对面的女子淡然一笑，道：
<br/>　　“后土？还是轮回？又有何分别？今后世上哪还有什么后土与轮回，我为平心，永镇幽冥的平心。”
<br/>　　平心娘娘脸上一片淡然神色，竟如同那合身于天道的鸿钧道祖一般，合身于道，已经到了至公无私的地步。
<br/>　　后土已经不在了，轮回也已经消失了，有的，只是平心娘娘！
<br/>　　只见平心娘娘挥手一道光华流过，忽然之间，九幽震动，赫见地府之中，平地升起一座宫殿，上书“平心殿”。殿门缓缓打开，平心娘娘走了进去。
<br/>　　“也罢，不管你是后土，还是轮回，都已经不重要了。那么，我也该完成我的使命了。”
<br/>　　三生道人望着进入殿门的平心，沉默片刻，忽然一笑，那叫一个洒脱。
<br/>　　“今我三生道人，入主地府，掌幽冥，主三生，以生死簿，束洪荒众生之灵。自今日起，幽冥地府正式运。十八层地狱，开，天下苍生，皆载入生死簿，阳间生灵，死后必入地府，评判功过，奖惩公罚……”
<br/>　　陡然，幽冥界中央，三生道人一声大喝，全身准圣后期的气势，绽放而出，惊动洪荒。
<br/>　　“什么？三生道人？那是谁？”
<br/>　　“三生道人，掌管地府？这……”
<br/>　　洪荒诸圣以及众神纷纷大惊，不知这三生道人是何人，轮回刚建，就敢在平心娘娘的眼下夺取地府职位，真是大胆。然而，对此平行娘娘竟然还不发怒？
<br/>　　地府之中，一阵阵阴气翻滚，九幽巨震，幽冥四方，陡然升起一片建筑，巍峨大气，古朴异常。
<br/>　　“彼岸花，三生石，奈何桥，以我生死簿为主，皆听调遣……”
<br/>　　三生道人一声冷喝，面色威压，带有一丝霸气。
<br/>　　“自今以后，吾为地府之主，三生大帝。”
<br/>　　之后，从三生道人身上再次走出三道身影，分别向着地府不同的方向飞去。
<br/>　　“我为地狱之主！”
<br/>　　不久，一声大喝从十八层地狱方向传出，惊动九幽，震动洪荒。
<br/>　　“我为九幽大帝！”
<br/>　　与此同时，九幽深渊之处，同样传来一声大喝。
<br/>　　“我为万灵冥帝！执掌幽冥万灵！”
<br/>　　另一个方向，同样传来一声大喝。
<br/>　　然而，三生大帝依旧不停，继续喊道：
<br/>　　“五方鬼帝出，十殿阎罗现！”
<br/>　　他一指点出，三生道人面前接连出现数十道身影，他们分别入主地府各处，坐镇幽冥界，各司其职！
<br/>　　九天之上，再次降下无数功德，乃是为三生大帝完善地府体系所降。
<br/>　　就这样，一个完整的幽冥界成型了，有平心娘娘执掌幽冥六道轮回，相当于一界天道，三生大帝为地府之主，之后五方鬼帝，十殿阎罗，各地判官等，体系完善。
<br/>　　从此以后，洪荒有了轮回！
