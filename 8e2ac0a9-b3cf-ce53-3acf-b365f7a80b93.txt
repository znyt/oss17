　　叶少阳环顾左右，冷笑道：“厉害啊，连轮回司都干起这种勾当了，想让我束手就擒，说出是谁下的命令，理由是什么，说出来我就跟你们走。”
<br/>　　三法王听了这话，表情有点尴尬，边上一个金甲鬼武士双手扶着长剑，傲然说道：“这里是轮回司，下令抓你的自然是转轮圣帝，至于为什么……阴司拿人，还要问为什么？”
<br/>　　当下对左右使了个眼色，大伙就要动手。
<br/>　　“站住！！”
<br/>　　叶少阳大喝一声，从兜里摸出天师牌，在他们面前晃了晃。“我是大帝亲封的人间天师，灵仙牌位，茅山掌教，就算是转轮王，想无缘无故拿我，也是有违阴司律法！”
<br/>　　狠狠瞪着那金甲鬼武士，道：“我特么现在站在这，你敢动我一下？”
<br/>　　众人听他这么说，一下子都愣住，互相看去，那金甲鬼武士目光阴鸷，突然冲叶少阳冷笑起来，“叶天师，你这是害怕了吗？”
<br/>　　“害怕？我害怕不小心弄死你啊！”叶少阳笑，“我掏天师牌，是要告诉你们，老子是大帝封的人间判官，你们没有大帝旨意，想拿我就是违法，我不认识你，但两位法王都挺熟的，我这也算先礼后兵了，真相动手，我打死不负责的啊！”
<br/>　　说完，将勾魂索解开提在手中，做好了动手的准备。
<br/>　　两位法王眼神更加犹豫起来，求助地朝徐文长看过去。
<br/>　　“小天师，你走吧。走了就别回来。”徐文长说完，用力推了叶少阳一把。
<br/>　　叶少阳犹豫了一下，就势往两位法王中间飞去。
<br/>　　两位法王往左右让了让，闪出一条缝隙，叶少阳从中间穿过，转头看了徐文长一眼，丢下一句话：“徐公，今天我看你面子，这就走了，将来但愿不要兵戎相见。”
<br/>　　叶少阳一口气朝司衙的大院外面飞，穿过中院的时候，一扫眼看到院子角落里站着个人，定睛一看，居然是父亲叶兵！
<br/>　　叶少阳当即就要飞过去。
<br/>　　叶兵冲他摇了摇头，脸上带着微笑，眼神中带着一丝柔软的怜爱。
<br/>　　叶少阳内心颤了一下，止住了冲动：本来自己就不能跟父亲有所联系，如今自己是逃兵，更不能给他带去麻烦。
<br/>　　一咬牙，纵身离去，心里头一阵发酸。
<br/>　　“徐公，这……”三法王为难地望着徐文长。
<br/>　　“这事有我，你们散了吧，我这就去见圣帝。”徐文长对他们拱了拱手，纵身飞往轮回大殿。
<br/>　　转轮王端坐在堂上，正在批阅文件。他身穿一套很宽松的麻布长袍，鼻梁上挂着一副老花镜，看上去像个人间的退休干部，只是微微下撇的嘴角，透着一种权力和威严的感觉。
<br/>　　徐文长走到他身边去，坐在他边上的椅子上，兀自发呆起来。
<br/>　　“如何了？”转轮王看着手里的文件，头也不抬地问道。
<br/>　　“明知故问。”徐文长摇了摇头，“只是这小子不太配合，我让他直接闯出去了事，结果他差点要打起来，没办法，只好明着放他走了。陈将军很不满意，估计回去之后，要在菩萨面前参你一本。”
<br/>　　转轮王不屑地笑了两声，道：“我原本可不理会的，我答应这么做，也是给你们几个一个面子……”
<br/>　　徐文长道：“了然，经过这件事，那小子也知道菩萨要对付他了，以后也会小心一些。”
<br/>　　“他若能想到这一层便好。”
<br/>　　“他是聪明人，会想到的。”
<br/>　　转轮王放下文件，发呆地望着前方，摇了摇头，暗叹了一声，道：“这事情，太费脑子。”
<br/>　　徐文长道：“四子之谜？”
<br/>　　转轮王没有回答，道：“你回头多找几个人，去人间走走，争取找到真相……”
<br/>　　徐文长叹息道：“这个够难的，除了大帝，没人知道真命是谁，我尽力吧。”
<br/>　　“对了，山海印的事，怎样，徐福跟那小子说了什么没有？”
<br/>　　徐文长想了想道：“说了个哑谜，我自然听不懂，那小子八成听懂了。”
<br/>　　“这么说，山海印果真在他身上？”
<br/>　　“八成。”
<br/>　　“那你还得多盯着他，免得他做出极端之事，一个徐福让我们找了上千年，这种事决不可再发生一回。”
<br/>　　徐文长点点头。
<br/>　　叶少阳速度回到了阴阳司。
<br/>　　大伙都在司衙里等着，见他回来，纷纷凑上来，询问他跟徐福见面的经过。
<br/>　　叶少阳把自己在轮回司被堵的事讲出来，大伙震惊。
<br/>　　“妈蛋，当我们这边没人吗，大伙一起去，找他们算账去！”瓜瓜招呼了一声，立刻得到大伙的响应，要操家伙跟他一起去轮回司。
<br/>　　“都别闹。”叶少阳让他们打住，“现在去轮回司闹，事情就闹大了，我是不怕，你们会惹出天大的麻烦，况且我也好好的不是。”
<br/>　　“有第一次，就有第二次啊！”瓜瓜嚷起来，“这次认怂了，下次还要被他们欺负！”
<br/>　　美华道：“老二你闭嘴吧，老大是要跟我们分析这件事，先讨论完了再去打架不迟！”
<br/>　　转而对叶少阳说道：“老大，这件事的主使，是转轮王？”
<br/>　　“八成，除了他，没人能差使动那些人，况且地方就在他们轮回司。”
<br/>　　“可是……如果是转轮王要对付你，徐公为什么还要来叫你。”美华分析道，“如果他是帮凶的话，那就没到底放你走……”
<br/>　　这一点，叶少阳回来的路上想过了，的确很矛盾，从他后来开门时镇定的表现来看，徐文长肯定早就知道他们会拦截自己，甚至这就是他自己安排好的，明面上是带自己去地牢找徐福，暗地里却安排人在地牢外面集结，伏击自己。
<br/>　　可是，他又为什么在神识中一再让自己逃走，后来甚至公开替自己拦住他们？
<br/>　　难道是后来良心发现，不想对付自己了？
<br/>　　如果他是这样出尔反尔、优柔寡断，那他根本不配做轮回司的师爷。
<br/>　　这时候橙子说道：“老大，徐公很有可能是在应付差事。转轮王要抓你，他没办法，只能照办，但因为私人关系，实在不想跟你打，本来是打算通过神识提醒你一下，放个水，结果你非要硬刚，他又实在没法下手，只好放你走了。”
