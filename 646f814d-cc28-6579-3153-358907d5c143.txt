　　金光险地内。
<br/>　　萧逸盘膝而坐，平伏着气息。
<br/>　　之前的一番大战，他虽未受什么重伤，但多少有些消耗。
<br/>　　半个时辰后。
<br/>　　“呼。”萧逸轻吐出一口浊气，缓缓站起。
<br/>　　一身精气神，已恢复到巅峰。
<br/>　　接下来，便是离开金光险地了。
<br/>　　“一月历练，倒是收获颇丰。”萧逸回头看了眼，自语一声。
<br/>　　不错，虽然只在金光险地历练了区区一月，但萧逸的收获极大。
<br/>　　修为方面，突破了两重，达到圣王境八重。
<br/>　　修罗战体，肉体力量方面，达到圣王境九重。
<br/>　　虽然妖兽精血还有些许，不过并不足以突破下一个大境界。
<br/>　　倒是金光丹方面，炼制了一大堆。
<br/>　　这一月历练，晚上与金刚白猿战斗半晚，剩下半晚便斩杀其它妖兽，一路横扫。
<br/>　　一月下来，倒也炼制了数千粒金光丹，其中大部分是圣皇境层次。
<br/>　　圣王境层次的金光丹，只有寥寥数百粒。
<br/>　　还有一份收获，便是那些最直接的金光力量。
<br/>　　那个巨大坑洞，长宽数十米，深度却达两万米之深，可想而知里头有多少金光力量。
<br/>　　且里头金光力量，异常浑厚澎湃，远不是金光险地夜晚上飘荡着的稀薄金光力量能比的。
<br/>　　而这些金光力量，此刻全部封存在八龙焚火炉内。
<br/>　　如此多的收获，此次历练，萧逸自然是满意的。
<br/>　　这还仅仅是历练一月时间。
<br/>　　当然了，事实上最大的收获，无异于八龙焚火炉连续四道封印的冲破。
<br/>　　还有里头的两只兽手，一只兽脚。
<br/>　　凭此，萧逸也有了斩杀武道大能的实力了。
<br/>　　萧逸握了握拳头，脸上尽是喜意。
<br/>　　嗖…思索罢这一月收获，萧逸闪身而离。
<br/>　　不多时，萧逸彻底出了险地深处，往险地边缘而离。
<br/>　　大半天后，萧逸终于出了金光险地。
<br/>　　不得不说，整个金光险地，确实无比辽阔。
<br/>　　“接下来，该去下一个地域了。”萧逸自语一声。
<br/>　　事实上，十八府地域，各府地域之内的险地，里头妖兽实力都差不多。
<br/>　　另外，里头也有一些实力极其恐怖的妖兽。
<br/>　　拿金光险地来说，继续深入的话，肯定会有比金刚白猿更强的妖兽。
<br/>　　只是，那个层次的妖兽，萧逸只能使用底牌去斩杀，如两只兽手。
<br/>　　但这不会有任何的历练效果。
<br/>　　故萧逸没有选择继续深入。
<br/>　　而其它十八府险地的妖兽，虽然实力差不多，但手段不同，属性也不同。
<br/>　　所谓历练，斩杀妖兽并不是目的，只是一个过程。
<br/>　　他需要的，是在这个过程中提升实力。
<br/>　　去其它险地，对付其它妖兽，见识其它手段，对他也是另一份考验。
<br/>　　这个过程中，感悟武道，完善自身剑道，亦是重中之重。
<br/>　　此时，萧逸已出了金光险地边缘。
<br/>　　这时，他正思索着去哪个地域。
<br/>　　忽然。
<br/>　　“嗯？”萧逸眉头一皱，看了眼不远处。
<br/>　　那里，一道火红色身影，冷眼盯着他。
<br/>　　“公孙火舞？”萧逸皱了皱眉。
<br/>　　“公孙姑娘有事找在下？”
<br/>　　看公孙火舞的神色，显然是在等他。
<br/>　　“你说呢？萧逸小贼。”公孙火舞咬了咬牙。
<br/>　　“你进金光险地一月，本姑娘也苦修了一月。”
<br/>　　“今日，再分个高下。”
<br/>　　“分个高下？”萧逸淡漠一笑。
<br/>　　“一月前，不是分出了？”
<br/>　　在萧逸的感知中，公孙火舞的气息，比一月前强些。
<br/>　　但也仅仅是强了些许罢了，仍旧只在圣王境九重修为。
<br/>　　“分出？哼。”公孙火舞冷哼一声。
<br/>　　“一月前，你那妖物忽然袭击，斩杀了万金府大长老。”
<br/>　　“本姑娘确实吓了一跳。”
<br/>　　“但事后想想，那不过是你那妖物的实力，却不是你萧逸小贼的实力。”
<br/>　　“若真靠你自己出手，怕是在万金府大长老手上走不过数招。”
<br/>　　公孙火舞冷声说罢，手中长鞭凭空而现。
<br/>　　啪…
<br/>　　长鞭一甩，噼啪有声。
<br/>　　“拿出你的真正本事，与本姑娘一战。”
<br/>　　公孙火舞长鞭一甩，直指萧逸。
<br/>　　萧逸见状，双眼一眯。
<br/>　　一月前，杀万金府大长老的，确实是雾妖。
<br/>　　不过，就在不久的前晚，萧逸亲手击杀了北隐宗大长老这位远比万金府大长老这个圣皇境巅峰更强的武道大能。
<br/>　　若是公孙火舞知晓此事，不知还是否有那个胆子这般挑衅。
<br/>　　萧逸摇了摇头，没有理会，自顾离去。
<br/>　　“小贼，给我站住。”公孙火舞冷喝一声。
<br/>　　手中长鞭，霎时一扬，瞬间出手。
<br/>　　萧逸眉头一皱，嗖…身影一闪，瞬间消失在原地。
<br/>　　萧逸的速度，快到了极致。
<br/>　　当他再次出现时，已在公孙火舞身前，指尖冷冷地抵在了公孙火舞咽喉处。
<br/>　　指尖上，一抹锋锐剑气，惊人无比。
<br/>　　若他此时愿意，一个念头便能取了公孙火舞性命。
<br/>　　“你…”公孙火舞感受到咽喉处惊人的锋锐，脸色一白。
<br/>　　她毫不怀疑，这道剑气瞬间就能切开她的喉咙。
<br/>　　“现在分出高下了。”萧逸淡漠说了一声，放下了手指。
<br/>　　公孙火舞不过是圣王境九重修为。
<br/>　　而萧逸，单单肉体力量便已达圣王境九重。
<br/>　　体修武者，本就碾压其他武者。
<br/>　　更别说萧逸修炼的乃是逆天体修功法，修罗战体。
<br/>　　要秒杀公孙火舞，不过是轻而易举之事。
<br/>　　萧逸放下手指，刚要离去。
<br/>　　公孙火舞脸色一怒，“你这忽然出手，与偷袭何异？不算，堂堂正正与本姑娘战一场。”
<br/>　　萧逸满头黑线。
<br/>　　即便这公孙火舞是修为不俗的武者，但她是个女人，而且是个刁蛮而不讲理的女人。
<br/>　　“败了便是败了。”萧逸淡漠道。
<br/>　　“别再纠缠不休，你太弱，在下没兴趣与你交手。”
<br/>　　话音落下，萧逸身影一闪，瞬间消失在原地。
<br/>　　不过数个呼吸间，萧逸已然远离。
<br/>　　公孙火舞看了眼周遭，丝毫感知不到萧逸的气息，亦感知不到萧逸往哪个方向而离。
<br/>　　“你…”
<br/>　　半晌，公孙火舞也只能愤愤地跺了跺脚，“好你个萧逸小贼。”
<br/>　　……
<br/>　　萧逸已远离千里之外。
<br/>　　十八府之间，虽然各自起码隔着十数乃至更多的地域。
<br/>　　但整体来看，十八府间各有相连。
<br/>　　金光府的两侧，分别是天索府与天刀府。
<br/>　　“天刀府？”萧逸思索了一下，摇了摇头。
<br/>　　“天索府？”萧逸自语一声，但一想到公孙火舞便是天索府少府主，又再次摇了摇头。
<br/>　　“从万金府出发，若是越过天索府，便是…”萧逸思索了一下，忽然眼前一亮。
<br/>　　“是天星府，便去天星地域吧。”
<br/>　　......
