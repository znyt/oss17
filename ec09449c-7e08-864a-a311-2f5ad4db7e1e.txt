<br/>    瑶池感觉到了，这一次的鬼门之争，又有人入场了。

<br/>        虽说不知道是谁，但是已经足够引起他们的警惕了。

<br/>        瑶池之上已经下达了最新命令：全面关闭仙路！

<br/>        天庭同样也是察觉到了不简单，瑶池栽了大跟头，不是自己动的手，那必定是第三方动的手。只是一个瑶池就已经够麻烦了，现在又来了第三方。为了防止对方继续增兵，天庭也是下达了最新的命令，全面关闭仙路。

<br/>        星空之中之所以划为各个星宿，星系，便是因为有着一颗颗大星隔开这些星辰，仙路一旦关闭，就必须强行突破，而这需要的代价，是巨大的。

<br/>        镇守星辰的仙人，将可以借用星辰的力量。

<br/>        赤龙星河河府，一座青色的山崖之上，有着一个茅草屋。

<br/>        在其中，一个赤着上身的汉子正在用铁水浇灌一颗头颅，片刻之后，他打开模具，还散发着热气的头颅，面孔分明是单流云的模样。
<br/>    
<br/>        新铸成的头颅，头顶之上连着一根锁链，他伸手，将这颗头颅系在自己的腰带上，在他的腰带上，还有着其他八个铁头颅。
<br/>    
<br/>        从他的背后，传来一声疯狂地笑声，“哈哈哈哈，段首，你又得了一颗好头。”

<br/>        白僵道：“这可由不得你，无面者阁下说了，这一次，无论如何，都要拿下这三头龙星宿的灵魂！”

<br/>        段首看向白僵，问道：“单流云乃是天庭将军，和你一样是金仙修为，你知道我是怎么杀掉他，不惊动其他人的吗？”
<br/>    
<br/>        白僵问道：“你是如何做到的？”
<br/>    
<br/>        段首
<br/>    
<br/>        伸手握住腰间的刀柄，断首大刀出鞘，刹那之后，他已经是出现在白僵的身后，白僵的脖子上，一条血线出现，白僵的头颅滑落，掉在地面上。

<br/>        白僵此时才真正感受到了什么叫做恐惧，段首的刀，太快了！

<br/>        白僵并没有死去，他毕竟是金仙，白僵的头颅看向出现在他身前的段首，段首伸手，将白僵的头颅抓在手中，白僵身上的鲜血顺着他的手流下。
<br/>    
<br/>        白僵道：“段首，你不能杀我的，我也是执刀人，你无法给无面者阁下交代！”

<br/>        段首呵呵一笑道：“无面者阁下在乎的只是灵魂，而不是你的命，只要灵魂没有出事，落在了我的手里，那么，我犯下的一切罪，都可以洗刷！”
<br/>    
<br/>        段首提着白僵的头颅，将白僵的头颅放进模具之中，问道：“有心思去想这些，还不如想想你需要什么配方的铁水！”
<br/>    
<br/>        白僵想要呼唤无面者阁下，但是却是想了起来，在这里，他是无法呼唤无面者的，这里是另外一个无面者的地盘。

<br/>        无面者和执刀人，只能在自己的地盘上呼唤对方。
